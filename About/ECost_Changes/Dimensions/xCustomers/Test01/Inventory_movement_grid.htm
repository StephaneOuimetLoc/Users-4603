Information

The inventory movement report displays the inventory movement grouped by department for the selected period. It takes the current inventory and adds or subtracts all transactions (purchases, adjustments, shorts and sales) back to the selected period.

The report is generated according to the criteria set in the Inventory movement by department wizard.

Layout

The inventory movement grid is located in the central part of the inventory movement grid page.

The default view displays fields related to the department (number, description and group), quantity (adjustments, purchases, sales) and total quantity (end of the week inventory, current inventory and differences in the inventory).

 

The grid view can be customized by hiding or adding columns from the column headers.

Content

This section focuses on the elements in the default view of the inventory movement grid.

It also describes the fields that can be added from the column headers.

Default view of the grid

The following elements are available by default:

ELEMENT DESCRIPTION
Period The time frame set for reporting the inventory movement.
It takes its value from the Period field in the inventory movement wizard.
Not editable.
Start The start date of the report.
It takes its value from the Start field in the inventory movement wizard.
Not editable.
Stop The stop date of the report.
It takes its value from the Stop field in the inventory movement wizard.
Not editable.
Target The target (POS or stores) of the inventory movement report.
It takes its value from the Target field in the inventory movement wizard.
Not editable
Department number The department number.
Department description  The department description.
Department group The group to which the department belongs.
Quantity starting The number of items in the department's inventory at the beginning of the reporting period.
Quantity adjustment (+)
 		The number of items that were added to the inventory during the reporting period.
Quantity purchases The number of items that were purchased during the reporting period.
Quantity sales
 		The number of items that were sold during the reporting period.
Quantity adjustment (-)
 The number of items that were taken out of the inventory during the reporting period.
Quantity ending
 		The number of items in the department's inventory at the end of the reporting period.
Total quantity EOW The number of items at the end of the week.
Populated when the reporting period is set to Week and Week by day.
Total quantity current The total number of items in inventory at the time of the report.
Total quantity difference
Total The inventory movement values for:
  • The item quantity: starting, adjustment (+), purchases, sales, adjustment (-) and ending.               
  • The total item quantity: EOW, current, difference.               

Additional grid columns

The column headers allow you to customize the grid view by adding columns that are not displayed by default.

The following additional columns are available:

COLUMN DESCRIPTION
Store ID The store number.
Populated when you check By store in the Inventory movement by department  wizard.
Store details Additional information about the store: number, name, city, class, group, location type, region, type, state.
The information is retrieved from the Store table in the eSuite System module.
Sub-department number The number of the sub-department to which the inventory items belong.
Sub-department description The description of the sub-department to which the inventory items belong.
Amount
Total amount
