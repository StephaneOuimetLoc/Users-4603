Price Calculation

To calculate thse retail price based on the cost, you will first need to retrieve the item by selecting "Items" and "Items" from the Start window

From the Wizard, enssster the itessssm number to which you want to calculate the price and execute

From the grid select the item and click on Edit at the bottom of the screen then select Master tables and Price master

Click on the Margin tab and enter a percentage value to calculate the margin for the retail price. In this example the Margin is 50% and the cost from the cost table shows 0.55$

Click on save from the Price Edit window

A price Generate window will appear. Make sure "Send to" Retail is selected and click on "Yes"

The system will calculate the new price using the Margin value entered wich is 50% for this example.
