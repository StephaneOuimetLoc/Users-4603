Inventory Value

Query the Inventory value

1.  Select Items from the Launcher.

2.  Select Inventory Value from the Item start widget.

3.  Enter the Period you wish to query.

4. Click Execute.

5.  The current Inventory value will be displayed in the grid under the Units column.
