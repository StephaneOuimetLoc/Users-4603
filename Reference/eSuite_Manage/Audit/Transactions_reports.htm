Transactions Report

eSuite Manage > Auditing > Transactions reports

Transaction history give details of the transactions performed during a certain period.

Transaction history:

  • All Transaction
  • Transaction details
  • Search for Functions
  • Search for totalizers
  • Search for Exceptions
  • Search for Reason Codes (under development)
  • Search for Operator Errors (under development)

 

 

 

The All transactions report wizard offers criteria selection that includes period, date range and target. Additional criteria may be added with the Add criteria button.

The Transaction list report displays all the transactions performed within a certain period and according to the selected criteria.

In the bottom menu, the Transaction button provides three options:

  • The All option displays the details of the selected transaction and the summary of its total amounts by totalizer.

  • The Detail option displays the information of the transaction in detail:

    Clicking on + displays the keystrokes used in the transaction:

NOTE: the transaction details can also be selected and executed through the Transaction history start menu.

  • The Total option displays the total amounts in the transaction by totalizer:

The Search for Function wizard offers criteria selection that includes period, date range, target, function, amount, Qty and weight.  Additional criteria may be added with the Add criteria button.

The transaction function list report displays only the transactions having used the function selected in the wizard criteria within a certain period and according to the additional selected criteria.

The Search for Totalizer wizard offers criteria selection that includes period, date range, target, totalizer, amount, Qty and weight.  Additional criteria may be added with the Add criteria button.

The transaction totalizer list report displays only the transactions having used the totalizer selected in the wizard criteria within a certain period and according to the additional selected criteria.

Back to top

The Search for Exceptions wizard offers criteria selection that includes period, date range, target and exception.  Additional criteria may be added with the Add criteria button.

The transaction exception list report displays only the transactions having used the exception selected in the wizard criteria within a certain period and according to the additional selected criteria.
