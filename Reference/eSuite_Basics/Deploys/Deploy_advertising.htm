eSuite - Deploy Advertising

eSuite > Tools > Deploy

Information

To be revised and update once this feature is available in eSuite.
 Raluca, 2019-10-16

The Deploy bitmaps job sends images of Bitmaps Folder from the Head office (HO) to the stores or lanes and/or eCommerce server.

The Deploy bitmaps job deploys the following information:

  • The images in Bitmaps Folder
  • Updating Bitmaps table (BMP_TAB)

To perform a Deploy bitmaps job:

  1. Click the arrow in the Tools entry in the Launcher to display its menu
  2. Select Deploy: the Start window opens in the Deploy work area
  3. Select Deploy advertising

NOTE: The Deploy advertising job requires the right security level. Different security levels allow different accesses.
