eSuite System - Periods Grid

eSuite System > Scheduler > Periods

Information

The Periods grid defines the different periods (fiscal and regular periods) of the store. The periods are used in reports.

By default, the list of periods contains six columns: Target, Type, Start date, End date, Number, and Description. The grid layout can be customized. For details, see the Grids in eSuite topic.

Periods are listed alphabetically in order of type. For each type, they are displayed in ascending order by end date.

Displaying the Periods grid

In the Launcher, click the Scheduler entry:

  1. The Scheduler window opens in the Scheduler work area.
  2. Click on Periods: the Periods grid opens in the List tab.

To create a new record, click Add in the toolbar; this will allow opening the Period table for editing.

To edit an existing record:

  • Double-click on it, or
  • Select it in the list and press the F3 key on the keyboard or click Edit in the bottom menu of the grid.

This will allow opening the Period table for editing.
