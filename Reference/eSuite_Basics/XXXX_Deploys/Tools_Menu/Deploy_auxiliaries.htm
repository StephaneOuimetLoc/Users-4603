eSuite - Deploy Auxiliaries

eSuite > Tools > Deploy

Information

The Deploy auxiliaries feature may be used to send the changes on the auxiliary tables to all devices running SMS. It allows you to send, consult and edit different objects on your SMS environment, including items, prices, batches, resources, targets, functions, routes and profiles. This includes price levels, like codes, Mix & match, bottle and environment fields, coupons, promotions, shelf IDs and routes.

The Deploy auxiliaries feature is common to most eSuite modules and is available from the Tools entry in the launcher.

Deploy auxiliaries

Select a target: click on the arrow in the Target field to access the list of targets.

In the Target window, select the target and click on OK.

Select the type of action you want performed on the database.

Select what you want to be sent through the auxiliary deploy.
