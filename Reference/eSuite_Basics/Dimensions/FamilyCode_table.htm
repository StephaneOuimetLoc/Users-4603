eSuite - Family Code Table

eSuite > Family

Information

The Family code table is a way for users to set up alternate groups for items, based on different aspects of a particular set of items.

Juices, for example, can be produced by different manufacturers but belong to the same sub-department, which makes it difficult to extract them from other items by using the type as a criterion. However, by defining different juice families (such as "Frozen juice"), each juice can be linked to one such family in order to easily select the frozen juice type only, regardless of the vendor, the sub-department or other eSuite divisions.

Displaying the Family code table

In the Launcher, click the arrow in the Dimensions entry to display its menu and then:

  1. Click on Families: the Family window opens in the Family work area
  2. Click on List: the Family tab opens in the same window, displaying the Family list
  3. To modify an existing family record, select it in the list and double-click on it, press the F3 key on the keyboard or click Edit in the bottom menu: the Family code table opens, overlapping the work area
  4. To create a new family record, click on Add in the bottom menu of the grid: the Family code table opens, overlapping the work area

Layout

* Click on a field to get information on it.

Content

Here is a description of the Family code table fields and their functionalities:

FIELD DESCRIPTION
Number The Number field contains the family number.
It is a required field used to uniquely identify records in the Family Code List.
Description The Description field contains the family code description.
It is not a required field; however, a descriptor should be entered as it helps identify the family.
