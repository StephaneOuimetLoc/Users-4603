Backup/Restore Transaction

ePOS > Functions > Transaction > Backup | Restore

Table of content

Information

A transaction can be left incomplete and be re-opened at a later time to finish the data capture. The Backup function is a way that can be used for that.

Note: A backed up transaction can be re-opened on any front-end workstation on your network. To re-open a backed up transaction, you must use the Restore function.

To backup the current transaction on the front-end workstation, Please follow the next steps:-

  1. Log into ePos
  2. Run a transaction
  3. Go to Functions
  4. Go to Transaction
  5. Click Backup




To Restore the current transaction on the front-end workstation, Please follow the next steps:-

  1. Log in ePos
  2. Go to Functions
  3. Go to Transaction
  4. Go to Restore
  5. Select the suspended transaction that you have previously backed up.
  6. Complete the transaction
