The Information functionality in eSuite offers details on the active session.

It is available from the Session button in the eSuite taskbar.

Layout

The Information button opens the Session information window.

Content

The following fields are available in the Session information window:

FIELD DESCRIPTION
Session information Shows the window's name.
Server Contains the name of the service that runs with eSuite.
Session number Shows the number of the active session, according to the store and terminal numbers, user level and date.
Store number Shows the store for which the eSuite application is installed and used.
Terminal number Indicates the node on which the eSuite application is installed and used.
User level Contains the identifier of the level of access.
Profile Information to come.
Transaction date Shows the date of the last transaction in eSuite.
Hardware profile link Shows information on the hardware profile.
Show options Opens the Installed options/samples window.
OK Closes the Session information window.

The Installed options/samples window

The following elements are available:

ELEMENT DESCRIPTION
Installed options/samples Shows the window's name.
Type Indicates the content type: option or sample.
Allows you to sort the column contents in ascending and in descending order.
Allows you to show/hide columns.
Hidden by default.
Name Indicates the name of the option or that of the sample.
Allows you to sort the column contents in ascending and in descending order.
Allows you to show/hide columns.
Last update Indicates the time of the last update.
Allows you to sort the column contents in ascending and in descending order.
Allows you to hide/show columns.
Type: Option Shows the name of the Option panel.
Option panel Displays the installed options.
Type: Sample Shows the name of the Sample panel.
Sample panel Displays the name of the installed samples.
Collapse Hides the Option panel.
Hides the Sample panel.
Expand Shows the Option panel.
Shows the Sample row panel.
Resize handles
Resize the page horizontally, vertically and diagonally.
They are visible in each corner and on each side of the window when you hover the mouse over the borders.
OK Closes the Installed options/samples window.
