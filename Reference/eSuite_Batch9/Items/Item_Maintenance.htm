eSuite - Item Maintenance

eSuite > Items > Master > Item List (New)

Information

The Items entry is common to most eSuite modules. Item maintenance can be carried out by managing items directly from the database or in a batch process.

The eSuite software keeps the item information in different tables, called item tables. They are essential for creating items, generating prices and tracking cost information. Item tables can be accessed from the Items menu in the launcher.

Creating an item

The eSuite software allows creating items and copying existing items to new ones.

Items may be created from the item grid. When the Add button is activated in the toolbar, it opens the Item creation table that includes the major sections of the main item tables:

 

See Creating items for details.

New items may also be created by copying existing items. When the Copy current item to a new item button is activated in the item grid, it opens the Item creation table and the Copy item window. This allows entering a new UPC and copying the information.

 See Copying a current item to a new one for details.

Editing an item

Items in eSuite may be modified from both the item grid or the item tables.

In the item grid:

  • Double-click the editable field where information needs to be changed. Editable fields in eSuite grids are identified by a small flag in the top right corner:
  • Save the changes.

 In the item tables:

  • In Items - General list, select the editable field where information needs to be changed.
  • Press F3 on the keyboard to open the item table based on the grid position.
  • Save the changes.

NOTE: Item tables may be opened from the Items entry in the launcher or by from Edit in the bottom menu of the item grid.

Item controls

The bottom menu in the item grid offers further controls for item maintenance: Edit, Analyze and Action:

Edit

The Edit control offers a complex menu that allows opening and editing item tables.

The table below briefly explains the menu options and their functionalities.

MENU SUB-MENU FUNCTION
Preferred Based on grid position (F3) Opens the item table corresponding to the field selected in the grid.
  All in one basic item (CTRL+F3) Opens the Item creation table with the information of the item selected in the grid.
  Multi core tables Opens the core information in the main item tables.
Master tables Main master Opens the Main item table.
  POS master Opens the POS table.
  Price master Opens the Price table.
  Cost master Opens the Cost table.
  Shelf location Opens the Shelf location table.
Special items Alternate codes Opens the Alternate codes table.
  Contract price Opens the Contract price table.
  Electronic coupon link Opens the Electronic coupon link table.
  Inventory trigger Opens the Inventory trigger table.
  Recipe Opens the Recipe table.
  Kit Opens the Kit table.
  Modifier Opens the Modifier table.
  Item classification Opens the Item classification table.
  Images Opens the Item images table.
Scale items Scale information Opens the Scale information table.

Analyze

The Analyze control offers tools for getting item information based on user-defined criteria:

  1. The Item sub query option groups and breaks down the information in the entire item grid.
  2. The Item-linked query allows additional tables to be joined with information related to the item that has been selected in the grid.

Action

The Action button allows assigning or unassigning promotions to the items selected in the grid. It also gives access to labels.
