eSuite - How to Generate Reports

eSuite General

Information

Reports in eSuite are available from the Start window of the related section. For example, item reports are located in the Items section and customer reports are available from the Customers section.

Report wizard

The wizard is a powerful search assistant. It boosts efficiency by offering multiple search fields and allowing layout customization.

Displaying the report wizard

Report wizards are available across modules. They can be accessed by clicking on the desired entry in the Launcher and making the right selection in the Start window:

 

Layout

The layout of the report wizards includes some common elements, with differences according to the selected report type (single total, multi totals, compare totals, etc.). Below are a few examples:

Generating reports

Wizards allow the selection of criteria that are necessary for executing reports. Depending on which particular report is required, additional criteria may be entered to filter the results of the report.

Setting the period

The Period section allows setting the time frame to a certain date or date range for comparison.  The time frame can be customized and tailored to fit your preference. For more information on how to configure the Periods consult the SMSMembers_V4.chm file for the topic: How To Customize the Report Period Time Frame.

  1. Select the time frame in the Period field.
    Use the left/right double arrows underneath to change values simultaneously in the Start and Stop fields.

  2. Enter the start date in the Start field or select it in the date picker.
    Use the left/right arrows underneath to move the date forward/backward.


    Inn Compare totals reports, this field is calledd Date and represents the comparison date.
  3. Enter the end date in the Stop field or select it in the date picker.
    Use the left/right arrows underneath to move the date forward/backward.


    In Compare totals reports, this field is called Compare and represents the comparison date.

Accessing targets and totalizers 

The Target and Totalizer fields provide access to further search tools.

  1. Click on the arrow to show the search tools option.

  2. Click on Show search tool to open the Target/Totalizer search window.
  3. Click on Go to display the list of targets/totalizers:

  4. Select the desired target/totalizer and click OK to set the selected value in the field.

Back to Generating reports

Setting ranges

Some of the fields in the report wizards provide access to the lists of totalizers, item, departments, etc. for a quick selection of ranges. The example below refers to the item code range in the Items - Multi totals report.

  1. Click the arrow in the Item code start field.

  2. Click on Show search tool to open the Item code start window.
  3. Click on Go to display the list of item codes:

  4. Select the desired item code and click OK to set the value in the Item code start field:

  5. Repeat steps 1 to 5 to set the desired value in the Item code stop field:

Back to Generating reports

Adding and deleting report criteria

Additional criteria may be entered to filter the results of the report.

  1. Click Add criteria:

  2. In the Select criteria window, check the boxes that correspond to the desired values.

  3. Click OK to display the new criteria in the wizard:

  4. Enter or select the desired values.

To delete new criteria, click next to the respective field.

Back to Generating reports

Breaking down report information

Depending on the type of report, the wizard allows breaking down information by date, store, item, etc. and displaying only records with no movement.

  1. Check any of the boxes in the Breakdown and Show only sections.

  2. Click Execute to open the report.

 

Back to Generating reports

Saving the configuration

Report wizards allow saving the search configuration:

  1. Add or delete criteria.
  2. Click on the arrow of the Save button to display its menu:


  3. Select Save as: the Save search window opens on the desktop.

  4. Enter the name of the new configuration.
  5. Click on OK to close the Save search window: the name of the new configuration is displayed in the Save configuration field.

Back to Generating reports

Deleting saved configurations

To delete a saved configuration:

  1. Click the arrow in the Saved configuration field to display its menu:

  2. Select the saved configuration to delete.
  3. Click on the arrow of the Save button.
  4. Select Delete: a confirmation message is displayed.
  5. Click on Yes.

Back to Generating reports

Grid layout

Grids in eSuite may be customized and saved in a public or private layout.

In report wizards, the Grid layout field automatically displays the default value, which means that the report will present information in its default form.

If the layout of the report has already been customized and saved, the Grid layout field offers the possibility of opening the report in the saved form.

  1. Enter the search criteria.
  2. Click on the arrow in the Grid layout field to display the menu.

  3. Select a saved grid layout.
  4. Click Execute to open the report with the saved layout.
