Inforggmation

The Base rights functionality in eSuite allows you to set the user level parameters. By default, basic user level settings are configured with minimal access rights.

The Base rights functionality is available from the Session menu in the eSuite taskbar.

Layout

The Base rights button in the Session menu opens the Start page.

 

The Base rights option in the Start page opens the Base rights page:



The layout is identical across modules. It includes the following sections:
  • Tab bar
  • Navigation bar
  • Title bar
  • Toolbar
  • Grid
  • Resize handles

Content

This section focuses on the components of the Base rights page and on the actions they trigger.

The tab bar

The tab bar is located at the top left of the Base rights page. It includes the Start and the List tabs.

The navigation bar

The navigation bar is located at the top right of the Base rights page.

It displays the following buttons:

BUTTON ICON DESCRIPTION
Freeze window size Freezes the window size to that set by the user.
Restore to default window size Resets the window size to the default one.
It reverses the action of the Freeze window size button.
Fit window to full screen Sets the window size to that of the screen.
Restore down from full screen Resets the window size to the default one.
It reverses the action of the Fit window to full screen button.
Fit window to desktop Sets the window size to that of the desktop.
Restore down from desktop Resets the window size to the default one.
It reverses the action of the Fit window to desktop button.
Move window Moves the window around the desktop and screen.
Close window Closes the window.

The title bar

The title bar is located at the top of the Base rights page.

It contains the name of the page.

The toolbar

The toolbar is located on the right of the title bar.

It contains the following buttons:

BUTTON ICON DESCRIPTION
Add record Adds a new record to the base rights grid.
Add record above Adds a new record above the one selected in the base rights grid.
Add record below Adds a new record below the one selected in the base rights grid.
Save Saves the changes made to the grid.
Undo changes Cancels the changes made to the grid.
Delete Deletes the selected record.
Prompt grab Allows you to select fields or lists in start pages, wizards, grids and windows.
The fields will be placed into the Base rights grid with the Grab field view button available in the taskbar.

The grid

The grid is located in the central area of the Base rights page.

It lists the existing user rights.

Data is displayed in columns and rows with flexible behavior:

  • Columns are movable and resizable.
  • Column headers allow you to show or hide columns.
  • Column headers also allow you to sort contents in ascending or descending order.

The following elements are available in the Base rights grid:

ELEMENT DESCRIPTION
Checkbox column The header checkbox allows you to select or unselect all grid records in a single click.
The column checkboxes allow you to select or unselect one grid record at a time.
Win column Shows the alias name of the window that will be impacted by the change.
The column cell is editable. It allows you to enter text or to select views through the search tool window.
When the column cell is empty, the change will impact all windows.
Query column Shows the ID of the element.
The column cell is editable. It allows you to enter text or to select an option among the following:
  • Add button
  • Delete button
  • Save button

When the column cell is empty, the change will impact all window components.
Level column Contains information on the user levels.
The column cell is editable. It allows you to enter text or to select an option from the list of user levels.
Action column Shows the type of change hat will be impact the window components.
The column cell is editable. It allows you to enter text or to choose an option among the following:
  • Read will set the component to read only.
  • Disable will disable and gray out the component.
  • Hide will hide the component completely.
Mandatory.
Default value: disable
Comment column Shows the notes on the user right.
The column cell is editable. It allows you to enter text.

The search tool window

The search tools are available from the drop-down menu of the Win column cell.

The search tool window contains the following elements:

ELEMENT DESCRIPTION
Close Closes the window.
Search Editable field. Allows you to enter text.
Select Confirms the selection of the view from the list.
Closes the window.
Places the selected view in the Win column of the Base rights grid.
Title column Contains the alias name of the view.
The column cell is not editable.
When double-clicked, it places the alias name in the Win column of the Base rights grid.
View column Contains the name of the view.
When double-clicked, it places the alias name of the selected view in the Win column of the Base rights grid.
OK Confirms the changes.
Closes the window.

The resize handles

The Base rights page provides eight resize handles, one in each corner and one on each side.

They allow you to resize the page horizontally, vertically and diagonally.
