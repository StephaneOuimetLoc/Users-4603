End of Shift-Day

ePOS > Functions > Tools > End of shift routine

Table of Content


  1. Log into ePos as a Manager
  2. Go to Functions
  3. Go to Tools
  4. Run End of shift routine or Function 952
  5. It will display a message saying "EOS in progress"
  6. Review details on the receipt
  7. Review details in the Financial > Multi totals report.


Note: You need to make sure there are no backed up/opened transactions in order for the EOS process to complete.
