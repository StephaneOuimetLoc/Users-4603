Information

The Coupons page of the eKiosk allows the customer to see all available coupons in the store.

Coupons Features

This section of the eKiosk contains certain features to help the customer to (see/find/clip) coupons. Here is a list of what this page supports:

  • Search: The customer can either type any text and SMS will find the coupon that has this text in it.
  • Promotions: This allows the customer to see the items in promotion in different categories such as This Week's Specials, Long Term Specials and Store/Manager's Specials.
  • Search by Category: Allows the customer to search their coupons via categories like Bakery, Deli, Fish, etc.
  • Search by Sub-Category: Allows the customer to search their coupons via sub-categories.
  • All available offers: This filter allows the customer to see all available offers in the store.
  • My offers ready to redeem: This filter allows the customer to view the coupons ready to redeem.
  • Old and used offers: This filter allows the customer to view only the old or used offers.

The Start Over button at the top is to refresh the search and go back to the original Coupons page.

From this page, when opening an coupon, it is possible to clip it to use in your order. See below what the prompt looks like.
