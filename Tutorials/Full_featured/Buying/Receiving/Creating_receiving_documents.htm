eSuite Buy - How to Manage Receiving Documents

eSuite Buy > Buy > Receive

Information

The eSuite Buy module offers the tools for managing the reception of the merchandise.

This document shows how to:

  • Create and edit a receiving document.
  • Add items.
  • Remove items
  • Manage quantities.
  • Suspend or close the receiving document.
  • Delete a receiving document.

Creating a receiving document

When merchandise has been ordered and received, the receiving document is created from the closed purchase order. When there is no purchase order (periodical orders from the same vendor, for example), the new receiving document must be created from scratch.

 To create a receiving document from a closed purchase order:

  1. In the launcher, click the arrow in the Buy entry to display its menu.
  2. Click on Receive: the Start window opens on the desktop.

  3. Click All next to Documents: the Open document grid opens on the desktop, displaying the receiving documents.
  4. Select the closed purchase order and right-click on it to display the available options.

  5. Click on Open to confirm the creation of the receiving document.

To create a receiving document from scratch:

  1. In the launcher, click the arrow in the Buy entry to display its menu.
  2. Click on Receive: the Start window opens on the desktop.

  3. Click on New: the New document window opens on the desktop.

  4. Select a vendor from the Vendor field.
  5. Click on Create to confirm the creation of a new receiving document for the selected vendor.

When a receiving document is created:

  • It is listed in the Open document grid.
  • Its status is OPEN.
  • The Document (full) option in the start menu is enabled.
    This gives access to the Document window with information on the document and tools to edit it.
  • It allows adding, removing or editing the items.

Adding items to the receiving document

The second step in the creation of a receiving document is adding items.

  1. Select Items in the Start window to open the item search wizard.
    The Items into current documents checkbox is selected by default: only items in the open receiving document will be displayed in the resulting grid.
    When unchecked, it allows displaying other items than the ones in the document.

  2. Add all necessary criteria and click Execute: the list of items opens on the desktop.



    In the document:
    • The Receive fields in the Case, Unit and Weight columns are editable.
    • The Total amount field is editable.
    It may be necessary to modify the values when quantities received do not match with quantities ordered.
  3. To edit the received quantities, double-click in the corresponding Receive field (Case, Unit, Weight) and add the received quantities.
    Press Enter after each new value.

  4. Click on Save in the toolbar: the new quantities and amounts are displayed in the Total column.

Editing the receiving document

Before closing the receiving document, you may want to revise and edit the information in it. This is done through the Document window, which offers the tools for editing general and specific information, managing quantities and amounts, suspending the receiving document or closing it.

  1. From the start menu, click on Documents - All: the list of receiving documents opens in the Open document grid.
  2. Select a receiving document.
  3. Right-click to display the list of options.

  4. Click on Open.
  5. Right-click again on the receiving document and select Show details: the Document window opens on the desktop.

The Document window of an open receiving is also available from the Document (full) option in the start menu. For details on the fields and functionalities, see Document window.

Editing general information

General information related to vendor and shipping may be edited in the Transaction header table.

  1. From the Document window, click on the Edit header button in the upper part of the document.
    The Transaction header table opens on the desktop.

  2. Make all necessary changes.
  3. Click on Apply changes in the toolbar.

For details on the fields and functionalities, see Transaction header table.

Editing specific information

Information related to items may be edited in the Transaction line receive tatable.

  1. From the Document window, select an item in the Lines section.
  2. Click on Edit selected line in the upper part of the section.
    The Transaction line receive table opens on the desktop.

  3. Make all the necessary changes.
  4. Click on Apply changes in the toolbar.

For details on the fields and functionalities, see Transaction line receive table.

Editing lines

Quantities (cases, units, weight) and amounts of the receiving document may be modified directly in the Lines section.

  1. Double-click in the case, unit, weight or amount column of the item.
  2. Enter or select the new values.
  3. If necessary, modify the amounts.

  4. ClClick Save all changes in the toolbar of the line section.

Voiding items

Items may be removed from the receiving document.

  1. In the Line section of the Document window, select the item to remove.
  2. Right-click to display the menu.
  3. Click on Void selected line: a confirmation message is prompted.

  4. Click on Yes to delete the item from the receiving document.

Suspending a receiving document

An incomplete receiving document may be suspended and edited at a later time.

  1. Open the Document window for the receiving document.
  2. Click on Suspend the document in the toolbar.
    The Document window closes.

To open a suspended document:

  1. From the Open documents grid, right-click on the suspended document.
  2. Select Open from the menu.
    The receiving document is available from the Document (full) option in the start menu.

Closing a receiving document

Receiving documents need to be closed for the billing to be intiated.

  1. From the list of documents, open the receiving document.
  2. From the Start window, click on Document (full): the Document window opens on the desktop.
  3. Click on Close document in the window toolbar: a confirmation message is prompted.

  4. Click on Yes to close the order.
  5. In the list of documents, click on Refresh in the bottom menu: the closed receiving document is not listed any more.

When a receiving document is closed, it is automatically transferred to the list of open documents in the billing.

NOTE: It is not possible to reopen a closed receiving document.

Deleting a receiving document

Open documents may be deleted from the Document window:

  1. In the toolbar, click on Cancel: a confirmation mesage is prompted:

  2. Click on Yes to delete the receiving document.
  3. In the Open document grid, click on Refresh in the bottom menu: the deleted document is not listed any more.
