mFloor - Advanced Search

mFloor Item Management

Information

The Advanced search function located in the mFloor Products module enables users to search for items based on the following criterion:

  • On promotion
  • Brand
  • Description
  • Aisle
  • Category
  • Department
  • Sub department

Searching using Advanced search

The advanced search function in mFloor enables users to search on a number of fields, some using wildcards and search for items.

To search for items using Advanced search:

  1. Click Products from mFloor main menu then click Advanced search
  2. At the Product Search screen enter the search terms in the Brand and Description fields, wildcards are automatically inserted before an after the term, and/or choose search criteria from the drop down fields.
  3. Click or tap Next
  4. If the criteria match items in the database, the Search result returns items matching the search.
  5. Click or tap an item in the search results to display the Item Information screen.
  6. At the Item Information screen users can click or tap the Item maintenance menu to access the various tables
