ePOS > Functions > Transactions > Suspend | Resume

Table of Content

Information

A transaction can be left incomplete and be re-opened at a later time to finish the data capture. The Suspend function is a way that can be used for that.

Note: A suspended transaction can only be re-opened on the front-end workstation where it has been suspended. To re-open a suspended transaction, you must use the Resume function.

To suspend the current transaction on the front-end workstation, Please Log into ePos and follow the next steps:- 

  1. Log in
  2. Run a transaction
  3. Go to Functions
  4. Go to Transaction
  5. Go to Suspend
  6. You should receive a message saying "The transaction suspended ok" indicating the transaction is suspended.
  7. Go to Resume - It will open a list of suspended transactions.
  8. Select the suspended transaction - You will read a message saying "Transac opened ok" to confirm that the suspended transaction has been opened/resumed successfully.
  9. Complete the transaction.
