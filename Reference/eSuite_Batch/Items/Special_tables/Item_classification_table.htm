eSuite - Item Classification Table

eSuite > Items > Master > Item list

Information

 

The Item classification table allows associating an item to the classification groups. An item can be part of several classification groups.

It is used to simplify the customer's search in eStore/mStore online shopping.

Display the Item classification table

In the Launcher, click the chevron in the Items entry to display its menu and then:

  1. Click on Master: the Start window opens in the Items work area.
  2. Click on Item list: the Wizard tab opens, displaying the Items - General list window.
  3. Enter the search criteria and click Execute: results will display in a record grid, in the Items tab of the same window.
  4. Select a record in the list.
  5. Click Edit in the bottom menu to display the options.
  6. Select Special items, then Item classification: the Item classification table opens, overlapping the work area.
  7. Click on Add in the toolbar to enable the fields and add a new record.

Layout

The Item classification table is divided in two parts: the Main section and the Classification grid.

Main section

*Click on a field to get information on it.

Classification grid

*Click on a field to get information on it.

Content

Here is the description of the Classification table fields and their functionalities:

FIELD DESCRIPTION
Main section
Item  The Item field displays the UPC of the item to which a classification will be associated.
Classification The Classification field contains the classification code and the description of the item selected in the item list.
For details, see the Associating items to classification groups topic below.
Classification grid
Classification column The Classification column contains all classification codes to which the selected item is associated. 
Description column The Description column contains the description of all classifications to which the selected item is associated.

Associating items to classification groups

Items can be associated to one or several classification groups:

  1. Select an item in the item list.
  2. Open the Item classification table.
  3. Click Add in the toolbar.
    The UPC of the selected item is added to the table and the fields are enabled:

  4. Enter the classification ID or click on the arrow to display the classification window.
  5. Save the changes: the classification code and its description are displayed.

Repeat steps 3 to 5 to add further classification groups to the item.
