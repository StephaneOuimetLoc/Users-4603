eSuite - Tare Table

eSuite > Items > Item auxialaries > Tare Table

Information

The Tare table contains tare weights that are linked to items.

Displaying the Tare table

In the Launcher, click the arrow in the Items entry to display its menu and then:

  1. Click on Auxiliary tables: the Start window opens in the Auxiliary work area.
  2. Click on Tare Table: the tab opens in the same window, displaying the Tare List.
  3. To modify an existing tare record, select it in the list and double-click on it, press the F3 key on the keyboard or click the Edit command in the bottom menu: the Tare list table opens, overlapping the work area.
  4. To create a new tare record, click on Add in the bottom menu of the grid: the Tare table opens, overlapping the work area.

Content

Here is a description of the Tare table fields and their functionalities:

FIELD DESCRIPTION
Number The Number field contains the tare number.
It is a required field used to uniquely identify records in the Tare List.
Descriptor The Descriptor field contains the description of the tare entry.
It is not a required field; however, a descriptor should be entered to easily identify the record in the Tare List.
Weight The Weight field contains the packaging weight to deduct from the full item weight when it is sold.
