eSuite - Shelf Location Table

eSuite > Items > Master > Item list

Information

The Shelf location table specifies where each item is located in the store, and controls the shelf facing.

Displaying the Shelf location table

In the Launcher, click the arrow in the Items entry to display its menu and then:

  1. Click on Master: the Start window opens in the Items work area.
  2. Click on Item list: the Wizard tab opens in the same window, displaying the Items - General List.
  3. Enter the search criteria and click on Execute: results will display in a record grid, in the Items tab of the same window.
  4. Select a record and click on Edit on the bottom of the item grid: the option menu is displayed.
  5. Select Master tables and then Shelf location: the Shelf edit table opens, overlapping the work area.

Layout

The Shelf table is divided into three parts: the main section, the bottom section and the Shelf table grid.

* Click on a field to get information on it.

Bottom section

Content

Here is the description of the Shelf table fields and their functionalities:

FIELD DESCRIPTION
Main section
Target The Target field contains the target identifier (the store group). 
It is a required field. 
The default value of the Target field is PAL.
Shelf ID The Shelf ID field contains the shelf identifier within the aisle where the item is located.
It is a required field.
Label template The Label template field contains the default type of shelf tag to be printed for the item.
It is used to filter the labels to print in the Labels module as well as on the PDA.
The label template can be a single tag or a template tag. See the Label template group table to get details on the template tags.
Main The Main field specifies whether a shelf label is to be printed in the store for this item (checked), or not (unchecked).
Position The Position field contains the facing position on the shelf where the item is located.
Side The Side field contains the side of the aisle where the item is located.
Aisle The Aisle field contains the aisle where the item is located.
Section The Section field contains the section of the store where the item is located.
Quantity The Quantity (Qty) field contains the number of shelf tags to print for this item.
Life The Life field contains the number of days the item is available for sale before being removed from the sales area.
Plano ID Description to come.
Bottom section
Facing X The Facing X field contains the number of facings on the shelf for the item. 
The default value of the Facing X field is 1.
Facing Y The Facing Y field contains the number of units in height included in one facing. 
The default value of the Facing Y field is 1.
Facing Z The Facing Z field contains the number of units in depth contained in one facing.
The default value of the Facing Z field is 1.
Total quantity The Total quantity field displays the total number of units on the shelf for the item, according to the facing numbers entered.
The Total quantity value is automatically calculated (Facing X x Facing Y x Facing Z).
Total X The Total X field displays the total width value that the item takes up on the shelf according to the item width and the number of facings.
The Total X value is automatically calculated (item width x Facing X).
Total Y The Total Y field displays the total height value, in centimeters, that the item takes up on the shelf according to the item height and the number of units in height contained in one facing.
The Total Y value is automatically calculated (item height x Facing Y).
Total Z The Total Z field displays the total depth value, in centimeters, that the item takes up on the shelf according to the item depth and the number of units in depth contained in one facing.
The Total Z value is automatically calculated (item depth x Facing Z).
Shelf table grid
Target column The Target column contains the list of target identifiers (store groups) for the item.
Shelf ID column The Shelf ID column contains the list of shelf identifiers within the aisle where the item is located.
