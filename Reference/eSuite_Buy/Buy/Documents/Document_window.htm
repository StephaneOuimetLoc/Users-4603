eSuite Buy - Document Window

eSuite Buy > Buy > Receive

Information

Information in the order, reception, billing and delivery documents is displayed using the Document window. It is available throught the Document (full) option in the start menu, when a document is opened from the Open document list.

The information in the Document window is grouped in several sections numbered below:

*Click on a section number for details.

See the table for details on each section.

NO. SECTION NAME DESCRIPTION
Toolbar The following tools are available:
  Payment
Allows choosing the payment method.
Enabled in DSD documents only.
  Add to bill
 Adds fees and deductions to the current bill.
Enabled in billing and DSD documents.
  Subtract from bill
 Subtracts fees and deductions from the current bill.
Enabled in billing and DSD documents.
  Charge to account
Charges the amount in the current bill to the account.
Enabled in billing and DSD documents.
  Suspend the document
 Suspends the document. This allows editing it later.
Enabled in purchasing, receiving, billing and DSD documents.
  Close the document
Closes the document.
Enabled in purchasing, receiving, billing and DSD documents.
  Cancel
Deletes the open transactions.
Enabled in purchasing, receiving, billing and DSD documents.
Edit header Allows editing the header section of the document.
Opens the Transaction header table.
Header Contains general information on the document.
May be edited through the Transaction header table.
Lines Lists all entries in the document.
Double-click to edit the following fields: Cases, Units, Weight, Amount.
Use the toolbar to edit or void lines and to save the changes.
Totals Displays the totals in the open document.
Information is divided by totalizer and total (unit, dollars, weight).
Toolbar (Lines) The following tools are available:
  Edit selected line
Opens the Transaction line table.
    Void selected line
 Prompts a confirmation message before voiding the line.
    Save all changes
 Saves all changes made to the lines grid.
Summary of totals Displays the summary of the information in the Totals section.
Visible from the Lines section.
