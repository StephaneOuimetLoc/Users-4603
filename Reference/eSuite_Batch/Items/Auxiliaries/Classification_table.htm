eSuite - Classification Table

eSuite > Items > Item auxialaries > Classification

Information

The Classification table contains a classification code that links the product to the classification group. Classification groups may be used to identify different groups of items such as gluten-free, organic and peanut-free. An item can be part of several classification groups.

It is used to simplify customer's search in eStore/mStore online shopping.

Displaying the Classification table

In the Launcher, click the arrow in the Items entry to display its menu and then:

  1. Click on Auxiliary tables: the Start window opens in the Item auxiliaries work area.
  2. Click on Classification: the tab opens in the same window, displaying the Classification list.
  3. To modify an existing item classification record, select it in the list and double-click on it, press the F3 key on the keyboard or click the Edit command in the bottom menu: the Classification table opens, overlapping the work area.
  4. To create a new item classification record, click on Add in the top menu of the grid: the Classification table opens, overlapping the work area.

Layout

*Click on a field to get information on it.

Content

Here is a description of the Classification table fields and their functionalities:

FIELD DESCRIPTION
ID The ID field contains the classification identification number of the item. 
It is a required field used to uniquely identify records in the Classification list.
Descriptor The Descriptor field contains the description name.
A description should be entered to identify the classification.
