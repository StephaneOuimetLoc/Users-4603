ePos - voids

epos operations

Information

The void function allows you to remove item lines during a transaction. There are two types of voids you can call in a transaction :

FCT 610 Void last item  and FCT 620 Void previous item.

Void Current

The Void Current Operation allows you to void any selected item line in a transaction.

  1. Open a epos session and login.
  2. Sell Items in a transaction.
  3. Select the line of the item you wish to void.
  4. Call the Void Current button.
  5. A message will pop up Void previous item 1=Yes/0=No ?
  6. Select 1 to void and Select 0 to Cancel.
  7. If you selected 1, the item line selected will be voided.


Void Last

The Void Last Operation allows you to void only the last item entered in the transaction.

  1. Open a epos session and login.
  2. Sell Items in a transaction.
  3. Call the Void Last button.
  4. A message will pop up Void Last item 1=Yes/0=No ?
  5. Select 1 to void and Select 0 to Cancel.
  6. If you selected 1, the last item entered in the transaction will be voided.
