eCommerce - Carousel Management

Manage the rotation of images found in the carousel

Information

In order to manage the rotation of images found in the carousel, you can modify the XML for the eStore and mStore respectively inside of the eSuite SYSTEM module. You go through the Edit navigation and in the resources option you will have both eStore and mStore Carousel menu options to edit individually.

Once you have added the images you want to use in the BITMAPS folder, you can edit the XML here by double clicking the relevant file and open the editing menu which contains three fields. Add a record line or edit existing images as needed.

File Contains the exact file name of the image (including the extension type)
URL A link that makes the image clickable to redirect to another page or website
Item Contains a UPC if you want the image to be clickable and redirect to specific products
