Table creation

@CREATE(<TableName>,<TableDefinition>)

If a file CREATE_TABLE_<TableDefinition>.sqi exist in the office directory, the system will execute it to create the table. You can use this to customize the table creation. Before the script gets called, the system assign <TableName> to the pool parameter CREATE_TABLE_NAME.

If not, the system will create a table in the database using the data dictionary fields.csv to obtain the file definition structure. The system will also combine the table definition structure of the data dictionary fields2.csv if the file exist.

Parameter Description Required/Optionnal
TableName Name of the new table Required
TableDefinition The column name of file fields.csv containing the table definition structure Required
Return value
None
Example
@CREATE(SAL_HDR,HDRSAL)
