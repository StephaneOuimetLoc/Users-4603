eSuite - Context tools

eSuite > Desktop

Information

The eSuite interface includes a contextual window whose behavior is identical across modules. The Context tools window collects and displays the information of the record that is selected in the desktop's active window.

It also provides additional tools linked to the record.

The Context tools window is located on the right side of the eSuite desktop.

Layout

The layout of the Context tools window varies depending on the type of installation (single target or multi-target) and on the entry selected in the launcher.

The following sections are common across modules.

  1. Title bar
  2. Panel bar
  3. Panels

Behavior

This section focuses on the behavior of the Context tools window components and on the actions they trigger.

The title bar

The title bar is located at the top of the Context tools window.

It contains the Collapse/Expand panel button that will minimize or maximize the window.

The panel bar

Panel bars are located above each panel.

It contains the Collapse/Expand panel button that will minimize or maximize the panel.

The panels

The panels display information about the record that is selected in the desktop's active window.

For example, when you click on Items > Master in the launcher, the desktop will display the Context tools window with the Items and the Dimensions sections expanded.

The information in the Context Tools window will vary based on what you select in the window that is active on the desktop.

For example, when you check an item in the results grid, the Context tools window will display in the Items and the Dimensions sections the information that is available for that record in the item and in the dimensions tables.

The following fields are common to the Context tools window across modules:

ICON FIELD DESCRIPTION
Collapse panel Hides the content of the panel.
Expand panel Shows the content of the panel.
Unlocked Indicates the value of the selected field is unlocked.
Click will toggle to a locked state.
Locked Indicates the value of the selected field is locked.
Click will toggle to a unlocked state.
Search Opens the search window for the selected field.
Linked query Opens the query menu linked to the selected field.
Edit Opens the table for the selected field.
