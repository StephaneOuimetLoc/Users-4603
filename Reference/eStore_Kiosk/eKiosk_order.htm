eKiosk - Ordering

eKiosk

Information

The Order page of the eKiosk allows the customer to order their items directly from this machine without having to wait in line to get served. This allows for a faster response time and diminishes the waiting line.

From this page, when selecting an item, it is possible to add it to your personal wish list and add it to your order. See below what the prompt looks like.
