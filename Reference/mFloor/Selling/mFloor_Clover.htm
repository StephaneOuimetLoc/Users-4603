mFloor - Payment on Clover Devices

mFloor Selling

Information

This document describes how to use the Clover for payment when using mFloor on a Clover Flex device.

Invoice Total screen

When using a Clover device, a new button labeled Pay will appear on the Invoice Total screen of mFloor.
This button triggers the payment with the Clover application by sending it the amount to be paid.

Payment Process

When pressing the Pay button, the Clover app opens asking to confirm the amount to be paid.
This amount is coming from mFloor directly and it's the full balance to pay.
Before completing this step, the Cashier can modify the amount if needed. When done, press the Charge button.

The Clover device will display a screen with instructions for the customer. The next steps are to be completed by the card holder (ie: Tap, Insert or Swipe).

Upon completion, the device is to be handed back to the cashier. The device will display a Receipt Options screen containing the following choices:

  • Printer Customer Receipt: This prints a bank stub directly from the Clover printer.
  • Email: This button send a copy of the receipt via email to the customer (The send receipt via email part of mFloor must be configured for this to work).
  • Text Message: This button send a copy of the receipt to the customer's phone via text (This feature is not currently supported)
  • Void Transaction: TBD
  • Done: This button bring the cashier back to mFloor.

If the payment is not complete, mFloor will show the Invoice Total screen again with the updated balance to pay.

Detailed Transaction Receipts

Currently it is only possible to send the receipt via email to the customer. mFloor will look into the transaction header to find if there is an email.
If not, it will prompt the customer to enter his email address in order to get his receipt. As previously mentionned, the Clover device can still print a bank stub directly on its printer.
