Periodic Count

Perform Counts for Past Dates

1.  From the Launcher, expand Inventory and select Count.

2.  From the Multi-count Start widget, create a New Multi-Count document.

3.  Select the date for which the count will be entered for and click Create.

4.  On the Multi-Count Start wizard select how you would like to search for your item. 

TIP: To search for items via wizard selection select Items. To see All items, select All.

5.  Enter the number of units for the items and Save.

Perform Counts by Sub-Department

1.  From the Launcher, expand Inventory and select Count.

2.  From the Multi-count Start widget, create a New Multi-Count document.

3.  Select a sub-department for which the count will be entered and click Create.

4.  On the Multi-Count Start wizard select Items - All. 

TIP: Only the items from the selected sub-department will be dispayed.

5.  Enter the number of units for the items in the Units column and click on Save .

Closing the Multi-Count document

  1. From the launcher click Inventory.
     
  2. From the Multi-Count Start widget, click Documents - All.
     

  3. Select a document and double click to open it in the context tools menu.
     
  4. Click the  icon in the Context Tools menu next to the document number.

  5. Select Close the document.
     

  6. Click Yes to confirm.
     


In the Opened documents grid, click   to refresh the list; the multi-count document is no longer listed.

NOTE: It is not possible to reopen a closed multi-count document.
