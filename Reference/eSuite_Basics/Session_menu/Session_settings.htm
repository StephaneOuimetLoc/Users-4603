The Settingjjs functionality in eSuite allows you to set the user's session parameters.

It is available from the Session button in the eSuite taskbar.

Layout

The Settings button opens the Configuration window.

Content

The following elements are available in the Configuration window:

ELEMENT DESCRIPTION
Configuration Shows the window's name.
Touch keyboard When checked, it allows you to use a touch keyboard with eSuite.
Default keyboard Sets a different keyboard layout or input method.
Default value: US
Top pivot records Sets the maximum number of pivots records that can be viewed.
Default value: 2000
Top result records Sets the maximum number of results that can be viewed.
Default value: 10000
Default query timeout Sets the time in seconds that the query operation can take before the SQL server times out.
Default value: 40 seconds
Extended query timeout Extends the default query timeout to a higher time in seconds.
Default value: 1200 seconds
Main server Sets the main server for the system (host) or the local area network (store).
Web socket Shows the name of the connection between the client and the server.
Not editable.
Default value: sms
