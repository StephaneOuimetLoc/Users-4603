Reset Average Cost

You can reset the average cost of an item by creating an inventory count.

Begin by running a report to retrieve the current average cost of the item. For example, you can run the report Inventory history which is one of many reports that provides the average cost.

In this example, the average cost is 8.75$ per unit where the case size is 12 units.

To reset the average cost

  1. Create an inventory count
  2. In the Line edit, modify the Avg. Cost field by entering the new value. Following the same example, we're changing the average cost from 105$ to 108$ where 12 units per case will average to 9.00$ per unit.

  3. Save and close the document.
  4. Validate the changes by running the Inventory value report again. For the same example, the average cost has changed to 9$ per unit (108$ per case of 12).
