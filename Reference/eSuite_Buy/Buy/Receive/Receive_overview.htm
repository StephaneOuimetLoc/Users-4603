eSuite Buy - Receive Overview

eSuite Buy > Receive

Information

The Receive entry opens a list of options for the management of merchandise. It includes tools to create and edit receiving documents.

This document provides an overview of the features specific to each option:

  • Full document
  • Documents
  • Documents - All
  • Documents - New
  • Vendors
  • Vendors - All
  • Items
  • Items - All

The following option is currently under development:   Scheduled delivery

Full Document

The Document (Full) option in the Start window is enabled when a receiving document is opened from the list of documents (Documents and Documents - All).

When enabled, it opens the Document window with information on the receiving document:

The general information in the receiving document may be edited with the Transaction header table.

The information on the items in the receiving document may be edited with the Transaction line receive table.

Documents

The Documents option in the Start window opens the Document search wizard that provides access to the list of open receiving documents and closed orders.

The Document search wizard offers criteria selection that includes vendor ID and name and item code and description. Documents may be sorted by type: open receiving, closed order or both. Additional criteria on POS description, sub-department, family and category may be added with the Add criteria button.

When executed, the wizard displays the Open document grid with the list of the documents that correspond to the chosen criteria:

In the bottom menu, Action allows to:

  • Open the selected receiving document, which enables the Document (Full) option.
  • Show details of the selected receiving document in the Document window.
  • Select the documents to use with the Multi-document feature in the Context tools window.

Documents - All

The Document - All option in the Start window gives direct access to the Open document grid, without having to go through the Document search wizard.

Documents - New

The Documents - New option in the Start window provides the tools for the creation of receiving documents.

When selected, it opens the New document window, it allows the selection of a vendor and the creation of the document:

See Creating receiving documents for details.

Vendors

The Vendors option in the Start window opens the Vendor search wizard that provides access to the list of vendors.

The Vendor search wizard offers criteria selection that includes vendor ID and description and item code and description. Additional criteria on POS description, sub-department, family, category and shopper level may be added with the Add criteria button.

When executed, the wizard opens the list of vendors:

In the bottom menu, Action allows creating a new purchase order for the vendor selected in the list. See Creating a purchase order for details.

Vendors - All

The Vendors - All option in the Start window gives direct access to the Vendor list, without having to go through the Vendor search wizard.

Items

The Items option in the Start window opens the Items - list of items wizard. This provides access to the items of the vendor in the receiving document.

The Item wizard offers criteria selection that includes item code and description, sub-department, category, family, report code, alternate code. The Vendor ID field is selected and enabled by default, allowing to display only the items offered by the vendor in the current document. Additional criteria on sub-department, category, family and report code ranges as well as and vendor code, batch and target may be added with the Add criteria button.

When executed, the wizard opens the list of the items corresponding to the chosen criteria:

 

The Case receive, Unit receive, Weight receive and Total amount fields are editable, allowing to set the respective values for each item in the receiving document. See Creating receiving documents for details.

In the bottom menu:

  • Edit provides access to all item tables.
  • Action allows confirming or not the quantity received for the items selected in the list.

Items - All

The Items - All option in the Start window gives direct access to the Item list, without having to go through the Item search wizard.
