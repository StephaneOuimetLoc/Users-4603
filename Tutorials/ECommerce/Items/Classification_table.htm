Information

The Classification table allows adding groups, that will be assigned to certain items, simplifying users' online shopping by sorting the displayed items by one or more groups in the eStore and mStore applications.

The Classification groups are used to identify different groups of items such as Gluten free, Organic and Peanut free, since it is possible to associate more than one classification group per item. Refer to the Classification table topic for details.

NOTE: When creating a new classification, the ID number must be unique. If an existing ID number is entered for a new record, a key violation error message appears when clicking 'Apply changes'.

The Item Classification table

The Item Classification table allows the association of one or more classification groups with an item. Refer to the Item Classification table topic of the Users.chm file for details.

NOTE: A classification number can only be associated once, if the same number is entered twice, a key violation error message will be displayed.

In the mStore application, under Filter all types of search are displayed. Then, under the Classification section, the user will choose between the Classification available to display items.
