Information

The eSuite grid allows you to export its data to Excel.

Layout

The export tools are located in the grid toolbar.

Export tools

The Export button in the toolbar opens the list of export tools.

TOOL DESCRIPTION
Only visibles Exports the visible grid contents to an Excel file.
Downloads the file to your computer.
All columns Exports all grid columns to an Excel file.
Downloads the file to your computer.
Options Opens the Export Excel window.
Allows you to change the name of the downloaded Excel file.

The Export Excel window

The Export Excel window opens when you click on the Options button in the Export menu.

The following elements are available:

ELEMENT DESCRIPTION
Excel Export Text field
Shows the window's name.
Close Button
Closes the window.
File name Text field
Allows you to change the name of the Excel field.
Default value: the grid's name followed by the current date.
Only visible columns Checkbox
When checked, it transfers only the grid visible columns to the Excel file.
When unchecked, it transfers all grid columns to the Excel file.
Checked by default.
Submit Button
Submits the changes.
