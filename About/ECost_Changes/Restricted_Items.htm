Restricted Items

In the Launcher, click the chevron in the Items entry to display its menu and then:

Click on Auxiliary tables: the Start window opens in the Auxiliary work area.

  1. Click on Restrictors: the tab opens in the same window, displaying the Restriction List.
  2. To modify an existing restriction record, select it in the list and double-click on it, press the F3 key on the keyboard or click the Edit command in the bottom menu: the Restriction table opens, overlapping the work area.
  3. To create a new restriction record, click on Add in the bottom menu of the grid: the Restriction table opens, overlapping the work area.

In this example, we're setting up a restriction for Alcool where it can only be sold between on Sunday between 06:00 and 23:00

Once the retstriction table is configured then you would need to link the item(s) to the restriction level where in this example it's level 4.

To do so, you will need to first retrieve the item(s) from the items-General list and edit the POS master table.

Click on the Restriction drop down and select level 4 then save.
