eSuite - Subqueries

eSuite > Launcher entry > Start page > Grid or list > Analyze

Information

Subqueries in eSuite allow you to perform advanced queries based on the current query results.

The Subqussssssery option is available from the Analyze control menu in most grids and reports. All the windows related to the subquery will open on the same desktop.

Layout

The Subquery page layout is identical across modules. It includes the following sections:

  1. Tab bar
  2. Navigation bar
  3. Title bar
  4. Content panel
  5. Resize handles

Content

This section focuses on the components of the Subquery page and on the actions they trigger.

The tab bar

The tab bar is located at the top left of the Subquery page. It includes the Start tab that displays a panel with options based on the current query.

The active tab is marked by a blue horizontal line.

NOTE: On some pages, the Start tab is replaced by the Start icon .

The navigation bar

The navigation bar is located at the top right of the Subquery page.

It displays the following controls:

CONTROL ICON DESCRIPTION
Freeze window size Freezes the window size to that set by the user.
It reverses the action of the Freeze window size button.
Restore to default window size Resets the window size to the default one.
Fit window to full screen Sets the window size to that of the screen.
Restore down from full screen Resets the window size to the default one.
It reverses the action of the Fit window to full screen button.
Fit window to desktop Sets the window size to that of the desktop.
Restore down from desktop Resets the window size to the default one.
It reverses the action of the Fit window to desktop button.
Move window Moves the window around the desktop and screen.
Close window Closes the window.

The title bar

The title bar is located at the top of the Subquery page.

It contains the name of the panel.

The content panel

The content panel is located in the central area of the Subquery page. It contains the options that are based on the entry selected in the launcher.

The resize handles

The Subquery page provides eight resize handles, one in each corner and one on each side.

They allow you to resize the page horizontally, vertically and diagonally.

The resize handles are visible when you hover the mouse over the Subquery window.
