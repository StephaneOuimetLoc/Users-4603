Information

The eSuite grid search tools allow you to look up information on the records in the grid.

Search menu and tools

The search tools are grouped in a menu that is located at the top of the grid.

The following search tools are available:
  1. Search for
    Allows you to add the search criteria.
  2. In column
    Opens a drop-down menu with options that are specific to the grid type (items, customers and so on).
  3. First
    Launches the search.
    Highlights the first result in the grid.
  4. Next
    Highlights the next result in the grid.
