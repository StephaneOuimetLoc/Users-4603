eSuite - Shelf ID Table

eSuite > Items > Item auxialaries

Information

The Shelf ID table specifies where each item is located in the store and controls the shelf facing.

Displaying the Shelf ID Table

In the Launcher, click the chevron in the Items entry to display its menu and then:

  1. Click on Auxiliary tables: the Start window opens in the Auxiliary work area.
  2. Click on Shelf ID: the tab opens in the same window, displaying the Shelves List.
  3. To modify an existing shelf ID, select it in the list and double-click on it, press the F3 key on the keyboard or click the Edit command in the bottom menu: the Shelf ID table opens, overlapping the work area.
  4. To create a new shelf ID, click on Add in the bottom menu of the grid: the Shelf ID table opens, overlapping the work area.

Content

Here is a description of the Shelf ID table fields and their functionalities:

FIELD DESCRIPTION
Target The Target field contains the target identifier, i.e. the store group. 
It is a required used to uniquely identify records in the Shelf table. 
The default value of the Target field is PAL.
Shelf ID The Shelf ID field contains the shelf identifier within the aisle where the item is located. 
It is a required field used to uniquely identify records in the Shelves List.
Aisle The Aisle field contains the aisle where the item is located.
Side The Side field contains the side of the aisle where the item is located.
Segment Description to come
Section The Section field contains the section of the store where the item is located.
Shelf life The Shelf life field contains the number of days the item is available for sale before being removed from the sales area.
Tag quantity The Tag quantity field contains the number of shelf tags to print for this item.
Label template The Label template field contains the default type of shelf tag to be printed for the item.
