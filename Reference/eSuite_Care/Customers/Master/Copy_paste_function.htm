eSuite - Copy and Paste Functionality

eSuite

Information

The eSuite copy functionality allows grabbing relevant data on customers and items from e-mail, Word, Note Pad documents, CSV or Excel spread sheets. The paste functionality allows the data to be displayed in the wizards. It can also be used to create batches (price, cost, POS, main item).

The copy and paste functionality is available from the Menu button in the customers, items and batch wizards.

The sections below show how to use this functionality for customers and batch wizards.

Copy and paste for Customer data

From the Customer wizard, click on the Paste customers from Excel option from the Menu pull-down button.

This will bring up the Clipboard paste search window with two options: Templates and Column and Row separators.

Templates are most often used with CSV & txt files to copy from tables and apply column and row separators directly in the functionality, once data is pasted.

\t Used as a Tab (Tabulation)
\n Used to moved code for end-of-line
\r Used to move the carriage back to the left side if the page

To save a template, data must first be associated with the use of Associated fields. Once the template saved, it will no longer be necesary to associate each field: paste in the data, select the created template from the pull-down list and the data fields will be automatically associated.

The captured data may be pasted in the Paste area as seen in the image below:

To associate the fields, click on the arrow in the column headers. Select Associated fields to view the fields that can be mapped in the pasted customer data.

FIELD DESCRIPTION
F1148 Customer ID
F1577 Alternate cust number (ex: Gift cards)
F1149 Customer first name
F1150 Customer last name
F1172 Customer phone number
F1573 Email address
F253D Last change date from
F253E Last change date to

The columns that may be associated display the fields name and number.

Make the required selection and click on Execute to run the query: the pasted data will populate the grid.

Copy and paste for Gift cards creation

Manage module> Balancing>Gift card>Adjustments

This will bring up the Gift card -Adjustment Wizard. Click on the Menu pull-down and select Paste list from Excel

At this point you will fall into the Clipboard Paste Search window where you may paste your file in that area below the text "Paste data here" highlighted in yellow.

NOTE: Be sure that when you copy over your text file (ex: Excel) that you have a Header row as this is required in order to capture the details you desire.

Excel example Excel pasted into the Clipboard

Then you may click on the Execute action button to bring you to the follow "Gift card -Adjustment edit" window to create the Adjustment (Amount), and Comment columns. Once entered the desired details you will need to click on the "Appy" action button to confrim and process your data entrie(s).

Copy and paste for Batch data

NOTE: The copy and paste functionality can be used with existing batches only; it becomes available in the Menu only after a batch is selected in the wizard.

In the Items section of the launcher, click on the Batches sub-menu to create future price, cost, POS and main item batches:

Click on New: the pull-down function will display the batch types by section.

Pull-down options in batch start windows

Batches can also be created from the Prices, Costs, POS information and Main item wizards. This allows creating a new batch without having to go back to the Start window.

The following pull-down options are available from the wizards:

Price batch wizard Cost batch wizard
   
POS info batch wizard Main item batch wizard

NOTE: The Copy and Paste items option is not be available for use unless a Batch and Batch type are selected in the wizard. It is therefore greyed out, as seen in the image below:

Copy and paste in the price batch grid

In the Price batch wizard, select the desired price batch: the Copy and Paste items functionality is now available in the Menu pull-down option.

Click on the Copy and paste items option: the Clipboard paste search window opens, allowing to paste (CTRL+V) the copied data.

NOTE: There are minimum fields required to be mapped before executing this function. When executing before mapping the data, the system will prompt an error message stating that the minimum required fields to process are not mapped. The required fields will be displayed in an error pop-up.
Here is an example of a price batch error message:

Below are the required fields to be mapped prior to executing copy-paste for price batches:

FIELD DESCRIPTION
F01 UPC code
F30 Price
F31 Price qty

When the copied data is displayed, the columns fields need to be mapped. Click on the arow in the column headers, then on Associated fields.
This opens a list of the available fields that may be mapped in the pasted item data.

Once the association done, the field number linked to each column is displayed in brackets next to the column name.

Click on the Execute button to run the query and the grid will be populated with the pasted data.
