Email Set-up for eSuite Hybrid PhantomJs scheduler instruction.

In order to create tasks in esuite that involve Email exchanges, this setup is necessary.

1. Installation/eMail setup

To install XchFTP option, follow these steps:

  • Fom the Launchpad application, under the Options tab of the Company Setup window, install the XchFTP option like you usually do.
    • The XchFTP option must be installed on the back office server.
  • To have the XchFTP working, there is some required setup as they are not automatically done when installing.
    1. From SMS PRO, From the menu listed at the top, click on System.

2. Installation/eMail setup in SMS PRO Ini preferred screens window

From SMS PRO:

  1. Click on Execute from the main header a drop list will display
  2. Click on Edit config file located towards the bottom of the drop list
  3. On the left-hand side for the Ini preferred Screens window locate Other Ini
  4. Click on Other Ini pull-down button, will display a drop list
  5. Click on General Email Setup
  6. This will open a window called, SMS Ftp Wizard - SMTP_CONGIF

  8. Enter the same information as you have done previously.
  9. Then Test the email set up to ensure you receive an email, by clicking on the Test button.
  10. Click on the Save button to ensure the final setting is saved.
