eSuite - Deploy Template

eSuite > Tools > Deploy

Information

To be revised and update once this feature is available in eSuite.
 Raluca, 2019-10-15

The Deploy template job sends templates from the Head office (HO) to the stores and the backoffice server to the front-end workstations.

The Deploy template job deploys the following information:

  • Bitmaps
  • Receipt header
  • HTML buttons
  • HTML templates

To perform a Deploy template job:

  1. Click the arrow in the Tools entry in the Launcher to display its menu.
  2. Select Deploy: the Start window opens in the Deploy work area.
  3. Select Deploy template.

NOTE: The Deploy template job requires the right security level. Different security levels allow different accesses.
