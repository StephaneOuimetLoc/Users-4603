eSuite Manage - Overview

eSuite Manage

Information

This document describes the concept of eSuite Manage.

Concept

LOC SMS eSuite is a browser-based software consisting of several different modules that provide specific access to the same information as is contained in SMS and SMS Pro. This solution is more versatile, as it utilizes a web interface instead of a locally-installed application and means that eSuite can be accessed via back office workstations without having to have LOC SMS installed. Through eSuite, store personnel can log in and can perform everyday operations just like they would on a regular back office server or workstation.

LOC SMS eSuite main login page

Modules

The eSuite software includes several modules: Care, Manage, System, Buying, Selling and Gas.

While each module provides access to specific information, most of them share common entries that link to core information.

The eSuite manage module

The eSuite manage module provides tools for managing and deploying data. It joins tables (Operators, Customers, Items) and Functionalities (Batches, Multi Totals and Labels).

  • It can be accessed at http://[localhost:port]/esuite/man/index.html.
