eSuite - General Changes Overview

eSuite > Items > General Changes 

Information

This document describes the General Changes feature is located under the Items section of eSuite Manage/Batch/Care/Buy or Gas.

NOTE: The feature was formerly called Batch All Changes in SMS Pro (prior to eSuite release).

Concept

The General Changes (Here in referred to as GCL report) is a report showing all the changes contained in un-executed batches sitting in the instore batches.  It will display the current value for any items and the new value coming from the batch and the batch number.

Access General Changes

This entry is available in the following modules:

  • Care
  • Buy
  • Sell
  • Manage

To access the GCL report click on the Items entry and then select General Changes

You will be prompted with the General change log 'Wizard'

Enter your search criteria

Click execute to run the report and you will be presented with the report:

The table below briefly explains the GCL report grid and its functionalities.

NO. NAME DESCRIPTION
Date This is the start date of the batch which contains the change on each line
Item Code UPC number for which the change will modify
Batch number The batch number in which the change on that line is contained
Batch Type Batch Type
Details More details on which target is being affected and which price level, if applicable.
