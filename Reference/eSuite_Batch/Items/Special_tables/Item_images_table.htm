eSuite - Item Images Table

eSuite > Items > Master > Item list

Information

The Item images table allows associating images to items and use them in xStore (eStore and mStore).

More than one image can be associated with an item, provided that each image has different characteristics from the other.

Displaying the Item images table

In the Launcher, click the arrow in the Items entry to display its menu and then:

  1. Click on Master: the Start window opens in the Items work area.
  2. Click on Item list: the Wizard tab opens, displaying the Items - General list window.
  3. Enter the search criteria and click Execute: results will display in a record grid, in the Items tab of the same window.
  4. Select a record in the list.
  5. Click Edit in the bottom menu to display the options.
  6. Select Special items, then Images: the Item images table opens, overlapping the work area.
  7. Click on Add in the toolbar to enable the fields and add a new record.

Layout

Content

Here is the description of the Item images table fields and their functionalities:

FIELD DESCRIPTION
Main section
UPC The UPC field contains the code of the item to which images will be associated.
Types The Types field contains the type of image (Main, Plano, Nutri) to be attached to the item.
Required field.
Angles The Angles field displays the angle in which the image will be attached to the item.
The available options are Main, Front, Left, Right, Top and Bottom.
Required field.
Size The Size field displays the size of the image.
The available options are Thumb, Detail and 50x50.
Required field.
Date The Date field contains the date on which the image(s) will be associated with the item.
Image The Image field contains the image path in the local machine.
It works with the Path field.
It is a required field.
Browse The Browse button allows selecting the image file.
Path The Path field contains the image path in the server.
It works with the Image field.
It is a required field.
Item image grid
Type column The Type column contains the types of images associated with the item. 
Angle column The Angle column contains the angles of the images associated with the item.
Size column The Size column contains the size of the image associated with the item.

Associating images to an item

More than one image can be associated to an item.

  1. Select an item in the item list.
  2. Open the Item images table.
  3. Click Add in the toolbar.
    The UPC of the selected item is added to the table and the fields are enabled:

  4. Select the type, angle and size of the image.
  5. Add a date.
  6. Click on Browse and select the image file.
    The Image field will diplay the image path in the local machine.
  7. Save the changes: the Path field displays the image path in the server.

Repeat steps 3 to 7 to link further images to the item. Make sure to select different values fo each of these images.
