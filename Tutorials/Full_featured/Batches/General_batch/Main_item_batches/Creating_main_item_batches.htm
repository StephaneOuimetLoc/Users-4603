eSuite - How to Create Main Item Batches

eSuite > Items > Batches > Main item

Creating main item batches

Batches are used for managing changes on multiple items. They may be main item batches, POS batches, price-related batches (future price, sale, TPR, instore, redeem, general price) or cost-related batches (future cost, allowance, rebate, discount, cost break, bill back, general cost).

The eSuite software also allows creating open batches that mix batch types and delete batches for removing groups of items.

This document focuses on how to create main item batches and how to populate them with items through the Main item entry in the Batches wizard.

Creating main item batches from the Main item entry

Batches in eSuite are created through the Batch header table, which contains the batch type, its unique number and the general batch information.

To create a main item batch:

In the launcher, click the arrow in the Items entry to display its menu and then:

  1. Click on Batches: the Start window opens in the Batches work area.
  2. Click on Main item to open the Main item batch wizard.
  3. Click on Add batch to open the list of item-related batch types.
  4. Select Main item: the Batch header table opens on the desktop.
    The following values are displayed: Batch ID, target, source, start date and time, end date and time, batch type and action.
  5. Fill in all necessary fields.
  6. Click Save in the toolbar to confirm the creation of the new batch.
    The Batch header table closes.
    The Main item batch wizard displays the new batch number and type.

The Execute button provides access to the list of items and all the tools for adding items to the new batch. See below for details.

Adding items to the batch

In eSuite, batches may be populated with multiple items at once or by adding one item at a time. Both aproaches are described below.

Adding a group of items to the main item batch

Batches in eSuite may be populated with multiple items at once. This is a fast and efficient method when the objective is to change basic item information, such as the future description.

Follow these steps:

  1. Open the Main item batch wizard.
  2. Click in the Batch field: the selection window opens on the desktop, displaying the list of main item batches.
  3. Select a batch and click OK to close the selection window.
    The wizard displays the ID and type of the selected batch.
  4. Add any other necessary search criteria.
  5. Click Execute to display the list of items.
  6. In the Long description column, make the necessary changes for the items that will be part of the selected batch.
  7. Click Save in the toolbar.
  8. To view the items that are part of the batch, go back to the wizard and check the Show only items into the batch field:
  9. Click Execute to list the items that are part of the batch.

Adding items one by one

Batches in eSuite may be populated by adding items one by one. This approach may take longer than adding multiple items at once, but it helps changing other item information than the basic one.

Follow these steps:

  1. Open the Main information batch wizard.
  2. Click in the Batch field: the selection window opens on the desktop, displaying the list of main item batches.
  3. Select the batch to add items and click OK to close the selection window.
    The wizard displays the ID and the type of the selected batch.
  4. Add any other necessary criteria.
  5. Click Execute to display the list of items.
  6. Select the item that will be added to the batch.
  7. Click Edit in the bottom menu.
    Select Item batch to display all the types of item-related future tables:
  8. Select Main item: the Future item table opens in a new window.
  9. Click Add in the toolbar to display the item information.
  10. Click on Pin in the toolbar: the future table will open on the desktop, next to the list of items.
  11. Make the necessary changes and click Save.
    The item containing the new information is added to the batch.
  12. Select another item in the list. This will:
    • Add the item to the current batch.
    • Refresh the future table with the information of the selected item.
  13. Make any necessary changes and click Save.

Repeat these steps for each item that will be part of the batch.
