About the SMS Member Help File

Use and search tips from the SMSMembers and SMSUsers help files. As these files contain quite a bit of information, the enclosed tips will help you find what you are looking for.

Information Access Methods

Contents Index Search

The easiest way to find a topic is to look it up in the table of contents tab. The contents tab includes all of the articles in the help file grouped by application or option categories. The only exception to this is the Training Documents category, which is grouped by level and subject.

When looking for information related to training, the table of contents can be used to find a specific session. In the example below, the Level 1 Installation and Configuration class is found by opening the Training Documents category in the table of contents and then opening one of the various training levels by clicking the + sign to expand the topics. Clicking a topic opens the corresponding document in the right hand side of the CHM viewer.

If the specific category section in which an article is located is not known, the topic can be looked up in the Index tab, which lists all of the title topics in alphabetical order.

The help files include a Search function to find help topics using keywords or phrases. The search function can also use search parameters to help narrow down topics that might be of interest.

Search parameters

AND parameter (default)

When more than one keyword is entered in the search box, the default behavior is to search for ALL the keywords appearing in topics, in other words, entering two or more search terms uses AND between them, which causes the search function to return only topics that include ALL of the search terms.

In the example below the search words inventory AND reports return 59 topics that contain both the words "inventory" and "reports" in the topic.

OR parameter

The OR parameter is used between search terms to look for ANY topics that include the keywords whether they appear together in the same topic or not.

In the example below the search words inventory OR reports return 462 topics that contain either the words "inventory" or "reports" in the topic.

Exact phrase

To search for an exact phrase place the keywords in double quotes. This way the search function will return fewer results as it will find ONLY topics that include that exact phrase.

Search results without quotes searches for both words appearing in a topic.

Search results with quotes searches for that exact phrase appearing in a topic.

Highlighting search terms

To help look for search words within the topics, the help file can highlight the keywords used in the search in the topics once they are opened.

To enable/disable the search highlight follow the steps below:

  1. Click Options menu
  2. Click Search Highlight On or Search Highlight Off
  3. To refresh the view and see the change to the highlighting open another topic in the search results.

Sorting

To change the sort order of the search results list, click on the column header.

Advanced Search options

At the bottom of the search tab there are several options to refine searches.

  • Search previous results: strong Searches for words within the last search results, this is handy for combining search words that would be otherwise difficult to do.
  • Match similar words: Searches for words with similar spellings, e.g.: search for report will get results for reporting, reports, etc...
  • Search titles only: Searches for keywords only in the title of topics.
