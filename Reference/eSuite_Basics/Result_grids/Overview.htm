eSuite - Grid overview

Table of contents

Information

Grids in eSuite display record information. They provide a complete set of features that cover organizing, filtering, sorting and resizing content.

Layout

The grid layout is identical across modules. It includes the following elements:

  1. Checkbox column
  2. Column headers
  3. Column contents     
  4. Rows

Content

This section focuses on the grid elements and on the actions they trigger.

The checkbox column

The checkbox column is located along the left side of the grid. It contains two types of checkboxes:

  • Select/Unselect all rows
    Located at the top of the checkbox column.
    Selects and unselects all rows on the current page.
    Mostly used in subqueries and linked queries.
  • Select/Unselect row
    Located on the left side of each grid row.
    Selects and unselects the row.

The column headers

The column headers contain descriptive text about the grid column. In eSuite, they can refer to one column (single-level header) or to several columns grouped under one header (multi-level header).

The headers provide the following features that allow column customization:

  • Resizing
  • Reordering
  • Showing/Hiding columns
  • Sort in ascending/descending order
  • Multi-sorting

The column contents

The columns display grid contents in vertical divisions.

The rows

The rows display grid contents in horizontal divisions separated by lines.
