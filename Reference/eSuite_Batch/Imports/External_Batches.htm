eSuite System - External Batches

eSuite System > Tools > Links

Information

The external batches screen is used to view/edit External batches.  It is accessible through the Links section in the System Module.  An external batch is composed of two parts.The first part is the batch header in the HEADER_DCT table.  The batch header tells eSuite what to do with the data in the batch.  The second part is the batch data which is stored in a temporary database table which gets automatically purged.

 

Function buttons in the Grid

1) Edit - The edit button will allow you to modify the batch header, that look like this when you click it:

You can place the batch header in 'WAIT' state if you do not want the batch to be executed until your manually remove the WAIT flag in the 'State' drop down box.

2) Open List - This is a view into the data of the batch.  It shows you each row of data that the batch contains.  Some fields in this grid are editable.  You can manually add/remove data into the grid as well.

3) Changes - This is a comparision of the data in the batch(Incoming) to the same record in its respective item table. Current values(old) and new values are shown so you know exactly what the item record will be updated with.

4) Execute - The selected batch(s) will be excuted. 

Confirm the execution date and click 'Confirm'
