eSuite - How to Create and Apply Price Change Using Like Codes

eSuite > Items > Item auxiaries > Like Codes

Information

The Like Code is used to facilitate items maintenance. When a change is required in the Price record of an item that has a Like code, the system can change the price record of all other items having the same Like code. Ex: Juice boxes (3 pack X 750 ml) as you seen in stores they have different flavours such as Apple, Strawberry & Orange.

The Like code is a 13 digit character that is used to group items. The Like code can be found in the Price table under the General tab.

Creating a Like Code to be selected from the pull down option:

A Like Code can be created 1 of 2 ways, see below:

1. Click on the Items section in the launcher menu

2. Click on the Auxiliary table in the sub-menu of the Items section (opening the Like Code List window)

3. Click on the plus icon (opening the Like code Edit window)

4. Enter the 13 digit Like code you wish to use and the description and click on the Checkmark icon to save the data entered. Once saved the Checkmark will become greyed out and you may close the Price Edit window by using the "X" on the top right-hand corner.

5. Once the Like code Edit window is closed, it will bring you back to the Like Code List grid window. In order to see the new information entered you must click in the refresh icon located at the buttom left-hand corner.

6. This will then update the listing in the Like Code List as seen here below, now displaying the Code and Descriptor "0000000000087 & Lays Chip" columns.

7. When clicked on the Like code pull-down iocn, inside the General tab from the Price table of the item.

8. These pre-created Like Codes will become selectable from the Like code search window.

9. Select the desired Like code and click the "OK" button and it will poplulate the item's Like code field. In order to save this new assigned Like code, click on the checkmark icon to save.

10. Once saved the Checkmark will become greyed out and you may close the Price Edit window by using the "X" on the top right-hand corner. Bringing you back into the Items - General List grid, clicking back onto the item you will then notice the Like code in the Context Tools under the Like code: section.

To manually enter a Like code in the Price table of the item, follow these steps :

NOTE: For this example We will used the "Power Juice" Apple, Strawberry & Orange items.

1. Click on Items section in the launcher

2. Click on Master in the sub-menu (open the Wizard Items - General list window)

3. Press the Execute button or enter the item UPC code of the item desire

4. Select the item to assign a Like code (using the check box). Note: The Like code in the Context Tools is blank for the selected item.

5. Click on the Edit button & from the pull-down mouse over the Master Tables & then click on Price Master table

6. Or from the grid, click on the Price table of the item and press F3 (both ways will bring up the Price Edit window).

7. In the Price Edit table, manually enter the 13 digit Like code of your choice. In order to group the other similar items together, the same Like code must be assigned to the other Items.

8. Once entered the 13 digit Like Code, you must click in on checkmark icon to save the data. Once saved the Checkmark will become greyed out and you may close the Price Edit window by using the "X" on the top right-hand corner.

9. This will bring you back into the Items - General list grid, must click the refresh icon lower left hand corner next to the Item quantity in order to see the updated Like code detail in the Context Tools.

10. Once the Item - General list grid is updated, select/click on the line item with the Like code entered, this new information will be displayed in the Context Tools under the Like code: section.

Like Codes options and definitions:

When applying changes on items that have a Like code, you will need to understand there are 3 areas that displays information in the Like code section located in the Context Tools.

Like code: System will display the like code assigned to the item.

 

Apply: this section offers 3 modes in the pull-down:

1. Prompt [Default] - System will always prompt a pop-up confirmation window asking, "Apply changes using like code?" with "Apply" or "Not Apply" buttons as options.

2. Always apply - System automatically updates other items with the same Like code and System will no longer prompt a pop-up confirmation window again while user is currently logged into eSuites.

3. Don't Apply - System will not Apply any changes using Like code. It is as if the function is turned off.

 

Actions: this section offer 4 modes in the pull-down:

1. Changes only [Default] - Changes only the Price fields altered by the user from the current items to other items with the same Like code, leaving all other fields in the Price table, for the Like code as they were.

2. Changes only and create as needed - Changes only the Price fields from the current items to other items with the same Like code, leaving all other fields in the Price table, for Like code items as they were. In the case where a Price table does not exist, its creates the Price table entry.

3. Full record - Overwrites the entire full Price table record for each item having the same like code. This default may not be changed. Like codes are used to manage items that are the same. In the majority of cases it is the entire record that should be copied.

4. Full record and create as needed - Overwrites the entire full Price table record for each item having the same like code. This default may not be changed. Like codes are used to manage items that are the same. In the majority of cases it is the entire record that should be copied. In the case where a tables does not exist, its creates the table entries.

Applying changes on Items with Like Codes

NOTE: For this example we will use the "Power Juice" items and have assigned the same Like Code to all 3 flavours (Apple, Strawberry & Orange).

To change the Price of the Items, select 1 item with the same Like Code and click in the Price field of the item. (changing Price from $7.85 to $7.75)

Notice the "Apply:" is set to Prompt and the "Action:" is set to Changes only.

1. Click in the Price field, enter the desired amount (ex: 7.75) and press the Enter key. Once the Enter key has be pressed the line selection will fall below the initial selected item, this is normal behavoir.

2. In order to save the Price change entered, Click on the Checkmark icon.

3. This will then prompt the Like code window, asking "Apply changes using the like code?". The actions of Apply and Not Apply can be selected by clicking on their respective button.

4. After clicking on the Apply button a "Loading" sign will appear while the system makes the changes requested. The Items- General List grid then will display the new price changes for said items.

NOTE: Once these Price changes have be completed, must run the DEPLOY_CHG.sql script in order to have these new changes at the store level.
