eSuite - Kit Examples

eSuite > Items > Master

Information

The Kit table is used to join multiple items and sell them as one item. It is used to manage kits that contain different items. Kits are considered items since they have their own UPCs; the major difference is that they include other items. A kit may contain an unlimited number of items. Each of these items must already exist in the system.

A kit links the big format of an item to its smallest unit. For example, a carton or cigarettes is linked to the pack and not the opposite.

Before creating a kit, you must adjust the inventory of the item by transferring the quantity in inventory for the big format (carton) into the smallest unit (pack). Once the kit has been created, the inventory of the smallest unit (pack) will be affected when the big format (carton) is purchased or sold.

After having created the first kit in the system, eSuite must be reloaded. The software checks if there are any kits in the system and if not, it disables the kit update engine to speed up the transaction process.

The Kit table is used to manage the kit. For kits that contain many of the same item, like a carton of cigarettes, use the Split code and Split quantity fields of the Cost table.

Example

 A basket that contains 1 bottle of shampoo, 1 bottle of conditioner and 2 soaps

The shampoo, the conditioner and the soap must already exist in the database.

  1. Create the basket item.
  2. In the item list, select the basket item. Click Edit in the bottom menu, then Special items and Kit: the Kit table opens on the desktop.
  3. Click Add in the toolbar to add a new item to the kit.
    Keep 1 in the Level field.
    Add the shampoo item in the Linked UPC field.
    Type 1 in the Multiple field.
  4. Click Add in the toolbar to add a new item to the kit.
    Keep 1 in the Level field.
    Add the conditioner item in the Linked UPC field.
    Type 1 in the Multiple field.
  5. Click Add in the toolbar to add a new item to the kit.
    Keep 1 in the Level field.
    Add the soap item in the Linked UPC field.
    Type 2 in the Multiple field.

  6. Click on the Generate ratio button to calculate the items ratios.
  7. Save the information.
