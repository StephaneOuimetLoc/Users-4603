Inventory Count

eSuite Buy > Inventory > Count

Information

This document describes how to perform the different required operations in an inventory multi-count document.

The possible operations are:

  • Create and edit a multi-count document
  • Add items to the multi-count document
  • Remove items from the multi-count document
  • Manage quantities into the multi-count document
  • Canceling a multi-count document
  • Suspend or close the multi-count document

All count documents are now created as multi-count. Therefore, the count option is no longer available.

In a multi-count document, the first count replaces the inventory quantity with the value entered.
Any following count for the same item on the same date is incremental.

Generating an inventory count sheet by sub-department

In the start menu, select count by sub-Department

Set the required criteria and filter the records then Execute

It will generate a count sheet

 

Creating a multi-count document

To create a multi-count document:

  1. In the Launcher, expand Inventory
  2. Select Count
     The Start window opens on the desktop

  3. Click New
     The New document window opens

  4. Select the inventory date and specify the vendor and/or sub-department or leave both blank
  5. Click Create to confirm the creation of the multi-count document

When a multi-count document is created:

The status is Open, the transaction number and date are shown in the Context Tools 

The new multi-count document is listed in the Open document grid that is available from the Count and All options in the Start menu.

Adding items to the multi-count document

Once a multi-count document is created, items can be added to it. At this point, the items, Lines and Header are no longer greyed out.

To add items:

  1. In the Start window, select items, fill in the criteria for the required items and Execute

    Or
     Select Items - All to display all items

  2. The Case, Unit, Weight and Cost fields are editable directly from the grid.
    Double-click in any editable fields in the under the Entry column and add the quantities then press Enter.

  3. Once the values are entered, save the changes
    The new values are displayed in the Daily column.

Editing the multi-count document

Before closing the multi-count document, it is possible to review and edit the information in it. This is done through the document window.

  1. Click on the hamburger icon   in the Context Tools and select show lines

  2. Modifications can be made directly in the Lines grid

    Or
    From Lines under the current Transaction by double clicking on a line:

    It will open a up a new Line edit window where to make the modifications

  3. Once the modifications are complete, then save  the changes.

Voiding items

Items can be removed from the multi-count document

  1. In the Lines window, select the item to remove and right-click to display the list of options and select "void" or you click on "void" from the bottom menu.

  2. Click Yes on the confirmation message:

Canceling a multi-count document

An open multi-count document can be deleted from the Context Tools:

  1. From the Context Tools, click on the hamburger icon   and select Cancel the document
  2. Click Yes to confirm

Suspending a multi-count document

Opened multi-count documents can be suspended and edited at a later time.

  1. To suspend a document, click on the hamburger icon  from the Context Tools
  2. Select Suspend the document

Closing a multi-count document

  1. To close a document, click on the hamburger icon  from the Context Tools
  2. Select Close the document
  3. Click Yes to confirm

Note: It is not possible to reopen a closed multi-count document.
