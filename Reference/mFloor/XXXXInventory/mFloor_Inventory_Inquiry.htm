mFloor - Inventory Inquiry

mFloor Procurement

Information

Inventory inquiry is used to query an item to get its current inventory.

To search for a inventory item:
  1. Type the item code in the Enter item code box at the top of the screen.

    or
  2. Tap or click and scan the item code

    or
  3. Tap or click and a list of items appears, choose an item by tapping or clicking it. In the case of a long list swipe or click and drag to scroll the list.
  4. The current inventory on hand and case size will be displayed along with the POS description.
