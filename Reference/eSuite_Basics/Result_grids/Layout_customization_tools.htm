Information

The eSuite grid allows you to customize its view through the column headers.

Additional customization tools are available in the grid's bottom menu.

Customization tools

The customization tools are available from the following elements in the eSuite grid:

  1. Headers
    Provide tools for resizing, reordering and sorting columns.
  2. Drop-down menus
    Provide tools for sorting columns and grouping rows.
  3. Cascading windows
    Provide tools for showing and hiding columns to the grid.
  4. Bottom menu
    Provides tools for setting the limit of records in the grid, refreshing the view, establishing the page size and navigating the pages.

Resize columns

You can resize columns with the resize cursor .

Tool properties:

  • Becomes visible when you hover the mouse over the header borders.
  • Allows column resizing by dragging the header borders.

Reorder columns

You can reorder columns by dragging their headers.

 

Tool properties:

  • Renders columns reordarable when you click on the header.
  • Allows dragging.
  • Shows a red icon that indicates you to keep dragging.
  • Shows a green checkmark that indicates you to stop dragging.

NOTE: Drag the multi-level headers to reorder all the columns grouped underneath.

Sort columns

You can reorganize columns with the sorting tool. This will sort one column at a time.

Tool properties:

  • Can be accessed from the header drop-down menu.
  • Allows sorting column contents in ascending order.
    Column header will show an arrow pointing upwards.
  • Allows sorting column contents in descending order.
    Column header will show an arrow pointing downwards.

NOTE: You can also sort columns by clicking on the header.

Sort multiple columns

You can reorganize the column contents with the Multi-sorting tool. This will sort multiple columns at a time.

Tool properties:

  • Can be accessed from the header drop-down menu.
  • Allows sorting multiple columns in ascending order.
    Column headers will show an arrow pointing upwards.
  • Allows sorting multiple columns in descending order.
    Column headers will show an arrow pointing downwards.

Reset sorting

You can return to the default sorting parameters with the Reset sorting tool.

Tool properties:

  • Can be accessed from the header drop-down menu.
  • Resets the sorting parameters to the default values.

Group rows

You can group rows by data field.

Tool properties:

  • Can be accesssed from the header drop-down menu.
  • Groups rows by data field.
    Data is retrieved from the tables that correspond to the records in the grid (item tables, customer tables and so on).

Show and hide columns

You can show columns in the grid or hide them with the checkbox tool.

Tool properties:

  • Can be accessed from the header drop down menu.
  • Opens cascading windows with checkboxes.
  • Allows checking columns and add them to the grid.
  • Allows unchecking columns and hide them from the grid.     

Set the result limit

You can change the total number of records that you want to see in the grid.

Tool properties:

  • Can be accessed from the Set result limit button in the bottom menu.
  • Opens the Result limit window.     
 

The following fields are available in the Result limit window:

FIELD DESCRIPTION
Top result records Sets the maximum number of results that can be viewed in the grid.
Default value: 10000.
Top pivot records Sets the number of pivot records that can be viewed in the grid.
Default value: 2000.
Server timeout Sets the time that the query operation can take before the SQL server times out.
Default value: 40 seconds.
OK Confirms the changes.
Cancel Cancels the changes.
Restore default parameters Restores to the default parameters.

Refresh the view

You can refresh the grid view to display the latest changes.

Tool properties:

  • Can be accessed from the grid's bottom menu.
  • Refreshes the data in the grid.
  • Displays the latest changes in the records or in the grid view.

Set the page size

You can set the number of records that you want to see on one page of the grid.

Tool properties:

  • Shows the number of records displayed in the active page of the grid.
  • Gives access to a drop-down list with display options (20, 50, 200, 800 records per page).

Navigate in grid

You can navigate in the grid page by page or from a desired page to another.

Tool properties:

  • First page navigates to the first page of the grid.
    Disabled when grid shows the first page.
    Enabled when the grid shows any other page.
  • Previous page navigates to the previous page of the grid.
    Disabled when grid shows the first page.
    Enabled when the grid shows any other page.
  • Page number switches to different pages in the grid.
    Allows you to enter page numbers and displays them when you press Enter on the keyboard or when you refresh the grid.
  • Next page navigates to the next page of the grid.
    Disabled when grid shows the last page.
    Enabled when the grid shows any other page.
  • Last page navigates to the last page of the grid.
    Disabled when the grid shows the last page.
    Enabled when the grid shows any other page.
