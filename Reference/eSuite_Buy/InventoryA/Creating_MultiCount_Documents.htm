Multi-Count Documents

eSuite Buy > Inventory > Count > Count by Sub-Department

Information

This document describes how to perform the different required operations in an inventory multi-count document.

The possible operations are:

  • Create and edit a multi-count document
  • Add items to the multi-count document
  • Remove items from the multi-count document
  • Manage quantities into the multi-count document
  • Delete a multi-count document
  • Suspend or close the multi-count document

All count documents are now created as multi-count. Therefore the count option is no longer available.

In a multi-count document, the first count replaces the inventory quantity with the value entered. Any following count for the same item at the same date is incremental.

Generating an inventory count sheet by sub-department

  1. In the Start menu, select Count by Sub-Department.
  2. Set the required criteria and filter the records.
  3. Click Execute to generate the report.

Creating a multi-count document

To create a multi-count document:

  1. In the Launcher, expand Inventory.
  2. Select Count.
    The Start window opens on the desktop.
  3. Click New.
    The New document window opens.
  4. Select the Store and the Inventory date.
  5. You can specify the Vendor and/or the Sub-department or leave both blank
  6. Click Create to confirm the creation of the multi-count document.

When a multi-count document is created:

  • Its status is Open. The transaction number and date are shown in the Context Tools.
  • The new multi-count document is listed in the Open document grid that is available from the Documents and Documents - All options in the Start menu.

Adding items to the multi-count document

Once a multi-count document is created, items can be added to it. At this point, the Items, Lines and Header are no longer greyed out.

To add items:

  1. In the Start  window, select Items. Then enter the criteria the search for the required items, and click Execute.

    or
    select Items - All to display all items.

    The Case, Unit, Weight and Cost fields are editable directly in the grid.
  2. Double-click in any editable field, add the quantities, then press Enter.
  3. Once the values are entered, click .
    The new values are displayed in the Resume and Cost columns.

Editing the multi-count document

Before closing the multi-count document, it is possible to review and edit the information in it. This is done through the Document window.

  1. From the Start menu, select Documents - All.
    The list of multi-count documents are displayed in the document grid.
  2. Select a document, and right-click to display the list of options.
  3. Select Open.
  4. From the Start menu, select Lines - All.
    The content of the document is displayed.
  5. Select a line and click Edit.
  6. Modify the values and click .
  7. Close the window and click to refresh the values displayed in the grid.

Voiding items

Items can be removed from the multi-count document.

  1. In the Lines section of the Documents window, select the item to remove and right-click to display the list of options.
  2. Select Void.
    A confirmation message is prompted.
  3. Click Yes to delete the item from the multi-count document.

Deleting a multi-count document

Opened multi-count documents can be deleted from the Document window:

  1. In the Context Tools window, click .
    A confirmation message is prompted.
  2. Click Yes to delete the document.
    In the Opened documents grid, click to refresh the list; the deleted document is not longer listed.

Suspending a multi-count document

Opened multi-count documents can be suspended and edited at a later time.

  1. Open the Document window for the multi-count documents.
  2. Click in the Context Tools window to suspend the document.

Closing the multi-count document

  1. From the Start menu, click Documents - All.
    The list of multi-count documents is displayed in the grid.
  2. Select a document and open it.
  3. Click in the Context Tools window.
    A confirmation message is prompted.
  4. Click Yes to close the document.
    In the Opened documents grid, click to refresh the list; the multi-count document is no longer listed.

    NOTE: It is not possible to reopen a closed multi-count document.
