eSuite - Preferred Shopper Double Coupon Examples

eSuite > Customers > Shopper Levels

Information

Another kind of promotion allowed by the Shopper level table is an add-on value on the vendor coupons, i.e. an additional amount given over the vendor coupons values.

Add-on values are applied on all types of vendor coupons, including the electronic ones.
By default, the system considers the add-on values as store coupons.

Examples

  1. The vendor coupons are doubled
  2. The vendor coupons up to $1.00 are doubled
  3. The vendor coupons are doubled for a maximum discount amount of $2.00
  4. The amount of $0.50 is added to the vendor coupons values
  5. The amount of $0.50 is added to the vendor coupons value for a maximum discount of $1.50

Vendor coupons are doubled

Set up the Shopper level table information:

  1. In the Launcher, click the arrow in the Customers entry to display its menu.
  2. Click on Auxiliary tables.
  3. In the Start window, click on Shopper levels.
  4. In the list of shopper levels, select the shopper level for which the value of the vendor coupons will be doubled.
  5. Click Edit in the bottom menu to open the Shopper level table.
  6. Type 2 in the Coupon multiply field.

  7. Save the changes.

When a vendor coupon is scanned or entered manually, the system automatically looks into the Shopper level table to check if an add-on value can be given.

Examples:

  • For a vendor coupon of $1.00, the total reduction will be $2.00.
    Vendor coupon value Add-on amount Restricted add-on amount Total reduction amount
    $1.00 $1.00   $2.00
    $1.00 $1.00 --- $2.00
  • For a vendor coupon of $2.00, the total reduction will be $4.00.
    Vendor coupon value Add-on amount Restricted add-on amount Total reduction amount
    $2.00 $2.00   $4.00
    $2.00 $2.00 --- $4.00

The vendor coupons up to $1.00 are doubled

Set up the Shopper level table information:

  1. In the Launcher, click the arrow in the Customers entry to display its menu.
  2. Click on Auxiliary tables.
  3. In the Start window, click on Shopper levels.
  4. In the list of shopper levels, select the shopper level for which the value of the vendor coupons will be doubled.
  5. Click Edit in the bottom menu to open the Shopper level table.
  6. Type 2 in the Coupon multiply field.

  7. Type 1.00 in the Maximum value field.

  8. Save the changes.

When a vendor coupon is entered manually or scanned, the system automatically looks into the Shopper level table to check if an add-on value can be given.

Examples:

  • For a vendor coupon of $0.50, the total reduction will be $1.00.
    Vendor coupon value Add-on amount Maximum coupon value Total reduction amount
    $0.50 $0.50 $1.00 $1.00
    $0.50 $0.50 --- $1.00
  • For a vendor coupon of $1.00, the total reduction will be $2.00.
    Vendor coupon value Add-on amount Maximum coupon value Total reduction amount
    $1.00 $1.00 $1.00 $2.00
    $1.00 $1.00 --- $2.00
  • For a vendor coupon of $2.00, the total reduction will be $2.00 (instead of $4.00).
    Vendor coupon value Add-on amount Maximum coupon value Total reduction amount
    $2.00 $2.00 $1.00 $2.00
    $2.00 --- --- $2.00
