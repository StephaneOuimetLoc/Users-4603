eSuite - Grid general settings

eSuite > Launcher entry > Start page > Wizard page > Grid > General settings

Table of contents

Information

The eSuite grid settings allow you to customize and save layouts in private or public views.

Layout

The setting tools are located in the grid toolbar.

Settings

The General settings button in the toolbar opens the list of the available settings.

SETTING DESCRIPTION
Keep layout locally - OFF Toggle switch
Ensures that the design changes in the grid cannot be saved in a layout.
Keep layout locally - ON Toggle switch
Ensures that the design changes in the grid can be saved in a layout.
Marks the Layout menu in the grid toolbar with an orange dot .
Saved layouts Button
Gives access to the list of saved grid layouts.
Opens the Grid layout setting window.
Restore to default layout Button
Restores the grid layout settings to the default values.
Removes the orange dot from the Layout menu in the grid toolbar.

The Grid layout setting window

The Grid layout setting window opens when you click on the Saved layouts button in General settings.

The following elements are available:

ELEMENT DESCRIPTION
Grid layout setting Text field
Shows the window's name.
Close Button
Closes the window
Layout Combo box
Shows the names of the saved layouts.
Allows you to enter the name of the new grid layout.
Check to save as "Public" profile Checkbox
Makes the saved grid visible to all operators.
Enables the Show button. 
Uncheck to be "Private" Checkbox
Makes the saved grid invisible to all operators except for the current one.
Disables the Show button.
Save Button
Saves the changes in the grid layout.
Show Button
Allows you to select a grid layout in the list.
Closes the window.
Refreshes the grid view to show the selected layout.Disabled by default.
Enabled when Check to save as "Public" profile is checked.
Delete Button
Deletes the selected grid layout.
