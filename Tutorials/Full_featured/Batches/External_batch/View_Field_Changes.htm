View Field Changes

To view the changes that are stored in the External batch, open the External Batches grid

1.  From the Batches module, select Imports in the launcher

2.  Select External Batches from the Links start widget

3.  Click Execute on the List batch Wizard

4.  Once you have the External Batches list grid, select the batch and click Changes at the bottom toolbar

5.  The External Batch Changes grid will open displaying the all the data contained inside the batch.  The Field name, Old value and New value is where you can see the new data that will be inserted if the batch is executed.
