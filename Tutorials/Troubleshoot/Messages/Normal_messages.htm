Manage Normal Messages

eSuite System > Messages

Information

The eSuite software keeps an audit trail of some processes, whether or not errors have occured. Normal messages can be the result of a successful process or electronic messages that may contain file attachments.

The normal message list builds up over time and requires regular maintenance. Depending on the number of automated procedures, services running and points of sale, the list can grow quite rapidly.

Viewing normal messages

To view a normal message:

  1. In the launcher, click the Messages entry: the Start window opens in the Messages work area.
  2. Click on Normal: the Files grid opens, displaying the list of normal messages.
    For details on the fields and functionalities of the grid, see the Messages topic.
  3. Select a file and double-click on it to open the message.

Layout

*Click on a field to get more information.

Content

The View file contains the normal message header, with general information on the file and the message content, with details on the file.

Below is the description of the normal message fields and functionalities.

FIELDS DESCRIPTION
Remove archive bit Under develoment.
Download The Download field allows saving the normal message on the local machine.
Filename The Filename field displays the name of the file.
Date The Date field displays the date and the time of the normal message.
Attributes The Attributes field contains the properties of the normal message:
A= Archive
R=Read-only
To The To field contains the node number of the PC to which the normal message is sent.
From The From field contains the node number of the PC that sends the normal message.
Subject The Subject field contains the topic of the normal message.
Message content The message content shows details on the normal message topic.

Printing normal messages

To print a normal message:

  1. Open the View file window for the normal message.
  2. Click Print in the toolbar.

  4. Click Print to set the parameters and complete the printing.

Deleting normal messages

Normal messages may be deleted from the Files grid:

  1. In the launcher, click the Messages entry: the Start window opens in the Messages work area.
  2. Click on Normal: the Files grid opens, displaying the list of normal messages.
  3. Select the message to delete.
  4. Click Delete data in the toolbar.

  6. Click Refresh in the bottom menu of the grid to confirm that the normal message has been deleted.
