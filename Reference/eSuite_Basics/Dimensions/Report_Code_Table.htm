eSuite - Report Code Table

eSuite > Dimensions

Information

The Report code table is a way for users to set up alternate groups for items, based on different aspects of a particular set of items. The Report code table is useful to regroup items for report purposes.

The Reports table provides a report code parameter in some item reports. Assigning report codes to items simplifies the item report procedures for mixed item reports. The report code is extremely useful for gathering unrelated items such as seasonal ones in a single report.

EXAMPLE: Many items are only sold during the Christmas season. Linking seasonal items with a report code can make it easy to view the sales for these items only and to compare them with regular sales.

The items are linked to a report code using the Report field of the Main item table.

Displaying the Report code table

In the Launcher, click the arrow in the Dimensions entry to display its menu and then:

  1. Click on Report Codes: the Report Code window opens in the RCode work area.
  2. Click on List: the RCodes tab opens in the same window, displaying the Report Code List.
  3. To modify an existing report code, select it in the list and double-click on it, press the F3 key on the keyboard or click Edit in the bottom menu: the Report code table opens, overlapping the work area.
  4. To create a new family record, click on Add in the bottom menu of the grid: the Report code table opens, overlapping the work area.

Content

Here is a description of the Report code table fields and their functionalities:

FIELD DESCRIPTION
Number The Number field contains the report code number.
It is a required field used to uniquely identify records in the Report codes list.
Description The Description field contains the report code description.
It is not a required field; however, a descriptor should be entered as it helps identify the report.
