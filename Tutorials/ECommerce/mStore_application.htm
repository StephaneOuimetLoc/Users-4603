Information

The mStore application is the mobile version of the eCommerce solution that allows users to purchase from a mobile device to LOC SMS, without having to maintain two different systems. For the desktop and tablet version, refer to the eStore application.

There are multiple advantages with the mStore application since it is linked to LOC SMS and uses the same database, structure and features. All orders created in mStore application are retrieved, processed and closed at the store level or the warehouse.

The mStore application is built with the Sencha framework and is classified as a web mobile application more than a typical mobile application. 

According to the customer's needs, the mStore application can be customized by using the Sencha tools provided with SMS. A license is required to use Sencha Architect.

The Login page

The Login page allows the user to logging into the mStore application, with there user name or email and password. Visitors must create an account or use 'Login as guest' to navigate in the mStore application. Here are the features of this page based on the demo of LOC Software:

  • User / Email Address
  • Password
  • Save password for next time
  • Login
  • Forgot your password
  • Login as guest (navigate without an account)
  • Create an account

Create an Account page

The Create an Account page allows the visitor to create an account to navigate in the mStore application. Here are the features of this page based on the demo of LOC Software:

  • First Name
  • Last Name
  • Phone (Home phone)
  • email address
  • Password and confirmation
  • Default store (need to select)
  • Email me deals and recommendations
  • Original Captcha or Google "I'm not a robot" Captcha (depending on the configuration)
  • Refresh captcha or Google reChaptcha
  • Have an account? Sign in
  • Create account (confirmation)

Shopping page

The Shopping page, accessible by using the magnifying glass icon or Show all departments, allows the user to see items by department or all items with the Show all items option. Here are the features of this page based on the demo of LOC Software:

  • Search field (UPC code, product description and brands)
  • Scan product (barcode icon)
  • Show all items
    • Sort (such as Product, Price, Popularity)
    • Filter (such as Brands, Preferences and Other)
  • Departments

To create and manage the classification table and the Item classification, refer to the Classification topic

Basket page

The Basket page displays the details of the current transaction to proceed to checkout. Here are the features of this page based on the demo of LOC Software:

  • Proceed to checkout
  • Search cart (sorting by different criteria)
    • Line
    • Department
    • Amount
    • Description
    • Location
    • Brand
  • Each product
  • Basket Summary
  • Continue shopping
  • Save basket
  • Cancel Order

When there is no transaction started, the Basket page accesses the list of suspended transaction allowing to resume a transaction.

NOTE: When the order is received in the selected store, an order confirmation email is sent to the user.

All Orders page

The All Order page gives access to the user's older orders and it is also possible to reorder items from this page. Here are the features of this page based on the demo of LOC Software:

  •  Search cart (sort by)

    • Filter orders still in standby, just submitted or closed

Coupons page

The 'Coupons' page displays all the targeted coupons available for the user and all the public coupons. All the targeted and public coupons must be clipped before being used in an order. Here are the features of this page based on the demo of LOC Software:

Points & Clubs page

The Points & Clubs page displays the name of each point program in which the user participates as well as their balance and the items that can be redeemed with points for each point program. Here are the features of this page based on the demo of LOC Software:

  • Points programs Name & Balance
  • Eligible items for redemption and number of missing points

Scan Product field

The Scan Product field, when used for the first time, the mStore application requires access to the camera of the device. When the user authorizes access, the camera will be used to scan the barcode of the searched product.

Recipes page

The Recipes page displays featured recipes, and a recipe can be selected to have a detail view. The recipe search tool allows sorting by Name, Preparation and Total, filter by type of recipes or by serving. It is also possible to add items to a shopping list by selecting the ingredients from the recipe. Here are the features of this page based on the demo of LOC Software:

  • Featured Recipes (select one to display details)
    • Recipe detail
      • Ingredients (items can be added to Cart and to a List)
      • Directions
      • Nutrition Facts
  • Find more recipes
    • Search field (recipe search)
    • Sort by Name, Preparation and Total
    • Filter by Type and Serving

NOTE: To create and display recipes as Featured recipes, contact the reseller.
