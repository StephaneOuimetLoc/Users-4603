eSuite - Restriction Table

The Restriction table contains information that allows controlling the item availability; in other words, one can set time limits for when the items can be sold.

By default, the restrictions are bypassed by the assistant, the manager, the owner and the programmer user levels. Ask the reseller for more details.

Displaying the Restriction table

In the Launcher, click the chevron in the Items entry to display its menu and then:

  1. Click on Auxiliary tables: the Start window opens in the Auxiliary work area.
  2. Click on Restrictors: the tab opens in the same window, displaying the Restriction List.
  3. To modify an existing restriction record, select it in the list and double-click on it, press the F3 key on the keyboard or click the Edit command in the bottom menu:
    • the Restriction table opens, overlapping the work area.
  4. To create a new restriction record, click on Add in the bottom menu of the grid: the Restriction table opens, overlapping the work area.

Content

Here is a description of the Restriction table fields and their functionalities:

FIELD DESCRIPTION
Level The Level field contains the restriction number.
It is a required field used to uniquely identify records in the Restriction List.
Descriptor The Descriptor field contains the description of the restriction entry.
It is not a required field; however, a descriptor should be entered to identify the restriction in the Restriction List.
Start The Start field contains the starting time at which the items linked to the restriction can be sold or are prohibited from being sold, depending on the value of the This is a permission and not a restriction field.
Stop The Stop field contains the ending time at which the items linked to the restriction can be sold or are prohibited from being sold, depending on the value of the This is a permission and not a restriction field.
This is a permission and not a restriction The This is a permission and not a restriction field specifies whether the periods defined by the Start and the Stop fields are permitted periods of sale (checked) or prohibited periods of sale (unchecked).
