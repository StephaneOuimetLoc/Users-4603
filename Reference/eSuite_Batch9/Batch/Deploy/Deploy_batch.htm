eSuite - Deploy Batch

eSuite > Tools > Deploy

Information

The Deploy batch job sends batches from the the Head office (HO) to the stores.

The Deploy batch job deploys the following batch types:

  • Future price
  • Future general price
  • Future sale
  • Future TPR
  • Future instore price
  • Future redeem
  • Future cost
  • Future general cost
  • Future allowance
  • Future rebate
  • Future discount
  • Future cost break
  • Future bill back
  • Future main item
  • Future POS
  • Open batch
  • Deletion batch (under development)

To perform a Deploy batch job:

  1. Click the arrow in the Tools entry in the Launcher to display its menu
  2. Select Batches: the Start window opens in the Batch work area
  3. Select Deploy batch: the Wizard window opens for batch selection
  4. Click in the Select a batch field
  5. Select a batch in the list
  6. Click on Confirm: the Wizard window opens for target selection

  7. Select a target and click on Confirm to deploy the batch
