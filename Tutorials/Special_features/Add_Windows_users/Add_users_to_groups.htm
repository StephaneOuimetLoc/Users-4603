Add Windows Users to the LOC SMS Windows Groups

This document describes the steps to add Windows users to the different LOC SMS Windows groups created during the installation of the LOC Store Management Suite software.

When you install the Share files component, four Windows groups are created. Each of them has specific characteristics that are important to know when it is time to add Windows users to them.

SMSOperators SMSProgrammers SMSRunAs SMSDenied
Windows users who belong to the SMSOperators Windows group are the regular day-to-day users of the LOC Store Management Suite software, i.e. they have access to the regular operations with regular rights (regular levels in the system). The access to the Logs folder is denied. Windows users who belong to the SMSProgrammers Windows group have administrative privileges inside the LOC Store Management Suite software, i.e. they have access to all operations with programmer rights (highest level in the system). The access to the Logs folder is denied. Windows users who belong to the SMSRunAs Windows group are denied file access to the LOC SMS files but they are granted access to the LOC SMS software through the SMSRunAs utility. See SMSRunAs utility for more details. Windows users who belong to the SMSDenied Windows group are prevented access to the LOC Store Management Suite software. This group is used to prevent users from accessing the LOC SMS software and its files, so they are the computer users who do not work with the LOC SMS software.

Procedure for a Workgroup computer

Note:
The steps are a slightly different if the computer is joined to domain.

  1. Press 'Windows key + X' and select 'Computer Management'.
  2. Expand the Local Users and Groups and select Users.

    The list of all Windows users is displayed.

  3. In the list of Windows users, double-click the user name of the user you want to customize (create a new user if the desired one does not exist).
  4. In the User Properties window, click the Member Of tab;

    The Member Of tab lists all Windows groups in which the current user is actually a member.

  5. To add a group to the list, click .

    The Select Groups window is displayed.

  6. In the Select Groups window, click .

    The Select Groups window is displayed.

  7. In this other Select Groups window, click .

    This lists all Windows groups that exist on the local machine.

  8. In the list of groups, select the Windows group to add, and click .

    This enters the selected group in the Enter the object names to select field of the first Select Groups window.

  9. Click to confirm the addition of the selected group.

    The Windows user is now member of the selected group.

  10. Click to close the User Properties window.

  11. Close the Computer Management window.

Procedure on Active Directory

To add a Windows user to a LOC SMS Windows group in Active Directory, follow these steps:

  1. From the Start menu, choose Administrative Tools > Active Directory Users and Computers.

    The Active Directory Users and Computers window is displayed.

  2. In the Active Directory Users and Computers window, expand the domain and select Users.

    This lists all existing Windows users on the domain.

  3. In the list of existing Windows users, double-click the username of the user you want to customize (create a new user if the desired one does not exist).

    This opens the Operator Properties window.

  4. In the Operator Properties window, click the Member Of.

    The Member Of tab lists all Windows groups in which the current operator is actually a member.

  5. To add a group in the list, click .

    This opens the Select Groups window.

  6. In the Select Groups window, click .

    This opens another Select Groups window.

  7. In this other Select Groups window, click .

    This lists all Windows groups that exist on the domain.

  8. In the list groups, select the Windows group to add, and click .

    This enters the selected group in the Enter the object names to select field of the first Select Groups window.

  9. Click to confirm the addition of the selected group.

    The Windows operator is now member of the selected group.

  10. Click to close the Operator Properties window.

  11. Close the Active Directory Users and Computers window.
