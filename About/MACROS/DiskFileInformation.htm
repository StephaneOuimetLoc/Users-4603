Disk file information

@dbHot(FINDFIRST,<FileMask>,<Count>)

This macro searches for the nth instance of a file name in a specified path.

Parameter Description Required/Optionnal
FileMask Path including mask to look for.
The path could be any UNC standard path and the mask may contain wild cards :
? : Match any one character
* : Macth 0, 1 or more caracters
Examples of mask : *.exe
A*.exe
MyFile.txt
Required
Count Nth instance in the file list return by the path. Optionnal
Default=1
Return value
If the file exist, the macro will return the file name, if not, it will return blank.
Examples
@dbHot(FINDFIRST,c:\temp\*.txt,2);
@dbHot(FINDFIRST,c:\temp\Myfile.txt);
@dbHot(FINDFIRST,\\ComputerName\temp\*.txt,2);
