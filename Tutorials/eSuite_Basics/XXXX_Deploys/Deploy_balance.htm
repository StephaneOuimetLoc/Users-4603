eSuite - Deploy Balance

eSuite > Tools > Deploy

Information

To be revised and update once this feature is available in eSuite.
 Raluca, 2019-10-16

The Deploy balance job sends balances from the Head office (HO) to the stores and from the backoffice server to the front-end workstations.

The Deploy balance job deploys the following information:

  • The bitmaps
  • The receipt header
  • The HTML buttons
  • The HTML balances

To perform a Deploy balance job:

  1. Click the arrow in the Tools entry in the Launcher to display its menu
  2. Select Deploy: the Start window opens in the Deploy work area
  3. Select Deploy balance

NOTE: The Deploy balance job requires the right security level. Different security levels allow different accesses.
