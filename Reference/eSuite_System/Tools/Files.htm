eSuite System - File Tools Overview

eSuite System > Tools > Files

Information

The Files module in Tools as helpful in order to execute or to view contents of Script files within the local SMS instance. There is a section specific for each different file type (extension) used in SMS/eSuite.

Start Menu

Available actions

  • Executing a script
  • Viewing contents of a script

Execute SQL script

Use this option in order to execute scripts having .sql as file extension. SQL Files use Structured Query Language (SQL) to modify the contents of a relational DB (STORESQL, LANESQL, etc). This includes modifying database structures, insertions, updates and deletion of records. SQL files can contain multiple scripts within a single file.

Execute SQI script

Use this option in order to execute scripts having .sqi as file extension. Files using Structured Query Language (SQL) that are used to modify the contents of a relational DB (STORESQL, LANESQL, etc). This includes modifying database structures, insertions, updates and deletion of records. SQI files are used to execute individual scripts.

Execute SQM script

Use this option in order to execute scripts having .sqm as file extension.SQM files are files that use Structured Query Language (SQL) that are used to map files. Example of this are files used in order to deploy data or configuration.

Execute SQR script

Use this option in order to execute scripts having .sqr as file extension. Files using structured Query Language that are used to create or generate reports using the STORESQL as recipient of the data.

Execute CGI script

Common Gateway Interface is an interface specification that allows to execute console applications on a server that generates web pages dynamically. Examples of CGI files used by LOC are the button pages or Reports that are generated dynamically upon script execution. 

 

Viewing contents of a script

  • Select the type of script from the Start menu
  • Search or select the specific script to view from the File list.
  • Press on View on the footer menu.

  • You can also execute the script once you've viewed its contents from the header of the View File window.

Executing a script

  • Select the type of script from the Start menu
  • Search or select the specific script to view from the File list.
  • Press on Execute on the footer menu. (Double clicking on the File will also result on the script's execution)

  • Once you confirm the execution of the script the execution command will be sent for execution.
