eSuite - Grid page

eSuite > Launcher entry > Start page > Wizard page > Grid

Table of contents

Information

The grid page opens on the desktop when you execute the search from the Wizard page.

It displays record information and includes tools such as commands, menu buttons and checkboxes to query and manipulate data.

Layout

The grid page layout is identical across modules. It includes the following sections:

  1. Tab bar
  2. Navigation bar
  3. Title bar
  4. Layout menu
  5. Toolbar
  6. Content panel     
  7. Bottom menu     
  8. Resize handles     

Content

This section focuses on the components of the grid page and on the actions they trigger.

The tab bar

The tab bar is located at the top left of the grid page. It includes three tabs:

  • Start
  • Wizard
  • Grid

The grid tab displays the panel containing the results of the search executed from the wizard.

When active, the grid tab is marked by a horizontal blue line.

You can navigate through the Start, the Wizard and the grid panels by clicking on the respective tab to activate it.

The navigation bar is located at the top right of the grid page.

It displays the following controls:

The title bar

The title bar is located at the top of the grid page.

It contains the name of the grid.

The layout menu

The layout menu is located on the title bar.

It provides access to the list of saved grid layouts.

It works with the Keep layout locally button in General settings (the Toolbar section).

The content panel

The content panel is located in the central area of the grid page.

It displays the information about the records that were retrieved from the search.

Data is displayed in columns and rows with flexible behavior.

  • Columns are movable and resizable.
  • The multi-level headers allow you to add and hide columns.
  • Rows can be sorted in ascending or descending order.
  • They can be grouped by fields.
  • They can be checked and unchecked with the help of checkboxes.

Below is the list of controls that are available in grids across eSuite.

The resize handles

The grid page provides eight resize handles, one in each corner and one on each side.

They allow you to resize the page horizontally, vertically and diagonally.

The resize handles are visible when you hover the mouse over the grid.
