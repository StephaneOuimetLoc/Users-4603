UPC change

Copy existing item to new code

First, you will need to retrieve the item that you want to copy.

In the launcher, click the arrow in the Items entry to display its menu and then:

Click on Master: the Start window opens in the Items work area.

  1. Click on Item list to open the wizard.
  2. Enter the search criteria.
  3. Click on Execute: results will display in the Items - General list window.
  4. Select the item to duplicate.
  5. Click Copy current item to a new item  in the toolbar.
 

The Item Creation table opens on the desktop, with the Copy item window overlapping:

When a new item is created from an existing one, information needs to be chosen and filtered carefully. Through Check all, Uncheck all and the checkboxes, the Copy item window is the utility that enables this process in an efficient and timesaving manner.

  1. From the item grid, select a record and click on Copy current item to a new item  in the toolbar.
  2. In the Copy item window, enter a new item number in the New UPC field.
    The Confirm button is enabled.

  1. Select the appropriate checkboxes.
  2. Click Confirm to save the changes and close the Copy item window.
  3. The Item Creation table displays the new item with the information copied from the existing one.
  4. Make any necessary modifications.
  5. Save the changes.
  6. Close the Item creation table.
  7. Refresh the item grid to see the new item record.
  8. Deploy changes.

Set the old code to Discontinued

Once the item is copied, then you can set the old code to discontinued by doing the following:

  1. From the Items - General list select the item
  2. Click on Edit, Master tables and POS master

Click on the Status drop down

Select Discontinued and check the box Not for sale and Save.
