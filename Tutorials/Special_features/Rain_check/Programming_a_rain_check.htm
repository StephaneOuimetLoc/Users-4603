eSuite - How to Program a Rain Check

eSuite > Items

Information

WARNING: To be revised and edited accordingly once the Label designer feature is available in eSuite.

A rain check is a coupon printed by the system when a promotional item is out of stock. The rain check is given to the customer for later purchase after restocking, at the promotional price, even if the promotion has ended. The promotional price is printed on the rain check.

Rain checks are printed on the coupon printer. This must be programmed and compatible with the coupon format.

Steps:

  1. Program the item as usual.
  2. In the POS table, check the Rain check field.

The Label designer tool allows designing customized coupon formats. The default format is saved as trs_pos_frm_cpn_win.htm in the Office/Cgi directory. Name the file of the customized coupon format as trs_pos_frm_[name of the form].htm.

To use a different coupon format than the default one:

  • Enter the name of the form in the Ticket field of the POS table:

NOTE: The system provides two samples of templates: "text" (trs_pos_frm_cpn_txt.htm) and "graphic" (trs_pos_frm_cpn_win.htm). The graphic template (CPN_WIN) can be used only with a true windows printer driver.
