Information


The Classification table contains a classification code that links the product to the classification group. Classification groups may be used to identify different groups of items such as gluten-free, organic and peanut-free. An item can be part of several classification groups.

It is used to simplify customer's search in eStore/mStore online shopping.

Displaying the Classification table

From down arrow in Items, select Item auxiliaries to display the Start Wizard

Select Classification to display the Classification List grid

Click   in the grid header toolbar to display the  Classification edit table

To create a Classification

1. Display the Classification table as above or click  in the toolbar of a populated edit table.

2. Enter an ID and a Descriptor

3. To save, click the  in the toolbar or hit enter on your keyboard

4. To review your entry, select  from the bottom menu to refresh the grid

To modify a Classification

1. From the Classification list grid, select/highlight the record you wish to edit

2. Double click on the entry, press F3 on your keyboard or click on Edit from the bottom grid menu and choose Classification

4. Modify as required

5. Save & close the edit window

6. Refresh

Applying a Classification

Once created, you may assign  various classifications to an item by using the Item Classification  Edit table found in Items.

1. To display the Classification Edit table

Items>Items - General list> Enter the code in the Wizard Any code field> execute

2. In the grid, select or highlight the item you would like to classify

3. Right click on the selection and choose Special Items> Item Classification, or from the bottom of the grid,  select the drop down on the menu  Edit> Special Items> Item Classification

4. Click the  on the Item Classification Edit table and note the item number is now listed in the toolbar

5. From the drop down menu in the Classification field, search/select the required entry and choose OK

6. To save and add another entry, press enter on your keyboard and repeat step 5
