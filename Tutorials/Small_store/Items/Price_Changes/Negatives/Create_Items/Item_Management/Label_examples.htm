eSuite - Generate Labels                       

eSuite > Items

Information

The common section of the eSuite software offers the tools for generating and printing shelf labels and promotional signs.

There are two ways of extracting data and generating labels:

  • Based on grid selection (items and batches), through the Labels - Item browser wizard.
  • Based on price events and on RFC scans, from the Labels entry in the launcher.

List of examples

  1. Generating labels from Items - General list
  2. Generating labels from the Batches entry

1. Generating labels from the item grid

Operators can generate labels from the general list of items. This is useful for the item maintenance and the creation of labels for specific items.

The example below illustrates how to generate labels from Items - General List.

  1. From the Items entry in the launcher, select Master.
  2. In the Start window, select Item list.
  3. In the Items - General list wizard, enter the search criteria.
  4. Click on Execute.
  5. From Items - General list, select the items that require labels.
    Note: You may increase the number of items displayed by page by clicking on the Page size control in the bottom menu.

  7. Click on Action in the bottom menu of the item grid.
  8. Select Generate label: the Label - Item browser wizard opens on the desktop.
  9. Select a label format: the Execute button becomes enabled.

  10. Select other necessary criteria (see Labels - Item browser table for details on the fields).
  11. Click on Execute: the Print label list window opens in a new tab, displaying all the label formats available for printing.
  12. Select the newly generated label from the list.
  13. To print the label, click Action on the bottom of the list and select Print label.
    This gives access to the Label print options window.

  14. To delete the label, click Action on the bottom of the list and select Delete label.

2. Generating labels from the batch grid 

The example below illustrates how to generate labels for a future sale batch.

  1. From the Items entry in the launcher, select Batches.
  2. In the Start window, select Batches.
  3. In the List batch wizard, enter the search criteria for the future sale batch.
  4. Click on Execute.
  5. From the list of in-store batches, double click on the future sale batch that requires labels: the Future sale batch window displays the list of items attached to the future sale batch.
  6. Select the items that require labels.
  7. Click on Action in the bototm menu of the grid.
  8. Select Generate label: the Label - Item browser wizard opens on the desktop.
  9. Select a label format: the Execute button becomes enabled.

  10. Select other necessary criteria (see Labels - Item browser table for details on the fields).
  11. Click on Execute: the Print label list window opens in a new tab, displaying all the label formats available for printing.
  12. Select the newly generated label from the list.
  13. To print the label, click Action on the bottom of the list and select Print label.
    This gives access to the Label print options window.
