The Cursor location functionality in eSuite allows you to record the sequence in which the cursor changes fields. This is useful when moving from field to field in a repetitive manner.

It is available from the Session menu in the eSuite taskbar.

Layout

The Cursor location button opens the Cursor location page.

The Cursor location page layout is identical across modules. It includes the following sections:

  • Tab bar
  • Navigation bar
  • Title bar
  • Move record
  • Toolbar
  • Grid
  • Resize handles

Content

This section focuses on the components of the Cursor location page and on the actions they trigger.

The tab bar

The tab bar is located at the top left of the Cursor location page. It includes the Cursor location list tab. It displays the panel containing the fields that have been recorded as cursor movements.

The navigation bar

The navigation bar is located at the top right of the Cursor location page.

It displays the following buttons:

BUTTON ICON DESCRIPTION
Freeze window size Freezes the window size to that set by the user.
Restore to default window size Resets the window size to the default one.
It reverses the action of the Freeze window size button.
Fit window to full screen Sets the window size to that of the screen.
Restore down from full screen Resets the window size to the default one.
It reverses the action of the Fit window to full screen button.
Fit window to desktop Sets the window size to that of the desktop.
Restore down from desktop Resets the window size to the default one.
It reverses the action of the Fit window to desktop button.
Move window Moves the window around the desktop and screen.
Close window Closes the window.

The title bar

The title bar is located at the top of the Cursor location page.

It contains the name of the page.

The Move record tools

The moving tools are located on the title bar. They act on the position of the fields that are selected in the content panel.

The moving tools include the following buttons:

BUTTON DESCRIPTION
Move record up Moves the selected field one step up.
Move record down Moves the selected field one step down.

The toolbar

The toolbar is located on the right of the title bar.

The following buttons are available:

BUTTON ICON DESCRIPTION
Save Saves the changes in the cursor movement sequence.
Prompts the opening of the Layout setting window.
Delete Deletes the fields selected in the content panel.
Prompt grab Enables the recording of cursor movements:
  1. Triggers the Grab a view field button  in the eSuite taskbar.
  2. Allows you to select fields in the window that is active on the desktop.
  3. Places the selected fields in the Cursor location list when you click on the Grab a view field button  in the taskbar.

The Layout setting window

The following elements are available

ELEMENT DESCRIPTION
Layout setting Shows the window's name.
Layout Allows you to enter text or select text from a list.
Save Saves the changes in the Layout field.
Delete Deletes the value selected in the Layout field.

The grid

The grid is located in the central area of the Cursor location page.

It lists the fields in the order that they were recorded.

Data is displayed in columns with flexible behavior.

  • Columns are movable and resizable.
  • Column headers allow you to show or hide columns.
  • Column headers also allow you to sort the column contents in ascending or descending order.

The following elements are available in the Cursor location grid:

CONTROL DESCRIPTION
Checkbox column The header checkbox allows you to select or unselect all grid records in a single click.
The column checkboxes allow you to select or unselect one record at a time.
Win column Contains the alias name of the field.
Not editable.
Field column Contains the field number.
Not editable.

The resize handles

The Cursor location page provides eight resize handles, one in each corner and one on each side.

They allow you to resize the page horizontally, vertically and diagonally.
