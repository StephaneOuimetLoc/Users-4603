eSuite - Wizard page

eSuite > Launcher entry > Start page > Wizard page

Table of contents

Information

The Wizard page opens on the desktop when an option is selected in the Start page.

It provides multiple criteria that help you refine searches and customize grids.

Layout

The Wizard page layout is identical across modules. It includes the following sections:

  1. Tab bar
  2. Navigation bar
  3. Title bar
  4. Filter controls
  5. Content panel
  6. Command area
  7. Resize handles

Content

This section focuses on the components of the Wizard page and on the actions they trigger.

The tab bar

The tab bar is located at the top left of the Wizard page. It includes two tabs:

  • Start
  • Wizard

The Wizard tab displays a panel with search tools based on the entry that you select in the launcher.

When active, the Wizard tab is marked by a blue horizontal line.

You can navigate through the Start and Wizard panels by clicking on the respective tab to activate it.

The navigation bar

The navigation bar is located at the top right of the Wizard page.

It displays the following controls:

CONTROL ICON DESCRIPTION
Freeze window size Freezes the window size to that set by the user.
Restore to default window size Resets the window size to the default one.
Fit window to full screen Sets the window size to that of the screen.
Restore down from full screen Resets the window size to the default one.
It reverses the action of the Fit window to full screen button.
Fit window to desktop Sets the window size to that of the desktop.
Restore down from desktop Resets the window size to the default one.
It reverses the action of the Fit window to desktop button.
Move window Moves the window around the desktop and screen.
Close window Closes the window.

The title bar

The title bar is located at the top of the Wizard page.

It contains the name of the wizard.

The filter controls

The filter controls are located on the right side of the title bar.

The following controls are available:

CONTROL DESCRIPTION
Clear all fields Erases the values in all the wizard's fields.
Save query Saves the configuration parameters.

The content panel

The content panel is located in the central area of the Wizard page. It includes fields and controls that facilitate the searching process.

The wizard tools trigger the opening of grid pages.

Below is the list of controls that are common to wizards across eSuite.

Provides access to search tools.

Opens the Select criteria window.

Grab fields Grabs fields from the item tables that are pinned on the same desktop as the wizard.
Complementary to the More criteria function.
Works with the Add selected button.

The command area

The command area is located at the bottom of the Wizard page.

It consists of three sections: the menu, the Execute button and the settings.

The menu provides the following options in wizards across eSuite:

OPTION DESCRIPTION
Copy and paste item filters Available in the item wizard.
Opens the Clipboard paste search window.
Allows you to paste relevant item data that has been copied from e-mail, Word, NotePad, CSV or Excel documents.
When the query is executed, the copied and pasted data will be displayed in the result grid.
Paste customers from Excel Available in the customer wizard.
Opens the Clipboard paste search window.
Allows you to paste relevant customer data that has been copied from e-mail, Word, NotePad, CSV or Excel documents.
When the query is executed, the copied and pasted data will be displayed in the customer grid.
Add scheduled tasks Available in all eSuite wizards.
Opens the Edit task window.
Allows you to create new tasks or to edit existing ones.
Show scheduled tasks Available in all eSuite wizards.
Opens the Tasks grid.
Clear all search fields Erases the values of all the fields in the wizard.

The Execute button has the following functions:

  • Launches the search according to the wizard criteria.
  • Opens the result grid.

The Settings button opens the parameters window.

The following parameters are available:

PARAMETER DESCRIPTION
Top result records Sets the maximum number of records that will be displayed in the grid.
Default value: 10000
Top pivot records Sets the maximum number of records that the grid may have in order to enable the Pivot button.
If there are more results in the grid than the set parameter, the Pivot button will be disabled.
Default value: 2000
Server timeout(s) Sets the time in seconds that the search operation will take in order to get data from the server.
If the operation takes longer than the set parameter, it will time out and cancel the operation.
Default value: 40
OK Saves the changes.
Closes the parameter window.
Cancel Cancels the changes.
Closes the parameter window.
Restore default parameters Restores the parameters to the default ones.

The resize handles


The Wizard page provides eight resize handles, one in each corner and one on each side.

They allow you to resize the page horizontally, vertically and diagonally.

The resize handles are visible when you hover the mouse over the Wizard window.
