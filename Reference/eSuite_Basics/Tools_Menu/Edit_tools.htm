eSuite - Editing grid information

eSuite > Launcher entry > Start page > Wizard page > Grid

Table of contents

Information

The eSuite grid allows you to edit some of the information directly from the grid rows.

Editing tools

The editing tools are available from the grid's editable fields.

Tool properties:

  • Allows text input in row fields that are marked by a small flag in the top right corner.
       
  • Allows you to double-click on the field to add text.
  • Highlights in red the modified fields when changes are not saved.
       
  • Saves the changes with the Enter key.
  • Can apply the same change to a range of consecutive records when you select them holding down the Shift key.
  • Can apply the same change to a range of unconsecutive records when you select them holding down the Ctrl key.
