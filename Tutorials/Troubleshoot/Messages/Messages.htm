eSuite System - Messages

eSuite System > Messages

Information

  • The Messages optiddddddddon in the Messages entry opens an index listing of all the urgent and normal messages in the system. This option is useful to get a quick overview of the current messages.
  • Messages are listed in the Folders window. Each message contains files that are listed in the Files window. When double-clicked, files open the View file window. See below.

The Folders grid

The Folders grid lists the urgent and normal messages grouped by topic. To access it:

  1. In the launcher, click the Messages entry: the Start window opens in the Messages work area
  2. Click on Messages to open the wizard
  3. Enter the search criteria and click Execute: results will display in the Folders grid

By default, the grid displays the following columns: Store, Terminal, Name, Folder type, Box, and Info. Columns may be added or removed by clicking on any grid header and selecting/deselecting options in the Columns menu:

Folders grid content

The Folders grid contains the list of normal and urgent messages grouped by topic.

*Click on a field to get more information.

Below is the description of the fields and functionalities:

FIELDS DESCRIPTION
Store The Store field contains the number of the store the message refers to.
Terminal The Terminal field contains the number of the terminal the message refers to.
Name The Name field contains the name of the terminal.
Folder type The Folder type field contains the type of folder (urgent or nomal) where the message is located.
Box The Box field contains the name of the directory where the message is stored.
Node The Node field contains the node to which the message refers.
Info The Info field contains information on the number of files in the message and their size.

The Files grid

The Files grid lists the files that are part of the same message. To access it:

  1. Select a folder from the Folder grid
  2. Double-click on it: the Files grid opens in a new tab, displaying the files that are part of the selected folder

By default, the list displays the following columns: Date, Subject, From and To. Columns may be added or removed by clicking on any grid header and selecting/deselecting options in the Columns menu:

Files grid content

The Files grid contains the files that are part of the same message.

*Click on a field to get more information.

Below is the description of the fields and functionalities.

FIELDS DESCRIPTION
Filter The Filter section allows refining the list of messages by applying various criteria (period, date, from, to and subject).
Checkboxes The checkboxes allow selecting and deselecting messages.
Date The Date field contains the date and the time of the message.
Subject The Subject field contains the topic of the message.
From The From field contains the node number of the PC that sends the message.
To The To field contains the node number of the PC to which the message is sent.

Viewing messages

To view a message:

  1. Select a file in the Files grid
  2. Double-click on it: the View file window opens on the desktop, displaying the message details

Message content

The View file contains the message header, with general information on the file and the message content, with details on the file.

*Click on a field to get more information.

Below is the description of the message fields and functionalities

FIELD DESCRIPTION
Remove archive bit Under develoment.

A= Archive

To The To field contains the node number of the PC to which the message is sent.

Printing messages

The View file window allows printing the message:

  1. Open the View file window for the respective message
  2. Click Print in the toolbar: the message opens in a new browser tab

  1. Click Print to set the parameters and complete the printing

Deleting a message

The Files grid allows deleting messages:

  1. Select the message to delete
  2. Click Delete data in the toolbar

  1. Click Refresh in the bottom menu of the grid to confirm that the message has been deleted

Deleting all messages

To delete all messages:

  1. Access the Folders grid.
  2. Click Delete all in the bottom menu: a confirmation message is prompted
  3. Click Yes to confirm
  4. Click Refresh in the bottom menu of the grid to confirm that the message has been deleted
