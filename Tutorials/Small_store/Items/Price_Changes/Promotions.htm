Setup Promotions

The eSuite software uses relational tables to store item information. The Price table contains all pricing information for the main item.

Displaying the Price table

In the Launcher, click the arrow in the Items entry to display its menu and then:

Click on Master: the Start window opens in the Items work area.

  1. Click on Item list: the Wizard tab opens in the same window, displaying the Items - General List view.
  2. Enter the search criteria and click Execute: results will display in a record grid, in the Items tab of the same window.
  3. Select an item.
  4. Click on Edit on the bottom of the item grid to display its menu.
  5. Select Master tables, then Price master: the Price edit table opens over the work area.

To add a Sale price:

Enter the new sale price along with the date and time for the sale period and save.

To add a TPR price:

Enter the new TPR price along with the date Ad value and save.
The Ad field, containing the identification letter of the advertising type used for the item.

To add an In-store price:

Enter the new Store price along with the date and select the proper checkbox and save.

Notes:
The TPR checkbox specifies whether the store price overrides the TPR price (checked) or not (unchecked).
The Sale field specifies whether the store price overrides all prices (checked) or not (unchecked).

In this example, we are overriding the tpr price.

 

To add a Package price:

Enter the selling price for the package under the Base tab and the qty that must be bought in order to qualify for this price.
In this example the selling price is set to 6.99$ if the customer buys a qty of 2.

Notes:
The Apply package price field must be checked in the corresponding shopper levels for the Package field to work.
By default, the package discount is applied on the last item. However, it is possible to apply this discount proportionally on all items (system.ini). See the dealer for more details.
To apply the package discount on subsequent items (i.e. on all items after the quantity is reached), enter the Keep going package price method (F) in the Method (Meth) field. 

To add a Limited price:

Enter the selling retail in the limit box and the qty.

The Retail Limit amount field contains the retail limited quantity price for the item.

Note:
 The Apply limited price field in the Shopper level table must be checked in the corresponding shopper levels for the Limit quantity to work.
