eSuite - Label Template Group Table

eSuite > Items > Item auxialaries

Information

To be revised and updated once the feature is available in eSuite.
SMS Pro images are for reference only; don't delete them when adding the eSuite equivalent, as mapping already done. (Raluca, 2019-09-20)

The Label template group table associates a label template name with an event tag. It allows to define a label template for each event under the same label template group.

Displaying the Label template group table

In the Launcher, click the arrow in the Items entry to display its menu and then:

  1. Click on Auxiliary tables: the Start window opens in the Auxiliary work area.
  2. Click on Label template group: the xxxxxx tab opens, displaying the Label template group list.
  3. To modify an existing label template group, select it in the list and double-click on it, press the F3 key on the keyboard or click the Edit command in the bottom menu: the Label template group edit table opens, overlapping the work area.
  4. To create a like code, click on Add in the bottom menu of the grid: the Label template group table opens, overlapping the work area.

Layout

* Click on a field to have information on it.

Content

Here is a description of the Label template group table fields and their functionalities:

FIELD DESCRIPTION
Label template group The Label template group field contains the master template name, which is used to link the item's tags to the label template group.
It is used as a key field with the Price source field to uniquely identify records in the Label template group table.
Price source The Price source field contains the price event to use.
It is used as a key field with the Label template group field to uniquely identify records in the Label template group table.
Label template The Label template field contains the default type of shelf tag to use with the selected price source.
