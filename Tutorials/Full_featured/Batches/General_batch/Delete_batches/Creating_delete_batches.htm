eSuite - How to Create Delete Batches

eSuite > Items > Batches

The delete batches are used to delete multiple items at a time.

New batches are created through the Batch header table that sets the batch type and its unique number. It also contains the general batch information.

This document focuses on how to create delete batches. It will also show how to populate them with items to delete, using the Delete batch option in the Batches wizard.

Creating a new delete batch

In the launcher, click the arrow in the Items entry to display its menu and then:

  1. Click on Batches: the Start window opens in the Batches work area.
  2. Click on New to open the list of batch types.
  3. Select Delete batch at the end of the list: the Batch header table opens on the desktop.
    The following fields are populated by default: Batch ID, Target, Source, Start date and Time, End date and Time, Type and Action.
  4. Make any necessary changes.
  5. Click Save in the toolbar to save the changes and close the Batch table.

Adding items to the delete batch

To add items to the delete batch, follow these steps:

  1. In the In-store batches list, click on the checkbox corresponding to the delete batch to select it.
  2. In the bottom menu, click on Edit to open the Batch header table.
  3. Click on the Open list button in the toolbar: the Delete batch wizard opens on the desktop.

  4. Enter the search criteria for the items that will be part of the delete batch.
  5. Click Execute.
  6. In the item list, check the items that will be part of the delete batch.
  7. Click Action in the bottom menu of the grid and select Mark to be deleted: each selected items will be identified by a red bin icon:
  8. Save the changes.
  9.  Deploy the batch.
