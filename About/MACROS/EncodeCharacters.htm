Encode/decode specials characters

Some characters have special significance in different cases (HTML, URL, Javascript,etc.) and should be converted to preserve their meanings.
For example, in HTML, it is not possible to use the < character as it is used to create and close tags. It must be converted to its corresponding &lt; entity to be displayed in the content of an HTML page.

@Fmt(2HTML,<TextValue>)

This macro encodes text to HTML by replacing:
    < by &lt;
    > by &gt;
    & by &amp;
    ( by [   and    ) by ]    for macro parentheses. i.e. @WizGet(A) will become @WizGet[A]
    double quotes " by &quot;

Parameter Description Required/Optionnal
TextValue Text to encode. Required
Return value
Encoded text
Example
@Fmt(2HTML,(<"A&B">)) will return [&lt;&quot;A&amp;B&quot;&gt;]

If Registry TheKey=(Description = @WizGet(DSC))
@dbHot(VARGET,TheKey,,MACRO)                       will return (Description = @WizGet(DSC))
@Fmt(2HTML,@dbHOT(VARGET,TheKey,,MACRO)) will return (Description = @WizGet[DSC])

@Fmt(HTML2,<TextValue>)

This macro decode text from HTML by replacing:
    &quot; by double quote "
    &lt; by <
    &gt; by >
    &amp; by &
    &space; by space
    &reg; by « (#174)
    &copy; by ® (#169)

Parameter Description Required/Optionnal
TextValue Text to decode. Required
Return value
Decoded text
Example
@Fmt(HTML2,[&lt;&quot;A&amp;B&quot;&gt;]) will return [<"A&B">]

@Fmt(2URL,<TextValue>)

This macro encode text to URL by replacing each space by +
 Also, if character not in : 0 to 9, a to z and A to Z the macro replace the character by %Hexadecimal ASCII Character (Ex : ! replaced by %21)

Parameter Description Required/Optionnal
TextValue Text to encode. Required
Return value
Encoded text
Example
@Fmt(2URL,@Loc Software!) will return %40Loc+Software%21

@Fmt(URL2,<TextValue>)

This macro decode text from URL by replacing :
    + by space
    %Hexadecimal ASCII Character by corresponding character (Ex: %21 replaced by !)

Parameter Description Required/Optionnal
TextValue Text to decode. Required
Return value
Decoded text
Example
@Fmt(URL2,%40Loc+Software%21) will return @Loc Software!

@Fmt(2JS,<TextValue>)

This macro encodes text to Javascript by replacing:
    backslash  \  by \\
    double quotes  " by \"
    single quote  ' by \'

Parameter Description Required/Optionnal
TextValue Text to encode. Required
Return value
Encoded text
Example
@Fmt(2JS,\don't Stop) will return \\don\'t Stop

@FMT(JS2,<TextValue>)

This macro decode text from Javascript by replacing:
    \\   by backslash  \ 
    \"   by double quotes  "     
    \'    by single quote  '

Parameter Description Required/Optionnal
TextValue Text to decode. Required
Return value
Decoded text
Example
@Fmt(JS2,\\don\'t Stop) will return \don't Stop

@Fmt(2SPACE,<TextValue>)

This macro encodes the space character.
Space is replaced by underscore character _.

Parameter Description Required/Optionnal
TextValue Text to encode. Required
Return value
Encoded text
Example
@Fmt(2SPACE,Loc Software) will return Loc_Software

@Fmt(SPACE2,<TextValue>)

This macro decodes the space character.
Underscore character _ is replaced by a space.

Parameter Description Required/Optionnal
TextValue Text to decode. Required
Return value
Decoded text
Example
@Fmt(SPACE2,Loc_Software) will return Loc Software

@Fmt(2QUOT,<TextValue>)

This macro encodes text by replacing :
    each quote by 2 quotes
    carriage return + line feed (#13#10) by 1 space
    carriage return (#13) by 1 space
    line feed (#10) by 1 space

Parameter Description Required/Optionnal
TextValue Text to encode. Required
Return value
Encoded text
Example
@Fmt(2QUOT,Don't stop believin) will return "Don''t stop believin"

@Fmt(QUOT2,<TextValue>)

This macro decodes text by replacing 2 consecutives quotes by 1 quote.

Parameter Description Required/Optionnal
TextValue Text to decode. Required
Return value
Decoded text
Example
@Fmt(QUOT2,Don''t stop believin) will return "Don't stop believin"
