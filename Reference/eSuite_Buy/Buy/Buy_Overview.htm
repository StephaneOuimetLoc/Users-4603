eSuite Buy - Overview

eSuite Buy > Buy

Information

The Buy entry in the eSuite Buy module contains functionalities for performing purchasing activities such as creating purchase orders, receiving merchandise, billing and Direct Store Deliveries (DSD).

The Buy entry is split into four sub-entries:

  • Order
  • Receive
  • Bill
  • DSD

The Context tools window is active on the right side of the desktop. It adjusts its layout according to the entry selected in the launcher and displays information relevant to the element that is active on the desktop.

Order

The Order sub-entry opens a list of options that provide purchasing features that are used on backoffice workstations. It also provides the tools for creating and editing purchase orders.

The following options are available from the Order start window:

OPTION DESCRIPTION
Documents (Full) Displays the purchase order that was last opened from the list of open orders.
Disabled by default. Enabled when a PO is opened from the list of open orders.
Documents Opens the Document search wizard with search tools related to purchasing.
Provides access to the list of open orders.
Documents - All Opens the list of open and suspended orders.
Documents - New Opens the New document window for the vendor selection.
Creates purchase documents per vendor.
Vendors Opens the Vendor search wizard.
Provides access to the list of vendors.
Vendors - All Provides direct access to the list of vendors.
Items Opens the Items - list of items wizard.
Provides access to the list of items displayed by vendor.
Items - All Provides direct access to the list of items displayed by vendor.
Suggestions Opens the list of options related to suggestions.
Schedules Under development.

Refer to the Order overview topic for details.
