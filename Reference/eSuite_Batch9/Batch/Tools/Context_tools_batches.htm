Create Batches With the Context Tools

eSuite > Context tools

Information

Batches are used to manage changes on multiple items effective at a later time. The eSuite software provides the Batch header table to create batches, the price and cost grids to populate them with items and the batch-type tables to edit the item information.

Batch-related actions are performed from the Batches entry in the launcher or through Context Tools.

The present document focuses on how to manage batches using Context Tools.

How to Display the batch panel in Context Tools

In the Launcher, click the arrow in the Items entry to display its menu and then click on Batches:

  • The list of batch-related entries opens in the Start window.
  • The Batch panel header becomes visible in the Context Tools, showing the Current batches section.

 

Batches in Context Tools

When Batches is selected in the launcher, the Context Tools window shows the Batch panel header with the Current batches section:

Current batches toolbar

The list below shows the icon image, the field name and the functionality of the buttons in the Current batches toolbar:   

ICON  FIELD DESCRIPTION
Select a batch Allows searching and choosing batches among the existing ones. For details, see List of current batches below.
  Create a new batch  Opens the Batch header table for the creation on new batches.
  Clear batch list  Allows emptying the list of batches displayed under Current batches.

List of current batches

To display a list of batches in the Current batches section, follow these steps:

  1. In the toolbar, click Select a batch : the Batch window opens on the desktop.
  2. Enter a criterion in the Search field and/or select a batch type in the Macro field.
  3. Click Go to open the list of batches corresponding to the search criteria.
  4. Choose one or several batches and click OK: the selected batch(es) are displayed under Current batches in Context Tools.

Create and edit batches

The examples below show how to create three types of batches using Context Tools:

  • A price sale batch for a future promotion
  • A price batch for adjusting the regular price at the end of the promotion, and
  • A batch for future cost increase

Create a price sale batch for a future promotion

  1. In the Launcher, click on Items, then on Batches: the Start and Context Tools windows open in the Batch work area.
  2. In Context Tools, click Create a new batch in Current batches toolbar: the Batch header edit window opens in the work area.
  3. Edit the following fields: Description, Start date and time, End date and time, Macro:

  4. Save the changes and close the Batch header window: the price sale batch is listed in the Context Tools under Batch - Current Batches.

  5. Click on Open list to open the Price sale batch window.
    The batch number and type are already set in the respective fields.
  6. Enter the search criteria for the Gatorade products that will be part of the promotion.

  7. Click on Execute to open the new price sale batch window displaying the items compliant with the search.
  8. In the Batch column, enter the price and quantity for each product that will be part of the promotion:
  9. Save the changes.

Create a price batch for the regular price at the end of promotion

  1. In the Launcher, click on Items, then on Batches: the Start and Context Tools windows open in the Batch work area.
  2. In Context Tools, click Create a new batch in Current batches toolbar: the Batch header edit window opens in the work area.
  3. Edit the following fields: Description, Start date and time and Macro:

    Note:
     Make sure no value is enetered in the End date and time fields.
  4. Save the changes and close the Batch header window: the price sale batch is listed in the Context Tools under Batch - Current Batches.
  5. Click on Open list to open the Price batch window.
    The batch number and type are already set in the respective fields.
  6. Enter the search criteria for the Gatorade products that will be part of the promotion.
  7. Click on Execute to open the new price batch window displaying the items compliant with the search.
  8. In the Batch column, enter the regular price and quantity for each product affected by the promotion:
  9. Save the changes.
