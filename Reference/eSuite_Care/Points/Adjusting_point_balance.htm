eSuite Care - Adjusting Point Balances

eSuite Care > Points > Point Balanace

Information

The eSuite Care module offers multiple functions and reports related to customer loyalty, including the point programs and the ability to adjust points easily and is a versatile manner.

NOTE: The eSuite Care module requires configuration to allow the adjustment of secondary point programs.

Adjusting points for customers

The eSuite allows the adjusting of customer's points balance on the points program. It can be used when correcting errors or providing extra points for purchasing incentives.

To adjust points for a customer balance, perform the following:

  1. In the launcher, click on Points: the Start window opens on the desktop.
  2. Select Point balance.
  3. In the wizard, add the necessary criteria (customer account number, name, etc.).
  4. Click Execute to open the balance report.

  5. Select the row containing the point program to be adjusted for the selected customer.
  6. In the bottom menu, click Action and select Adjust points from the menu: the Point adjustment window opens on the desktop.

  7. If necessary, select the right program in the Point program field.
  8. Enter the quantity of points to adjust in the Adjustment field.
    The point balance can be adjusted down (negative number), if configured to allow a negative value. See your reseller for details.
  9. Select a value in the Reason field.
  10. Click Submit at the bottom left of the window to update and display the new values.
