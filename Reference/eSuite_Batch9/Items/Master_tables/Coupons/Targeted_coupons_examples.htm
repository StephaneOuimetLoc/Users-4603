eSuite - Targeted Coupons Examples

eSuite > Items > Master

Information

Targeted coupons are store and vendor coupons given by the system when the specified criteria are reached. They can be for specific customers or they have to be clipped from the store's Web site.

The targeted offers require the Bank_SMSPayment option.

Here are some examples of how to create targeted coupons.

List of examples

  • Buy a 24-pack of Redbull, get $1.00 off
  • Buy two bags of Ruffles chips, get one of the two bags for free
  • Buy a bag of Uncle Ben's instant rice and get a 10% discount
  • Buy an instant rice item and a corn cream item, get $1.00 off (discount applied proportionally on both items)

Buy a 24-pack of Redbull, get $1.00 off

 1. Add the electronic coupon criteria in the main section of the coupon wizard.
  Value field: select Cents off. 
  Discount field: enter 1. 
  Coupon type field: select Targeted. 
  Description field: add a meaningful description. 
  Sub-dpt. field: select a sub-department. 
 
 
2. In the Criteria tab, click Add condition to add the coupon criteria.
  Condition field: select Item sales. 
  Equal to field: enter the pack of Redbull UPC. 
  Minimum measure field: select Quantity. 
  Minimum value field: enter 1. 
  Select Make sure the discount affects the item(s) in this criteria.
This method forces the system to apply the targeted coupon discount on the pack of Redbull. 
  Add a Manual description (optional).
  Click OK to confirm the changes.  
 
 
3. In the Details tab: 
  Funding field: select Store or Vendor.
  Event type field: select Sale. 
  Enter a Start date. 
  Enter an End date. 
 
 
4. Add any necessary description in the Descriptions tab.
 
5. Click Save in the toolbar to finalize the coupon creation.
  The system will automatically assign the record a coupon code.
 
  The Criteria tab will display the following information (Text view and Data view).
      
 
6. Add the offer to a customer account with SMSPayment.

Buy two bags of Ruffles chips, get one of the two bags for free

In this example, all flavors of Ruffles chips must be included in the same promotion; the same promotion identifier must be entered in the Promotion ID field of the POS table for each Ruffles flavor.

 1. Add the electronic coupon criteria in the main section of the coupon wizard.
  Value field: select Buy X get Y free. 
  Buy X field: enter 2. 
  Get Y field: enter 1.
  Coupon type field: select Targeted. 
  Description field: add a meaningful description. 
  Sub-dpt. field: select a sub-department. 
 
 
2. In the Criteria tab, click Add condition to add the coupon criteria.
  Condition field: select Promotion ID sales. 
  Equal to field: enter the promotion identifier for the bags of Ruffles. 
  Minimum measure field: select Quantity. 
  Minimum value field: enter 2. 
  Select Make sure the discount affects the item(s) in this criteria.
This method forces the system to apply the targeted coupon discount on the bags of Ruffles. 
  Add a Manual description (optional).
  Click OK to confirm the changes.  
 
 
3. In the Details tab: 
  Funding field: select Store or Vendor.
  Event type field: select Sale. 
  Enter a Start date. 
  Enter an End date. 
 
 
4. Add any necessary description in the Descriptions tab.
 
5. Click Save in the toolbar to finalize the coupon creation.
  The system will automatically assign the record a coupon code.
 
  The Criteria tab will display the following information (Text view and Data view).
          
 
6. Add the offer to a customer account with SMSPayment.
