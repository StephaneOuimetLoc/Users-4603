mFloor - Selling Configuration

mFloor Configuration

Information

The Selling configuration screen provides the settings for the selling module: Order, Shipping, Multi shipping and Invoicing.

How to change the selling configuration

Modifying the selling configuration settings is the same as that for any other mFloor module. Once modified, the individual settings are saved to a profile. In this, way individual users may have their own settings and preferences.

To access and change the Selling config settings follow the procedure below:

  1. From the Selling menu click or tap Config.
  2. Enter the access code, by default it is normally "999" and click or tap OK.
  3. The Config sale screen presents the user with three identical Goto exist settings for three different sections of the selling module:

    Goto exist

    In each case the Goto exist setting is a numeric field equivalent to the Go to item line field in PDAs and this field essentially changes the behaviour of how an item is added to a selling document:

    1 = Item is added to a new record.

    2 = Item is added to an existing record.

    Further information can be found in the SMSMembers.chm file under the section Hardware/PDA/Advanced/Profiles.

    Transaction printer

    PORTABLE2 is the printer defined by default, you can change this value by selecting another printer from the drop-down menu. Before changing the printer here, you must set up the printer in SMS Pro. You can find more information on PORTABLEx printers on the help document Setup Ini File - Other Printers on SMS Members (Configuration, INI files, Setup, Other Printers).

    Report printer

    You can find more information for this printers on the help document Setup Ini File - Other Printers on SMS Members (Configuration, INI files, Setup, Other Printers).

  4. When all settings have been changed click or tap Save and the Save profile dialog box appears. To discard changes and return to Products menu, click or tap Cancel .
  5. Enter the profile name associated to the user/operator or enter "1" and click or tap OK, to save the profile to the Default profile used by ALL users. More information on profiles can be found in the Profile - Operator table field topic in the SMS Users help file.
  6. On successful saving of the profile the Profile saved dialog box displays the name of the saved profile for confirmation.
