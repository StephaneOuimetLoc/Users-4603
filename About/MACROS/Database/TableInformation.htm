Table informations

@dbHot(FINDTABLE,<TableName>)

This macro checks if a SQL table exist.

Parameter Description Required/Optionnal
TableName Name of SQL table Required
Return value
Return the table name if the SQL table exist. If not, it return blank.
Example
@dbHOT(FindTable,CLK_TAB) will return CLK_TAB

@TOOLS(TABLES_RECORDCOUNT)

This macro returns the list of all SQL SMS tables with its records count.

Parameter Description Required/Optionnal
TableName Name of SQL table Required
Return value
Format : For each table : <TableName>=<RecordCount>
            One table per line.
Example
@TOOLS(TABLES_RECORDCOUNT) will return
ACC_TAB=59
ALLOW_BAT=0
ALT_TAB=27
BTL_TAB=42
CAT_TAB=61
etc.
