eSuite - Launcher

eSuite

Table of Contents

Information

The eSuite includes a launcher that is specific to each module, with some elements that are common across modules.

The launcher is a multi-level element. It contains entries and sub-entries that provide access to information windows.

The launcher is located on the left side of the eSuite application.

Layout

The eSuite launcher includes the following components:

  1. Entry menu
  2. Toolbar

Behavior

This section focuses on the behavior of the launcher entries and on the actions they trigger.

It also describes how the toolbar works.

The entry menu

The launcher displays a menu of clickable entries, some of which are expandable. They display a drop-down menu with sub-entries.

For example, the Items entry expands into a sub-menu with three sub-entries: Master, Batches and Item auxiliaries.

The entries open views in the taskbar and information windows on the desktop.

For example, when you click on the Items > Master entry:

  • The taskbar opens the Items view.
  • The desktop displays the Start page with item-related options.
  • The Context tool window expands the Item and the Dimensions sections.

The toolbar

The toolbar is located at the bottom of the launcher. It consists of three buttons:

  • Show in a new desktop
  • Add to current desktop
  • Collapse menu

The Show in a new desktop button opens the same entry in distinct desktops. This allows working simultaneously on the same entity (customers, items, etc.) while preserving the information in each context.

For example, when you are modifying a group of items and you want to have access to the general item list in a separate view, you may click on Show in a new desktop to open the general list of items in a new desktop. This means that there will be two Items desktops, one showing the group of modified items and the other showing the general list of items.

The Context Tools window will show the information selected in the active desktop.

How it works:

  1. Click on Show in a new desktop.
  2. Select an entry in the launcher to open its view in a new desktop.

The Add to current desktop button adds multiple entries to the same desktop. This allows you to work simultaneously on different views in the same context.

For example, when you are modifying a group of items and you want to access to the general item list in the same view, you may click on Add to current desktop to open the general list of items in the same desktop. That means there will be one Items desktop showing both the group of modified items and the general list of items.

The Context Tools window will show the information selected in the active window.

How it works:

  1. Click on Add to current desktop.
  2. Select an entry to open its view in the current desktop.

The Collapse menu button narrows the menu so that it shows only the entry icons.

How it works:

  1. Click on Collapse menu to narrow the launcher.
  2. You may hover the cursor over the icons to display their name and expand their menu.
  3. Click on Open menu to expand the launcher.
