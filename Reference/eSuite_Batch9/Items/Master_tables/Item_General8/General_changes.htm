eSuite - General Changes

eSuite > Items > Batches > General changes

Information

The General changes feature allows displaying the information that has been updated, deleted or added through the future master tables, including location, classification, coupons and images.

To view the general changes:

  1. In the launcher, select Items.
  2. Click on Master: the Start window opens, displaying the item-related options.
  3. Click on General changes.

The General changes wizard

The General changes wizard includes criteria such as the start and end dates of the changes, item code and description, target, vendor ID, item sub-department and item category. The More options button adds further criteria criteria related to the item whose information has changed.

The wizard checkboxes are selected by default. They allow breaking down the results by main item, POS, location, classification, coupons, images, price and cost.

Start and end dates

The start and end dates represent the period of time that is set for the query. When first accessing the wizard, both fields are automatically populated with the current date and the time range 00:00:00 to 23:59:59.

The information is editable and must be filled out in both fields for the query to be executed. To do so, enter the date and time or select them from the picker:

The following buttons are available:

  • OK confirms the time and date selection.
  • Today sets the current date and time.
  • Now sets the current time and date.

The date format is mm/dd/yyyy.

The time format is hh:mm:ss.

General changes grid

The General changes grid shows specifically where the changes have occurred: date, target, price level, source (batch or table), number and description of the field where changes were made, old values and new ones, identifier.

Use the sorting tool to customize the grid (reorder, add or hide columns):

Here is the description of the columns and the information they may contain.

COLUMN DESCRIPTION
Date The date and time of the changes.
Target The target identifier (store group).
Price level The price level set for each modified item.
Source
Batch The identifier of the batch where changes were applied.
Table The table that was used to make the changes.
Visible when the checkbox corresponding to the table is selected in the wizard.
Changes
Field ID The number of the field that was modified.
Field description The field description.
Old value The original value of the field.
New value The new value of the field.
Identifier The vendor ID and the buying format of the item.

In the bottom menu of the grid:

  • Edit calls up the table where the changes were made.
  • Analyze allows performing item subqueries and linked queries.
