eSuite - Shelf Location Table     

eSuite > Items > Item auxiliaries > Shelf Location    

Table of Contents

Information

There are two tables in eSuite to assist you in locating and managing items on display.

The Shelf Location table specifies where each item is located in the store.

The Shelf table can further refine the item positioning within the shelf, the number of preferred facings,  tags required, plan-o-gram data etc.

You can use shelf data to assist customers looking for something, to expedite the flow of picking of online orders or to assist in maintaining vendor facing requirements to qualify to display money.

Displaying the Shelf Location Table

From the launcher in eSuite Care, Manage, Buy or Sell, choose Items drop down and select Item auxiliaries to display the Start wizard

Select Shelf Location to display the Shelf Location  then select  in the grid tool bar to display the edit table

To create a Shelf Location

1. Display the Shelf Location Edit Table

2. Enter Shelf ID (requirement)

3. Enter Aisle, Side, Section and tag quantity if desired

4. From the dropdown select a label template if desired.

5. To save, click the  or press enter on your keyboard

6. To view your entry, select refresh  from the bottom menu

To modify a Shelf Location

1. From the Shelf Location grid,  select/highlight the record you wish to edit

2. Double click, choose edit from the bottom menu or press F3 on your keyboard

3. Make the modifcations required

4. Save

5. Refresh the grid

Applying a Shelf Location

Once the Shelf Locations have been created, they are applied to the item with addtional positioning details using the Shelf Edit table found in Items

1. To display the Shelf Edit table

Items> Items - General list > Enter the code in the Wizard Any code field > Execute

2. In the grid, right click on the selection and choose Shelf or, from the bottom of the grid,  select the Edit menu drop down, and navigate to Master tables > Shelf

3. Click the  on the Shelf Edit tool bar and note the item number appears in the toolbar and a target is assigned

4. Click the down arrow on the Shelf ID  and choose the Shelf Location from the list or enter a new Shelf ID

5. Add the other parameters for the Shelf accordingly

6. Save

To modify a Shelf

1. Access the Shelf Edit table from the Items grid

2. Modify as required

3. Save

Note: At the time of writing, it does not appear you can have the same Shelf location assigned in different sections, for example a shelf 100 in Grocery as well as shelf 100 in Dairy.
