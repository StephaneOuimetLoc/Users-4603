eSuite Gas - Unmatching Report

eSuite Gas > Unmatching Report

Information

Unmatching reports give an overview of information such as matching or unmatching transactions. Report can be execute from Store and from the Host.

To access the unmatching report:

  1. In the launcher, click on Gas
  2. Click on Unmatching report: the Start window opens, displaying the report-related options
  3. Select the right type of report
Match/unmatch reports:
  • Match/unmatching transactionlist

Match/unmatching transaction list

The Match/unmatching transaction list wizard offers criteria selection that includes date and target.

The Match/unmatching transaction list report displays SMS transactions, console transaction in order to validate information.
