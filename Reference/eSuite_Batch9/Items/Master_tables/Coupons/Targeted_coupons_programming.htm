eSuite - How to Program Targeted Coupons

eSuite > Items > Master > Coupon List

Information

Targeted coupons are store and vendor coupons given by the system when the specified criteria are reached. They can be for specific customers or they have to be clipped from the store's Web site.

Like any item in the system, the targeted coupons have their own UPCs; they are considered negative items.

By default, the targeted store coupons are linked directly to the items like and they affect the item sales.
By default, the targeted vendor coupons are added as new registrations in the transactions and they do not affect the item sales.

Targeted coupons may be programmed in the classical way, namely by entering their POS and price information and defining the criteria by means of the Electronic coupon link table.

However, eSuite offers the Coupon wizard that facilitates the creation of coupons by grouping all criteria (type, method, quantity, amount, etc.) that must be respected for the coupon to be applied.

By default, a targeted coupon is only given once in a transaction, even if its criteria are matched several times, except if it is added more than once in the customer's account.

Only one targeted coupon discount may be applied on an item, but some methods may be set in the targeted coupons definition to change these default behaviors. For details, see Coupons.

Targeted coupon example

The Coupon wizard is used to program the criteria that must be respected for the targeted coupons to be applied.

Below are the steps for programming a targeted coupon. For more examples, see Targeted coupons - examples.

Buy 2 bags of Ruffles chips and a bottle of Gatorade for $7.00

This coupon is offered for 2 bags of any Ruffles chips flavor and 1 bottle of any Gatorade flavor.

All Ruffles flavors must be part of the same promotion (in the example below, 210).
All Gatorade flavors must be part of the same promotion (in the example below, 211).

 1. Add the electronic coupon criteria in the main section of the coupon wizard.
  Value field: select Bundle price. 
  Price: enter 7.00
  Coupon type field: select Targeted. 
  Description field: add a meaningful description. 
  Sub-department field: select the right sub-department. 
   
 
2. In the Criteria tab, click Add condition to add the criteria related to the Ruffles chips promotion.
  Condition field: select Promo ID sales. 
  Equal to field: enter the Ruffles chips promotion identification number. 
  Minimum measure field: select Quantity. 
  Minimum value field: enter 2.
  Multiplication and accumulation field: select Accumulate for bundle or percent.
This method (see Data view in Criteria tab) specifies that the Ruffles chips are part of the bundle.
  Add a Manual description (optional).
Click OK to confirm the changes and close the Add criteria window. 
   
 
3. In the Criteria tab, click Add condition to add the criteria related to the Gatorade promotion.
  Condition field: select Promo ID sales.
  Equal to field: enter the Gatorade promotion identification number.
  Minimum measure field: select Quantity.
  Minimum value field: enter 1.
  Multiplication and accumulation field: select Accumulate for bundle or percent.
This method (see Data view in Criteria tab) specifies that Gatorade is part of the bundle.
  Select Make sure the discount affects the item(s) in this criteria.
This method (see Data view in Criteria tab) forces the system to apply the targeted coupon discount on the bottles of Gatorade. 
  Add a Manual description (optional).
  Click OK to confirm the changes and close the Add criteria window.
   
 
5. In the Details tab:
  Funding field: select Store or Vendor.
  Event type field: select Sale. 
  Enter a Start date. 
  Enter an End date. 
   
 
6. In the Descriptions tab:
Expanded description: enter the expanded description of the promotion (optional).
   
 
7. Click Save in the toolbar.
  The system will automatically assign the record a coupon code.
   
  The Criteria tab will display the following information (Text view and Data view):
           
 
8. Add this offer to a customer account.  

In this example:

  • The customer pays $7.00 for 2 bags of Ruffles chips and 1 bottle of Gatorade.
  • The targeted coupon discount, which is the difference between the total value of the 3 items and $7.00, is applied on the bottle of Gatorade.
