eSuite Manage - Financial Reports

eSuite Manage > Financials

Information

The Financial or store reports produce sales figures to balance the main financial information, for the period and dates requested.

They are available from eSuite Manage:

  1. In the launcher, click on Financials: the Start window opens, displaying the options related to the financial information.
  2. Select the right type of report.

Financial reports:

  • Single total
  • Multi totals
  • Compare totals
  • Transaction details (under development)
  • Orphan movement (under development)
  • Accounting report

 

 

NOTE: For details on the list of totalizers, refer to the Totalizer grid topic in Members.chm (eSuite > System > Edit).

Single total

The Single total wizard offers criteria selection that includes period, date range, totalizer and target. It allows breaking down the information by date and by store and showing only totalizers with no movement. Criteria related to movement may be added with the Add criteria button.

The Single total report displays one store total for the selected totalizer and period.

In the bottom menu:

  • Analyze allows performing subqueries and displaying the transaction details.
    It also provides access to the following reports: Financial - compare totals, Operators - single total and Operators - multi totals.
  • Analyze also allows to drill down into the Total.  You can choose to drill down to
    • Transaction detail (TRS)
    • Saved transaction (SAV)
    • Electronic Journal (XML)

Multi totals

The Multi totals wizard offers criteria selection that includes period, date range, target and totalizer. It allows breaking down the information by date and by store and showing only totalizers with no movement. Criteria related to movement may be added with the Add criteria button.

 

The Multi totals report displays all store totals for the selected period and totalizer range.

In the bottom menu:

    • Analyze allows performing subqueries and displaying the transaction details.
      It also provides access to the following reports: Financial - compare totals, Operators - single total and Operators - multi totals.
    • Analyze also allows to drill down into the Total.  You can choose to drill down to
      • Transaction detail (TRS)
      • Saved transaction (SAV)
      • Electronic Journal (XML)

Compare totals

The Compare totals wizard offers criteria selection that includes period, date and date compare, target and target compare and totalizer range. It allows breaking down the information by date and by store and showing only totalizers with no movement. Criteria related to movement may be added with the Add criteria button.

 

The Compare totals report displays the comparison between different totalizers from the same or a different store, for the selected period.

In the bottom menu:

  • Analyze allows performing subqueries and displaying the transaction details.
    It also provides access to the following reports: Financials - compare totals, Operators - single total and Operators - multi totals.

Accounting report - Store Balance sheet

The Balance sheet wizard offers criteria selection that includes period range and target.

The Balance sheet report displays the store accounting balance sheet for the selected period in a balance sheet style. Debit/Credit columns are shown for every department with a grand total.
