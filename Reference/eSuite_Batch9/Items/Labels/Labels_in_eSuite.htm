eSuite - Label Management

eSuite > Items > Master > Item list

Information

The common modules of the eSuite software offer the tools for generating and printing shelf labels and promotional signs.

The present document shows how to:

  • Generate labels from the item and the batch grids
  • Generate labels from the Labels entry
  • Print labels
  • Display the list of labels to print
  • Restore print jobs

It also describes the layout of the Item browser wizard and the Item label table.

Generating labels

There are two ways of extracting data and generating labels:

  • Based on grid selection (items and batches), through the Labels - Item browser wizard.
  • Based on price events and on RFC scans, from the Labels entry in the launcher.

Generating labels through the Item browser wizard

The label generation based on grid selection can be done with the Labels - Item browser wizard from the general list of items and from the list of batches.

To access the Labels - Item browser wizard from the general list of items:
  1. In the launcher, click the arrow in the Items entry to display its menu.
  2. Click on Master: the Start window opens in the Items work area.
  3. Click on Item list to open the wizard.
  4. Enter the search criteria then click on Execute: results will display in the item grid.
    It is possible to increase the number of items displayed by page with the Page size control in the bottom menu.
  5. Select the items that require label printing.
  6. Click on Action in the bottom menu of the item grid.
  7. Select Generate label: the Labels - Item Browser wizard opens in the work area.
  8. Enter the label criteria.
  9. Click on Execute: the Print label list opens, displaying the labels that are ready to print.

To access the Labels - Item browser wizard from the list of batches:

  1. In the launcher, click the arrow in the Items entry to display its menu.
  2. Click on Batches: the Start window opens in the Batches work area.
  3. Click on Batches to open the wizard.
  4. Enter the search criteria and click on Execute: results will display in the in-store batches grid.
  5. Double click on the batch that requires label printing to display the items.
  6. Select the items that require label printing.
  7. Click on Action in the bottom menu.
  8. Select Generate label: the Labels - Item Browser wizard opens in the work area.
  9. Enter the label criteria.
  10. Click on Execute: the Print label list opens, displaying the labels that are ready to print.

Generating labels from the Labels entry

The Labels entry in the launcher provides tools for editing, printing, displaying the list of the labels and restoring print jobs.

Generate labels based on price events

The Generate labels based on price events utility can be used to identify the items for which the system has to print labels for a specific date. The system takes care of all batches and events defined in the items and compiles the next event for the selected date. For details on this option, refer to the Price events topic.

Generate labels based on RFC Scan

Currently under development.

This option extracts labels for all the items that have been scanned by the RFC label generation page. The items scanned by MFloor or the PDA devices will be available for creating labels.

Printing labels

The label printing tool in eSuite is located in the Labels entry under Tools.

  1. Click on Labels: the Start window opens in the Labels work area.
  2. Click on Print label to open the label wizard:
  3. Select a label format.
  4. Select a target.
  5. If necessary, select the By store checkbox to have the results displayed by store number.
  6. Click on Execute: the Print label list tab opens in the same window, displaying the list of labels that are ready to print.

  7. Select a label and click Action in the bottom menu of the list.
    • View items will open the list of items for which labels will be printed.
    • Delete label will delete the selected label.
    • Print label will open the printing parameters.
  8. Select Print label.
  9. Add the print settings in the Label print options window.
  10. Click Print: a confirmation message will be prompted.

  11. Click Yes to delete the selected labels.
    Deleted labels may be restored with the Restore print jobs utility.
  12. Click No to keep the selected labels in the Print label list.

Showing the print items

To display the list of labels that are ready to print and the items these labels have been generated from:

  1. In the launcher, click on Tools, then on Labels: the Start window opens in the Labels work area.
  2. Select Show print items: the Labels - show print items wizard opens on the desktop.

  3. Enter the search criteria.
  4. Click Execute: the Show print items grid opens in a new tab, displaying the labels that are ready to print and the items that are attached to each label.
    By default, items are grouped by label format:

  5. To edit a label, select the item in the grid and click Edit in the bottom menu: this opens the Item label table.
    See the Editing labels section for details.

Editing labels

Labels in eSuite are edited in the Item label table:

  1. In the launcher, click on Tools, then on Labels: the Start windows opens in the Labels work area.
  2. Select Show print items.
  3. Enter the necessary criteria and click Execute: the Show print items grid opens on the desktop.
  4. Select the item whose label requires editing.
  5. Click Edit in the bottom menu: the Item label table opens on the desktop.
  6. Make any necessary changes and click Save in the top menu.

Restoring print jobs

The Restore print jobs option helps recover deleted labels.

To restore print jobs:

  1. In the launcher, click on Tools, then on Labels: the Start window opens in the Labels work area.
  2. Select Restore print jobs: the Labels - Restore jobs wizard opens on the desktop.
  3. Set the date range.
  4. Define the target for which the old label batches are restored.
  5. Click Execute: the Restore job list opens on the desktop.
  6. Click View to see the list of deleted labels.
  7. Click Restore: the Print label list opens. It displays the list of labels that are ready for printing, including the restored ones.
