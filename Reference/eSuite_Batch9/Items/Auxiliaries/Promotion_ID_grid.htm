eSuite - Promotion ID Grid

eSuite > Items > Item auxialaries

Information

The Promotion ID grid displays the list of promotions that are active in the system and the IDs that help identify these promotions.

Promotion IDs are an efficient way of grouping different items. Note that an item can be part of more than one promotion.

When a record is selected, the Promotion ID grid:

  • Gives access to the list of items that are grouped under the selected promotion ID. The listed items may be edited, analyzed, assigned further promotion IDs and unassigned from the current promotion ID.
  • Gives access to the list of coupons that are linked to the selected promotion ID. The listed coupons may be edited.
  • Allows removing the selected promotion ID and the items/coupons linked to it.

Displaying the Promotion ID grid

In the Launcher, click the arrow in the Items entry to display its menu and then:

  1. Click on Auxiliary tables: the Start window opens in the Auxiliary work area.
  2. Click on Promotion IDs: the tab opens in the same window, displaying the Promotion IDs grid.

Layout

*Click on a field to get information on it.

Content

Here is a description of the Promotion ID grid fields and their functionalities:

FIELD DESCRIPTION
PID The PID field contains the promotion identifier (alphanumeric).
Linked to The Linked to field contains information on the item(s) and coupon(s) that are part the promotion IDs.
Items The Items button gives access to the list of items that are grouped under the same PID.
For details, see the Items linked to PID section below.
Coupons The Coupons button gives access to the list of coupons that are linked to the selected promotion ID.
For details, see the Coupons linked to PID section below.
Remove The Remove button deletes the selected promotion ID.
For details, see the Remove PID section below.
