eSuite - How to Copy Items

eSuite > Items > Master > Item list

Information

The eSuite software provides tools to copy item information from an existing item and create a new item. The information that will be copied to the new record is selected in the Copy item window and edited in the Item creation table.

Once the new items have been created, changes in the database have to be deployed. The new information will be sent to the front-end workstations and the cash registers on the network.

Display the Copy item window

In the launcher, click the arrow in the Items entry to display its menu and then:

  1. Click on Master: the Start window opens in the Items work area.
  2. Click on Item list to open the wizard.
  3. Enter the search criteria.
  4. Click on Execute: results will display in the Items - General list window.
  5. Select the item to duplicate.
  6. Click Copy current item to a new item  in the toolbar or press Ctrl+D on the keyboard.
    The Item Creation table opens on the desktop, with the Copy item window overlapping:


See Creating an item for details on the Item Creation table fields and functionalities.

Copy item window

* Click on a field to get more information on it

Copying the current item to a new item

The Item Creation table allows copying the current item to a new one through the Copy item window.

When a new item is created from an existing one, information needs to be chosen and filtered carefully. Through Check all, Uncheck all and the checkboxes, the Copy item window is the utility that enables this process in an efficient and timesaving manner.

  1. From the item grid, select a record and click on Copy current item to a new item  in the toolbar.
  2. In the Copy item window, enter a new item number in the New UPC field.
    The Confirm button is enabled.
  3. Select the appropriate checkboxes.
  4. Click Confirm to save the changes and close the Copy item window.
  5. The Item Creation table displays the new item with the information copied from the existing one.
  6. Make any necessary modifications.
  7. Save the changes.
  8. Close the Item creation table.
  9. Refresh the item grid to see the new item record.
  10. Deploy changes.
