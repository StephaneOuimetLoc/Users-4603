Table of Contents 

Information

The Like code table is used to maintain codes that are then used to link items through the Like code field of the Price table.

Like codes are used where there are multiple flavours of an item at the same cost, ie chips, gum, pop etc.   Using like codes can greatly speed up item maintenance and creating promotions, as you only need to change one of the items in the group and the system will ask to auto apply to all the others with the same code.

Displaying the Like code table

From the launcher in eSuite Care, Manage, Buy or Sell, choose Items drop down and select Item auxiliaries to display the Start wizard

Select Like Codes to display the Like Code List then select   in the grid tool bar to display the edit Table

 

To create a Like Code

1. Display the Like Code Edit table    

2. Enter Code and Descriptor

Note: A code can be a UPC of one of the items in that Like Code group or a number sequence of your choice

3. To save, click the  in the toolbar or hit enter on your keyboard

4. To review your entry select refresh   at the bottom of the grid

To modify a Like Code

1. From the Like Code list grid, select/highlight the entry you wish to edit

2. Double click on the entry, press F3 on your keyboard or click on Edit from the bottom grid menu.

3. The edit table will open

4. Modify as required

5. Save

6. Refresh

Applying a Like Code

Once created, the corresponding like code number needs to be assigned to the Like Code field in the General tab of the Price Edit table.

Warning: The General tab is not displayed by default, use caution when creating items using the copy feature.
