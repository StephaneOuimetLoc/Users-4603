eSuite Manage - Gift Card Operations

eSuite Manage > Balancing

Information

Open session in LOC SMS eSuite, Management Suite in order to have access to the next gift card operations.

1) Create gift cards

This option allows you to create new gift cards and assign it a value.

  1. Open the Balancing menu and click on Gift
  2. from the menu, select Create gift card range
  3. Gift cards can be created (Only numeric values are accepted):
    • by given a range:
      • From: enter the starting gift card number
      • To: enter the ending gift card number
    • by providing the required quantity:
      • From: enter the starting gift card number
      • Quantity: enter number of gift cards to be created
  4. A value can be assigned to the gift cards to be created
    • Assign the dollar value in the field Value
  5. Click on Execute
  6. The list of gift cards to be created will display. Select the gift cards to be created:
    • if no check box is selected all the given gift card range is going to be created
    • or select all the gift cards in the list using the checkbox on the left of the column title Number
  7. Click on Apply

2) Increase gift card value

This option allows you to increase the gift card value on an existing gift card.

  1. Open the Balancing menu and click on Gift
  2. from the menu, select Create gift card range
  3. The gift cards value can be increased in a given range (Only numeric values are accepted):
    • From: enter the starting gift card number
    • To: enter the ending gift card number
    • Value: Enter the amount in dollars to be added to the gift cards.
    • Add to existing card: Check this checkbox
  4. Click on Execute
  5. The list of gift cards to be modified will display. Select the gift cards to modify using the checkbox at the left of the gift card or select all the gift cards in the list using the checkbox on the left of the column titled Number
  6. Click on Apply

3) Reduce gift card value

This option allows you to reduce the gift card value on an existing gift card.

  1. Ppen the Balancing menu and click on Gift;
  2. from the menu, select Create gift card range.
  3. The gift cards value can be reduced in a given range (Only numeric values are accepted):
    • From: enter the starting gift card number.
    • To: enter the ending gift card number.
    • Value: Enter the amount in negative dollars to be reduced from the gift cards.
    • Add to existing card: Check this checkbox.
  4. Click on Execute
  5. The list of gift cards to be modified will display. Select the gift cards to modify using the checkbox at the left of the gift card or select all the gift cards in the list using the checkbox on the left of the column titled Number.
  6. Click on Apply

4) Send the Gift Card status by email

The Gift Card status emails are sent by SMS using a task, so in order to sent an email you first need to create a task.

Create the task

  1. Open the Balancing menu and click on Gift;
  2. from the Gift card menu, select Balance;
  3. on the Gift card - Balance window click on Add Criteria button;
  4. select the criteria(s) to be used as a filter. Once done click on OK;
  5. once back on the Gift card - Balance window add the values to the criteria fields;
  6. click on button Menu and select Add scheduled tasks, window Edit task will pop up.
  7. On field Task type select Send notification to these customers;
  8. On field Template select eSuite_care_email_pay_gift_expiry.htm.
  9. The task could be programmed to run periodically, select the Recurrence, Due date/time and Adjust date. If you want to run this task once, then don't select the previos fields, eSuite will delete the task once you run it.
  10. Use field Description to identify this task. Click OK

  11. The template eSuite_care_email_pay_gift_expiry.htm will email all the customer Gift cards and the expiry date. The expire date is calculated based on the last time the card was used (rpt_pay_n.F65) plus the parameter defined in:
    Office\Pay_Server_Config.ini

    [SCHEDULE]
Gift_Expiry_days=



Execute the task

The task can be automatically processed by SMS or it can be executed manually. Follow the next steps to manually execute the task.

  1. Open session in LOC SMS eSuite, Management Suite;
  2. open the Tools menu and click on Tasks;
  3. on the Task window click on Task, the Tasks window will open with all the available tasks;
  4. select the task you want to execute and click on buttton Edit, the Edit task window will open;
  5. Click on Execute now.

Known errors

Error Type
Error finding customer record Warning In order to transfer the value from a gift card to another, the gift card needs to be assigned to a customer. Refer to section Associate a gift card to a customer to know how to associate a gift card to a customer.
AI: Account Inactive Error In order to use a Gift card on a transaction, the gift card must have an Active status.
