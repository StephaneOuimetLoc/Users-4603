mFloor - General Configuration

mFloor Configuration

Information

The General configuration screen for mFloor provides lookup and scanner-specific settings. The screen include drop down, text boxes and checkboxes for setting different parameters.

Changing General Config settings

Modifying the general configuration settings is the same as that for any other mFloor module, however the general settings apply to ALL mFloor modules. Once modified, the individual settings are saved to a profile, in this way individual users may have their own settings and preferences.

To access and change the General configuration settings, follow this procedure:

  1. From the Main menu click or tap General config.
  2. Enter the access code (the default value is 999) and click or tap OK.
  3. At the Config SMS screen there are several settings available to the user:

    1. Lookup order
       This setting modifies how items are looked up when scanned or typed in. The default entry (123) refers to (UPC(1), Alternate(2) and vendor(3). Click or tap to open the dropdown and select from the choices provided. Once chosen the Lookup order field will initially show the chosen order e.g.: "UPC and alternate", however once saved and user opens the configuration once again, the Lookup order field will then show the numerical equivalents e.g.: "12"
    2. Scan prefix - The scan prefix is set in the scanner and sent to SMS with the rest of the code (example: S08A for NCR). This field is used to strip this code so that SMS can decode the barcode.
    3. Scan suffix - The scan suffix is set in the scanner and sent to SMS with the rest of the code. This field is used to strip this code so that SMS can decode the barcode.
    4. Strip Check digit from scanner - If the scanner scans all the data and there are no check digits set up in the item codes in the database, then this field must be selected to strip out the check digit. This setting is  either configured in the scanner or must be set in SMS, this field.
    5. Strip last digit from manual entry - Same as the previous field, however it is used when codes are entered manually into mFloor. This is normally selected if the store uses normal UPC codes and cleared (unselected) when store uses their own item barcodes.
    6. Strip check digit from EB - Strips the check digit for those using the Enterprise Browser in the mobile device.
  4. When all settings have been changed click or tap Save and the Save profile dialog box appears. To discard changes and return to Products menu, click or tap Cancel.
  5. Either enter the profile name associated to the user/operator or enter "1" and click or tap OK to save to the Default for ALL users. More information on profiles can be found in the Profile - Operator table field topic.
  6. On successful saving of the profile the Profile saved dialog box displays the name of the saved profile for confirmation.

Return to mFloor - Main menu
