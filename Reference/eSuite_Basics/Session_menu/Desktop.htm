eSuite - Desktop Layout

eSuite > Taskbar > Session

Information

The Desktop layout functionality in eSuite allows you to save the position, size, layout and search parameters of the desktop widgets.

It is available from the Session button in the eSuite taskbar.

Layout

The Desktop layout window contains a drop-down list. It allows you to choose among the saved layouts and display the selected desktop in the work area.
