Information

Categories in eSuite are used for classifying items that have similar characteristics but are assigned to different departments, sub-departments, vendors or to other eSuite divisions.

Juices, for example, can belong to the same sub-department, which makes it difficult to extract them from other items by using the type as a criterion. However, by creating a category called 'Vegetable juices' and linking the vegetable juices to that category, it becomes easy to select the vegetable juices only, regardless of the vendor, department or sub-department they belong to.

Categories assist us to further refine the focus of reporting as well as organize items in a more consumer friendly fashion when needed for catalogs etc.

Displaying the Category table

From the launcher in eSuite Care, Manage, Buy, Sell or Batch, click the down arrow in the Dimensions entry to display the Categories menu:

Click on Categories: the Wizard Start window opens in the Categories work area

Click on List: the Categories tab opens in the same work area, displaying the Category List grid

Click on  in the grid toolbar to display the Category Edit table

Information:  eSuite does not have any category group tables, however the Group 1, 2 & 3 fields are available to facilitate custom reports. Please see the Reference Table for details on how to implement this additional organizational tool.

To create a Category

1. To create a new category, display the Category Edit table

click on  in the Category List toolbar

   

or click on  in the Category Edit tool bar from a populated table

2. The Category Edit table opens in the work area

3. Enter number & suitable descriptor (minimum requirements)

4. Enter desired attributes to assist in refining category management further

Information:  eSuite does not have any category group tables, however the Group 1, 2 & 3 fields are available to facilitate custom reports. Please see the Reference Table for details on how to utilize this additional organizational tool.

Note: Categories can be linked to different sub-departments than the items included in them, as well they may be linked to a taxable and non-taxable sub-department at the same time.  This feature allows for the preservation of a store sub-department structure when that of the Head Office is different.

5. Click the  in the toolbar or hit enter on your keyboard to save

6. From the bottom menu, refresh the grid to see your entry displayed

To modify a Category

1. From the Category List grid, select the entry you would like to edit

2. Click Edit on the bottom menu & choose Category (F3) or press F3 on your keyboard

3. The Category Edit table opens to display the selected category

4. Modify as required

5. Save
