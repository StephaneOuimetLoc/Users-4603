eSuite Buy - Order Overview

eSuite Buy > Buy > Order

Information

The Order entry opens a list of options that provide purchasing features used on backoffice workstations. It includes tools for creating and editing purchase orders.

This document provides an overview of the features specific to each option:

  • Document (Full)
  • Documents
  • Documents - All
  • Documents - New
  • Vendors
  • Vendors - All
  • Items
  • Items - All
  • Suggestions

The Schedules option is currently under development.

Document (Full)

The Document (Full) option in the Start window is enabled when a purchase order is opened from the list of orders (Documents and Documents - All).

When enabled, it opens the Document window with information on the purchase order:

The general information in the purchase order may be edited with the Transaction header table.

The information on the items in the purchase order may be edited with the Transaction line order table.

Documents

The Documents option in the Start window opens the Document search wizard that provides access to the list of purchase orders.

The Document search wizard offers criteria selection that includes vendor ID and name and item code and description. Additional criteria on POS description, sub-department, family and category may be added with the Add criteria button.

When executed, the wizard displays the Open document grid with the list of the open and suspended purchase orders:

In the bottom menu, Action allows to:

  • Open the selected purchase order, which enables the Document (Full) option.
  • Show details of the selected purchase order in the Document window.
  • Select the documents to use with the Multi-document feature in the Context tools window.

Documents - All

The Document - All option in the Start window gives direct access to the Open document grid, without having to go through the Document search wizard.

Documents - New

The Documents - New option in the Start window provides the tools for the creation of purchase orders.

When selected, it opens the New document window, it allows the selection of a vendor and the creation of the document:

See Creating a purchase order for details.

Vendors

The Vendors option in the Start window opens the Vendor search wizard that provides access to the list of vendors and allows the creation of a new PO.

The Vendor search wizard offers criteria selection that includes vendor ID and description and item code and description. Additional criteria on POS description, sub-department, family, category and shopper level may be added with the Add criteria button.

When executed, the wizard opens the list of vendors:

In the bottom menu, Action allows creating a new purchase order for the vendor selected in the list. See Creating a purchase order for details.

Vendors - All

The Vendors - All option in the Start window gives direct access to the Vendor list, without having to go through the Vendor search wizard.

Items

The Items option in the Start window opens the Items - list of items wizard. This provides access to the items of the vendor in the open purchase order.

The Item wizard offers criteria selection that includes item code and description, sub-department, category, family, report code, alternate code. The Vendor ID field is selected and disabled by default and the wizard will open only the list of the items offered by the vendor in the open purchase order. Additional criteria on sub-department, category, family ranges and vendor code, batch and target may be added with the Add criteria button.

When executed, the wizard opens the list of the items offered by the vendor in the current document:

The Case, Unit, Weight and Cost fields are editable, allowing to set the respective values for each item in the purchase order. See Creating a purchase order for details.

Items - All

The Items - All option in the Start window gives direct access to the Item list, without having to go through the Item search wizard.

Suggestions

The Suggestions entry in the Start window offers additional purchasing tools.

The Based on sales is currently available. It suggests items sold by the current vendor according to criteria based on item movement:

When executed, the wizard opens the list of items, allowing to submit suggestions based on movement:

In the bottom menu, Accept opens the list of the items with the suggested quantities. See Suggestions - Based on sales for details.
