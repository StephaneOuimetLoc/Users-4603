Displaying window

Tables in eSuite are accessible from grids:

  1. Acces the respective grid (items, departments, customers, etc.).
  2. Select a record.
  3. Press F3 on the keyboard or click the Edit command in the bottom menu and select an option.

The table will open over the desktop area, displaying the record information.

Layout

By default, tables in eSuite open in their most complex format. This includes the navigation toolbar, the table toolbar and the main section. Depending on the type of record, tables may also include bottom tabs and table grids.

Table toolbar

Each button in the table toolbar triggers an action related to the table layout and content.

Below is the complete list of toolbar controls in tables across eSuite:

ICON FIELD DESCRIPTION
Item number Shows the UPS of the selected item.
Collapse left Allows hiding the table grid.
Active by default when the table opens.
Expand left  Allows showing the table grid.
Collapse bottom Allows hiding the bottom tabs.
Active by default when the table opens.
Expand bottom Allows showing the bottom tabs.
Show store prices Shows the store prices for the selected item.
Available in multi-tenant installations.
Generate price Generates price for the selected item.
Opens the Price generate window.
Add Allows creating a new record.
Delete Deletes the record displayed in the table content.
Save Saves the modifications in the table.
Inactive by default when the table opens for editing an existing record.
Active by default when the table opens for creating a new record.
Undo Cancels all modifications in the table.
Inactive by default when the table opens for editing an existing record.
Active by default when the table opens for creating a new record.
Refresh Refreshes the information in the table.

Content area

Tables display fields with various functionalities and different behaviors that depend on the type of data they contain.

The list below describes the eSuite table fields and their functionalities:

ICON FIELD DESCRIPTION
Text field Allows entering, modifying and deleting information.
Checkbox Allows selecting/unselecting the information in the field.
Date picker Allows entering or selecting a date and inserting it in the field.
Multiple selection combo box Allows entering one or more values or selecting them from a list.
Once the values are selected, the field displays their identifiers.
Single-selection combo box Allows entering one value or selecting it from a list of mutually exclusive values.
Once the value is selected, the field displays the name of that value.

Table grid

The table grid summarizes information on the record selected in the table.

Data is displayed in columns and rows with flexible behavior:

  • Columns are movable and resizable.
  • Records may be sorted in an ascending or a descending order.
  • Column headers allow showing/hiding information based on drop-down selection.
