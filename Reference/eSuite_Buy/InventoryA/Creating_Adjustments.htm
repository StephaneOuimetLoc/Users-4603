eSuite Buy - How to Create Adjustments

eSuite Buy > Inventory > Adjustment > Documents

Information

This document describes how to perform the different required operations in an inventory adjustment document.

The possible operations are:

  • Create and edit an adjustment document
  • Add items to the adjustment document
  • Remove items from the adjustment document
  • Manage quantities into the adjustment document
  • Delete an adjustment document
  • Suspend or close the adjustment document

Steps

To create an adjustment document:

  1. In the Launcher, expand Inventory.
  2. Select Adjustment.
    The Start window opens on the desktop.
  3. Click New.
    The New document window opens.
  4. Select the Store and the Inventory date.
  5. You can specify the Vendor and/or the Sub-department, or leave both blank.
  6. Click Create to confirm the creation of the adjustment document.

When an adjustment document is created:

  • Its status is Open. The transaction number and date are shown in the Context Tools.
     
  • The new adjustment document is listed in the Open document grid that is available from the Documents and Documents - All option in the Start menu.


Adding items to the adjustment document

Once an adjustment document is created, items can be added to it. At this point, the Items, Lines and Header are no longer greyed out.

To add items:

  1. In the Start window, select Items. Then enter the criteria the search for the required items, and click Execute

    or
    select Items - All to display all items.

    The Case, Unit, Weight and Cost fields are editable directly in the grid.

  2. Double-click in any editable field, add the quantities, then press Enter.
  3. Once the values are entered, click .
    The new values are displayed in the Context Tools.

Editing the adjustment document

Before closing the adjustment document, it is possible to revise and edit the information in it. This is done through the Document window.

  1. From the Start menu, select Documents - All.
    The list of opened adjustment documents are displayed in the document grid.
  2. Select a document, and right-click to display the list of options.
  3. Select Open.
  4. From the Start menu, select Lines - All.
    The content of the document is displayed.
  5. Select a line and click Edit.
  6. Modify the required adjustment values and click .
  7. Close the window and click to refresh the values displayed in the grid.

Voiding items

Items can be removed from the adjustment document.

  1. In the Lines section of the Document window, select the item to remove and right-click to display the list of options.
  2. Select Void.
    A confirmation message is displayed.
  3. Click Yes to delete the item from the adjustment document.

Deleting an adjustment document

Opened adjustment documents can be deleted from the Document window:

  1. In the Context Tools window, click .
    A confirmation message is prompted.
  2. Click Yes to delete the document.
    In the Opened documents grid, click to refresh the list; the deleted document is no longer listed.

Suspending an adjustment document

Opened adjustment documents can be suspended and edited at a later time.

  1. Open the Document window for the adjustment documents.
  2. Click in the Context Tools window to suspend the document.

Closing the adjustment document

  1. From the Start menu, click Documents - All.
    The list of adjustment documents is displayed in the grid.
  2. Select a document and open it.
  3. Click in the Context Tools window.
    A confirmation message is prompted.
  4. Click Yes to close the document.
    In the Opened documents grid, click to refresh the list; the adjustment document is no longer listed.

    NOTE: It is not possible to reopen a closed adjustment document.
