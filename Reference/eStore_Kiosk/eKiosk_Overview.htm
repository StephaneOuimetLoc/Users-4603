eKiosk - Overview

eKiosk

Information

The SMS eKiosk option provides a convenient, time-saving kiosk solution accessible on a Kiosk system. The option's design is similar to the eStore concept, which is very much user-friendly and provides specific features for ordering.

Features

The eKiosk contains a specific set of features to be used in a store by the customers. It is widely based on an eCommerce website but the UI is more towards touch screen panels and using a mouse to naviagte. Here is a list of the current features part of the eKiosk:

  • Price Checker: View item's information and price before buying it.
  • Flyer: View and navigate through the flyer directly on the kiosk. Items are mapped to help customer to see their information right away.
  • Ordering: The main feature of the eKiosk is to allow customers to order via these machines.
  • Account Management: When a customer logs in using his account, he can manage his information and see his point balance live.
