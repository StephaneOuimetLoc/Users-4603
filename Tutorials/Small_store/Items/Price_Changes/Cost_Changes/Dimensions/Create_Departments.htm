eSuite - Department Table

eSuite > Dimensions > Departments > List

Table of Contents

Information

The Department table contains information for groups of items, such as taxability, restrictors, miniumum & maximum values etc.  Some points of sale, especially cash registers, use department keys to sell non-coded items. The eSuite software uses departments as a way of grouping both coded items and direct key sales for reporting.

Instead of linking coded items directly to departments, eSuite uses sub-departments for grouping items, and then links the sub-departments to departments. This allows for another report level, regrouping item codes in a more specific manner.

The sub-departments also have their own settings, such as taxability, maximum and minimum entries for sales and voids etc., that are applied when coded items or direct key sales are performed.

Note:   All Departments should have Sub-Departments linked to them

Displaying the Department table

From the launcher in eSuite Care, Manage, Buy, Sell or Batch, click the down arrow in the Dimensions entry to display the Departments menu:

Click on Departments: the Wizard Start window opens in the Department work area.


Click on List: the Departments tab opens in the same work area, displaying the Department List grid

 

Click on in the grid toolbar to display the Department Edit table

    

The Department Edit table is displayed

To create a Department

1. To create a new department, display the Department Edit table

click on  in the Department List grid toolbar

or click on  in the Department Edit toolbar from a populated table

  

2. The new Department Edit table opens in the work area

3. Enter number, description & group (minimum requirements)

Note:  Group allows for further combining of departments for financial reports ie: Group 'Grocery' could be applied to combine departments Grocery, Grocery Taxable, Frozen Foods, Drinks etc. 

4. Apply the desired attributes to the Department by checking off selections and entering values

Note:  Attributes that can be be set via Department can be implemented for a smaller range of goods using Sub-Departments or, item by item via the POS Master table

Information: Please see the Reference Table for programming details on certain attributes, for example: Cashier age, Customer age, Others Flags, Stop discount etc

5. Click the  in the toolbar or hit enter on your keyboard to save

6. From the bottom menu, refresh the grid to see your entry displayed

Warning: Restrictions implemented can be by-passed by operator level 5 or higher.  See the reseller regarding possible alternate configurations. (System.ini file)        

       

To modify a Department

1. From the Department List grid, select/highlight the entry you would like to edit

2. Click Edit on the bottom menu & choose Department (F3) or press F3 on your keyboard


3. The Department Edit table opens

4. Modify as required

5. Save

Warning:  Once created, departments should not be deleted as the link to the reporting history for that department will then be lost.

To add an image for your Department

1. Select or highlight the Department entry you would like to add an image for

2. Click Edit and choose Images from the drop down list

3. Department Images Edit table opens


   

4. Click on in the Department Images Edit toolbar

5. Enter the types, Angles & Size information

6. Browse & select the image of choice

7. Click the  in the toolbar or hit enter on your keyboard to save
