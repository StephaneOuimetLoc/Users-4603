eSuite - Department Images Table

eSuite > Items 

Information

The Department images table allows associating images to departments and use them in xStore (eStore and mStore).

More than one image can be associated with a department, provided that each image has different characteristics from the others.

Displaying the Department images table

In the Launcher, click the arrow in the Dimensions entry to display its menu and then:

  1. Click on Departments: the Start window opens in the Items work area
  2. Click on List: the Departments tab opens, displaying the list of departments
  3. Select a record in the list
  4. Click Edit in the bottom menu to display the options
  5. Select Images: the Department images table opens, overlapping the work area
  6. Click on Add in the toolbar to enable the fields and add a new record

Content

Here is the description of the Department images table fields and their functionalities:

FIELD DESCRIPTION
Main section
Department number The Department number field contains the identifier of the department to which images will be associated.
Types The Types field contains the type of image (Main, Plano, Nutri) to be attached to the department.
Required field.
Angles The Angles field displays the angle in which the image will be attached to the department.
The available options are Main, Front, Left, Right, Top and Bottom.
Required field.
Size The Size field displays the size of the image.
The available options are Thumb, Detail and 50x50.
Required field.
Date The Date field contains the date on which the image(s) will be associated with the department.
Image The Image field contains the image path in the local machine.
It works with the Path field.
It is a required field.
Browse The Browse button allows selecting the image file.
Path The Path field contains the image path in the server.
It works with the Image field.
It is a required field.
Table grid
Type column The Type column contains the types of images associated with the department. 
Angle column The Angle column contains the angles of the images associated with the department.
Size column The Size column contains the size of the image associated with the department.

Associating images to a department

More than one image can be associated to a department, provided that each image has different characteristics from the others.

  1. Select a department in the department list
  2. Open the Department images table
  3. Click Add in the toolbar
    The Department number is added to the table and the fields are enabled.
  4. Select the type, angle and size of the image
  5. Add a date
  6. Click on Browse and select the image file
    The Image field will diplay the image path in the local machine.
  7. Save the changes: the Path field displays the image path in the server

Repeat steps 3 to 7 to link further images to the item. Make sure to select different values for each of these images.
