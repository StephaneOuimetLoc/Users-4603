mFloor - Open Document

mFloor Procurement

Information

The Open document/Open Inventory screen is the main window from which users create the various inventory count documents in mFloor. This document describes how to work with the Open document and perform the following tasks:

  • Creating and adding items to a new count document
  • Opening a previously suspended count document
  • Closing a count document and updating the current inventory on hand
  • Using the main count application menu

Creating and adding items to an Inventory count document

The same procedure is used to create and populate any of the inventory documents available in the Open Inventory add screen.

To create a new count document follow the steps below:

  1. From the mFloor Inventory main menu click or tap Open document
  2. The Open Inventory screen appears and may display a list of existing suspended inventory documents. To scroll through the list of documents swipe the screen or if using a mouse, click and drag on the list.
  3. To add a new count tap or click Add in the upper right hand corner.
  4. The New Inventory screen displays a list of various inventory documents that can be created. The table below gives a brief explanation of each inventory count document along with the abbreviation that displays in the Type field of the document header screen as shown below. Click or tap to choose the desired inventory document according to the required count.
    Inventory Document Type Type field abbreviation Description
    Inventory count COUNT Normal inventory count done at a periodic interval (i.e. yearly/fiscal) where the store inventory is corrected.
    Multi count MULTI Inventory count which adds count of items to previous count.
    Inventory adjust ADJUST Count for loss of merchandise due to damage, miscounts, thefts, etc...
    Production PROD Count an item that is produced using a recipe and automatically updates the inventory of items used in the recipe.
    Back store count BCOUNT Count of back store inventory.
    Back store pickup BPICK Count of item movement from the back store to store floor.
    Back store put away BAWAY Count of item movement into the back store inventory.

Opening a previously suspended Count

When working with count documents users can suspend documents using the application menu or documents are automatically suspended when users tap or click when leaving a count.

To open a previously suspended count document follow the steps below:

  1. If not already in the mFloor application, open the mFloor application and log in.
  2. At the main menu of the mFloor application tap or click Inventory module.
  3. At the Inventory module main menu click Open document.
  4. The Open inventory menu appears showing open/suspended counts, if the list is long, swipe or click and scroll to view all counts.
  5. Tap or click on the desired count to open it.

Closing a count document to update current inventory on hand

Closing a count document removes it from the list of open documents seen in the Open count screen in mFloor.

To close a count document do the following steps below:

  1. If not already done, select the count to be closed from the list of open count documents as explained above.
  2. Once the count is open To close the count tap or click the application menu and click or tap Close document.
  3. mFloor returns to the the Invoice menu.

Using the inventory count document application menu

When a user is in a inventory count document an icon appears for the inventory application menu in the top right hand corner of the screen.  

  • To show items in the current count click or tap Show Lines.
  • To delete the inventory count document click or tap Delete Document.
  • To suspend the count  click or tap Suspend Document.
  • To close and post count document click or tap Close Document.
