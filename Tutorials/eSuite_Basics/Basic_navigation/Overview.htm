Information

The eSuite software uses a uniform interface consistent across all modules. This helps you to easily find information and functionalities.

Layout

The eSuite software featurssssssssses a clean and versatile layout for easier navigation.

The overall interface is split into several sections around a central work area.

NO. SECTION DESCRIPTION
Launcher Contains entries and sub-entries that open information windows on the desktop.
Specific to each module, with some entries common across modules.
Taskbar Contains controls, tabs and commands that allow you to switch between modules, navigate through views and get session information.
Context tools window Collects and displays the information of the record that is selected in the desktop's active window.
Desktop area Allows you to organize your tasks in multiple views. It is configured to display a multitude of information windows, wizards, grids and tools
