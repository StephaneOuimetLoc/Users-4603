eSuite - How to Add Shopper Discounts by Sub-Departments

eSuite > Customers > Customer Auxiliaries > Shopper Levels

Information

The eSuite software provides tools for defining a shopper discount by sub-department.

This discount supersedes the regular sub-department discount. If a shopper discount is specified for a certain shopper level, the system will use this discount. Otherwise, it will cascade back to the sub-department discount. This way, a shopper discount must only be specified when it is different than the sub-department discount. A shopper discount can be specified for each shopper level in each sub-department.

To add a shopper discount by sub-department, the Shopper level table and the Sub-department table have to be open on the same desktop at the same time. Follow the steps below.

Opening the Shopper level table

  1. In the Launcher, expand the Customers entry and select Auxiliary tables.
  2. In the Customer auxiliaries window, click on Shopper levels: the list of shopper levels opens on the desktop.
  3. Select a shopper level.
  4. Click Edit in the bottom menu to open the Shopper level table.
  5. Pin it to the desktop by clicking the Pin icon in the tolbar.

Opening the Sub-department table on the same desktop

  1. In the Launcher toolbar, click Add to current desktop.
  2. In the Launcher, expand the Dimensions entry and select Sub-departments.
  3. In the Sub-department window, click on List: the list of sub-departments opens on the desktop.
  4. Select a sub-department.
  5. Press F3 to open the Sub-department table.
  6. Pin it to the desktop by clicking the Pin icon in the toolbar.
  7. Place the Shopper level and the Sub-department tables next to each other on the desktop.

Adding the shopper level discount to the Sub-department table

  1. In the Sub-department table, click in the shopper discount section.
    The current shopper level is automatically displayed in the Shopper column. 

  2. In the Sub-department table, double click in the Discount column and enter the percentage discount:

  3. Click in the shopper discount section to save the changes.

NOTES:
  • The shopper discount is automatically recalculated if the customer changes at any time during the selling process.
  • The shopper discount replaces the sub-department discount and not the global shopper discount defined in the Shopper level table; both discounts can be given on the same item.
  • The shopper discount is still considered as a sub-department discount and is shown in place of the sub-department discount.
