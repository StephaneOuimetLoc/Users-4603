eSuite - Vendors Table  

eSuite > Dimensions > Vendors > List       

Table of Contents  

Information

The Vendor tabdddddle contains the vendor/supplier information required to facilitate the procurement of products and manage the cost of goods related to those purchases. Vendor information is required to populate the costing record in the Cost master table. 

You may have multiple vendors linked to a single item, with each vendor cost record supporting its own item cost, format and vendor code identifier.  In this instance of multiple vendor options, you only need to flag & maintain the cost record pertaining to your primary vendor at that time.  Both the vendor number and name are displayed when a vendor is selected for an item in the Cost master table.

 

Note: The Vendor table is security referenced, meaning different user security levels allow for different accesses. By default, clerks can only view the list, managers can add new vendors, owners and programmers have complete access.


Displaying the Vendor table

From the launcher in eSuite Care, Manage, Buy, Sell or Batch, click the down arrow in the Dimensions entry to display the Vendors menu

Click on Vendors: the Vendors Wizard Start menu opens in the Vendors work area.

Click on ALL: the Vendors tab opens in the same work area, displaying the Vendor List grid

Click on  in the grid toolbar to display the Vendor Edit table

The Vendor Edit table is displayed

If you do not see the full table, click the down arrow in the Vendor Edit header to fully expand the table, click again to go back to simple view.

To create a Vendor

1. To create a new Vendor, display the Vendor Edit table

click on  in the Vendor List grid toolbar

or click on  in the Vendor Edit tool bar from a populated table

2. The new Vendor Edit table opens in the work area

3. Enter Number, Vendor Name (minimum requirements)

Information:  Number is referred to as Vendor ID in the Cost Master and in many of the wizard & reporting options

Note: Number/Vendor ID does not have to be numeric. It can contain characters, numbers and symbols, except for @ / \ & < > ' " and 'space'. An error message will be displayed if any of these symbols are entered in the field.

4. There are 4 separate tabs at the bottom of the Vendor Edit table, each providing the option to enter targeted vendor criteria.  Enter the information in these fields as required for your needs.

+

Information: The details available in these tabs, for example, address & contact information,  will be of assistance when using the eSuite Buy module as it will displayed on the purchase document.

5. Click the  in the toolbar or hit enter on your keyboard to save

6. From the bottom menu, refresh the grid to see your entry displayed

To search for an existing Vendor

There are two methods of searching for your vendor, with the Vendor Search Wizard or using search from the Vendor List grid

1. To search with the wizard

from the Vendor Wizard start menu choose List

enter your search criteria in any one, or combination, of the fields displayed

Information:
Vendor name and Item description fields use wild cards (*) to allow for partial description

Item code needs to be the expanded 13 digit UPC code

More critera will present a list of addtional fields to select and search by, for example, phone number, city etc. These search fields can be used alone or in conjunction with the 4 default search fields

choose execute or press enter to have your results displayed in the grid

Note: If your search critera provides no results, you can select the Wizard tab to go back to the search and adjust your critera


2. To search from grid

from the Vendor Wizard Start menu, select List and execute the Vendor search with no search critera entered

or select All

this will display a list of all the vendors in the grid

  1. select the magnifying glass
  2. choose the drop down in the in column field and select the search method of choice
  3. enter the criteria accordingly in the search for entry window
  4. press enter
  5. use the First and Next selections to scroll through the results to find the desired record 


To edit Vendor information

1.  Search for the desired vendor

2. From the grid select/highlight the vendor required

3. From the bottom grid menu, select Edit or press F3 on your keyboard

4. The Vendor Edit table displays the selected vendor

5. Modify as desired

6. Save
