eSuite - D3 pivots

eSuite > Launcher entry > Start page > Wizard page > Grid > D3 pivots

Table of contents

Information

The eSuite grid D3 pivots allow you to retrieve the information from pivot tables and structure it in a custom chart. 

Layout

The 3D pivot layout includes the following sections:

  1. D3 pivot chart   
  2. All fields 
  3. Values  
  4. Row labels
  5. Column labels    

The D3 pivot chart

 

The following elements are available:

ELEMENT DESCRIPTION
Vertical axis Shows data sourced from the Row labels section.
Vertical axis label Identifies the data contained in the vertical axis.
Horizontal axis Shows data sourced from the Column labels section.
Horizontal axis label Identifies the data contained in the horizontal axis.
Data labels Show details about the chart data.
Data table Shows the results of the calculations from the Values section.
Groups the results in chart items, based on the Row labels and the Column labels sections.
Chart item tooltip

Provides information about the selected chart item.
Appears when you hover the mouse over the respective chart item.
Legend Shows the metrics of the horizontal axis.

The All fields section

 

The following elements are available:

ELEMENT DESCRIPTION
All fields Shows the window's name.
Collapse fields
  		Hides the following sections: All fields, Values, Row labels, Column labels.
Expand fields
 Expands the following sections: All fields, Values, Row labels, Column labels.
Enabled in the Configurator window.
Field list Contains all the fields that are available in the pivot grid.
Allows you to drag fields and drop them to the Values, Row labels and Column labels sections.

All fields menu

Click on any field to open the menu.

The following actions are possible:

  • Add to Row labels drops the selected fields in the Row labels section.
  • Add to Column labels drops the selected fields in the Column labels section.
  • Add to Values drops the selected fields in the Values section.

The Values section

The following elements are available:

ELEMENT DESCRIPTION
Values Shows the window's name.
Field area Allows you to drop fields from the All fields section.
Retrieves the values in the dropped fields.
Applies custom aggregations such as average, count, max, min, sum and so on.
By default, the function is indicated between brackets next to the field's name.

Click on any field in the Values section to open its menu.

 

The following actions are possible:

  • Move up
    Places the field one step up in the section.
    Always disabled in D3 pivots.
  • Move down
    Places the field one step down in the section.
    Always disabled in D3 pivots.     
  • Move to beginning
    Places the field on the first position in one click.
    Always disabled in D3 pivots.      
  • Move to end
    Places the field on the last position in one click.
    Always disabled in D3 pivots.     
  • Move to Row labels
    Drops the field in the Row labels section.
  • Move to Column labels
    Drops the field in the Column labels section.
  • Move to Values
    Drops the field to the Values section.
  • Remove field
    Clears the field from the Values section.
  • Field settings
    Opens the Field settings window.
    Allows you to customize the settings of the selected field.

The Field settings window

The following fields are available:

FIELD DESCRIPTION
Field settings Shows the window's name.
Source name Text field
Contains the field ID designed in the store procedure.
Custom name Input field
Allows you to modify the default value.
Align Drop-down list
Allows you to choose among Left, Center, Right.
Default value: End.
Format as Drop-down list
Allows you to format the field value in the Pivot table.
Summarize by Drop-down list
Allows you to choose among the custom aggregations.
OK Saves the changes.
Cancel Cancels the changes.

The Label filters menu

 

The label filters open the Label filters window.

The following fields are available:

FIELD DESCRIPTION
Label filter Shows the window's name.
Includes the name of the selected field.
Show items for which the label Drop-down list
Allows you to select an option among those that are listed in the Label filter menu.

Text field
Allows you to enter text.
Case sensitive Checkbox
Sets to case sensitive the value chosen in the Show items for which the label field.
OK Confirms the label filter parameters.
Cancel Cancels the label filter parameters.

FIELD DESCRIPTION
