mFloor - Products

The mFloor Products menu provides tools for users to do quick and advanced searches, perform item maintenance, create batches and store scans.

Products sub-menus

The Products menu presents 5 choices to the user:

  • Quick search - Enables users to quickly search for an item by entering or scanning a code.
  • Advanced search - Enables users to perform a more advanced search using varied criteria.
  • Select batch - Used for creating and editing batches.
  • Store scan - Utility used for scanning items and placing them on shelves.
  • Config (configuration) - Menu for setting configurations specifically for working with items.

Click on the buttons corresponding to the mFloor utility on the screenshot below to read further information on the use of the various menus.

Searching for an item using Quick search

Quick search enables users to find an item by typing in or scanning a barcode and open that item in the Item Information screen.

To search for items using the Quick search

  1. Click or tap the Quick search button.
  2. At the Quick Search screen type a description and press Enter or scan the barcode.
  3. At the list of results click or tap an item to open it in the Item Information screen.
          

Maintaining an Item using Item Information screen and task menu

Once an item is located, the Item Information screen provides menus to access to other views and navigate between the different item tables.

To maintain an item using the Table menu follow the procedure below:

  1. Use Quick search to locate an item and click or tap item to open it in the Item Information screen.
  2. Click or tap the Table menu to view the list of Item tables available.
    • Verify - Returns to the Item Information screen from other item table screens.
    • Main - Item Main table
    • POS - Item POS table
    • Price - Item Price table
    • Cost - Item Cost table
    • Shelf Location - Item shelf location table
    • Alternate Code - Item Alternate Code table
    • Item pictures - Item Bitmaps table
    The screenshots below display the various table views for an item and can be used for reference to find a specific field to change or information to view.

Working with the Task menu

The task menu is accessed from the Item Information screen when an Item is opened using Quick or Advanced search, or from the Scan item screen when doing a Store scan.

The task menu consists of a number of commands as shown in the table below. Some of the entries link to articles on those topics where users can find additional information.

Deploy Change The command deploys all changes done to the Item tables out to the targets set forth in SMS.
Instant Label Command to extract information for label batches and prepares a series of labels for later executing at the backoffice server or workstation, as is normally done.
Print Label Command to print labels immediately for the currently selected item to a printer of the user's choice.
Report by Date Report, which displays item movement for a given time period.
Main menu Returns user to the mFloor Main menu.
   

Report by Date

The Report by date displays item movement for a given time period.

To run the report follow the steps below:

  1. If not already done, log into mFloor and on the Main menu click Products.
  2. Select an item by scanning it, entering the code manually in the Enter item code box or using either Quick seach or Advance search
  3. At the Item Information screen for the selected item click then click Report by Date
  4. At the Item by Date report screen select the Period, Format, Start, and Stop date click or tap Next

A note on Period and date ranges

When choosing Weekly, Monthly or Yearly periods for Report by Date users must leave the default dates in the Start and Stop date fields and use the Past   and Future date buttons to change the date ranges.

If users choose daily, they may put in any range they require.
