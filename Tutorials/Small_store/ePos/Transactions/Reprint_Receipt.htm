Reprint Receipt

ePOS > Functions > Operation > Receipt copy

Table of Content

Information

The receipt copy function prints a duplicate receipt of the current completed transaction.

To print a duplicate receipt, please follow the next steps:

  1. Go to Epos and log in as a cashier.
  2. Run a transaction
  3. complete the transaction
  4. You can either click on Receipt copy on the main POS screen or go to operation > Receipt copy. It will print a duplicate receipt.

Note: To print a duplicate receipt, the transaction has to be completed.
