Text routines

The macros in this section allow you to manipulate text.

@Fmt(<TextOperationAlign>,<Text>)

This macro aligns and formats a text value.

Parameter Description Required/Optionnal
TextOperationAlign Text operation to align a text value. Required
Text Text to be align. Required
Return value
Text aligned
Examples
@Fmt(L,MyText)              will return "MyText"
@Fmt(L10,MyText)           will return "MyText    "
@Fmt(R10:#,MyText)        will return "####MyText"
@Fmt(R10:SPACE,MyText) will return "&nbsp;&nbsp;&nbsp;&nbsp;MyText""
@Fmt(C10,MyText)           will return "  MyText"
@Fmt(L5,MyBigText)         will return "MyBig"
@Fmt(R5,MyBigText)         will return "gText"
@Fmt(C5,MyBigText)         will return "MyBigText"

@Fmt(<TextOperationMultiLines>,<Text>)

This macro formats multiple text lines.
Formatting with this concept will allow to word-wrap the strings and insure that none of the words will be cut.
The macro support an unlimited number of lines.

Parameter Description Required/Optionnal
TextOperationMultiLines Text operation to extract a string in a multi lines text Required
Text Text to be format. Required
Return value
Text of the requested line number.
Examples
@WizSet(TEXT=Computer programming is a way of telling a computer how to do certain things.);
@Fmt(M1:25,@wizGet(TEXT)) will return Computer programming is
@Fmt(M2:25,@wizGet(TEXT)) will return a way of telling a
@Fmt(M3:25,@wizGet(TEXT)) will return computer how to do 

@Fmt(<TextOperationSubString>,<Text>)

This macro returns a substring from a text.

Parameter Description Required/Optionnal
TextOperationSubString Text operation to extract substring from a string. Required
Text Text to be format. Required
Return value
Substring from the text.
Examples
@Fmt(S5:4,Loc Software)   will return  "Soft"
@Fmt(S5,Loc Software))    will return  "Software"

@Fmt(<TextOperationFind>,<Text>)

Operation on text to retrieve text value from a list of key-value.

Parameter Description Required/Optionnal
TextOperationFind Text operation to retrieve text value from a list of key-value. Required
Text List of key-value. Required
Return value
Value of key.
Examples
@WizRpl(KEY="/Item1=AA /Item2=BB /Item3=CC");
@Fmt(F-Item1,@WizGet(KEY)) will return AA
@Fmt(F-Item2,@WizGet(KEY)) will return BB
@Fmt(F-Item3,@WizGet(KEY)) will return CC

@Fmt(NUS,<Text>)

This macro returns an empty string if <Text> is equal to n/a.

Parameter Description Required/Optionnal
Text Text to be format. Required
Return value
Text formatted
Examples
@Fmt(NUS,n/a)       will return ""
@Fmt(NUS,My Text) will return "My Text"

@Fmt(CHR,<IntValue>)

This macro converts an <IntValue> Integer into ASCII character.
You can use this to stop the execution of a script by doing @FMT(CHR,27) which mean Escape, which mean stop.

Parameter Description Required/Optionnal
IntValue Integer value to be converted. Required
Return value
ASCII character
Examples
@Fmt(CHR,64)       will return A
@Fmt(CHR,97)       will return a

@Fmt(ORD,<Character>)

This macro returns the ordinal value of a ASCII character.

Parameter Description Required/Optionnal
Character ASCII Character Required
Return value
Ordinal value
Examples
@Fmt(ORD,A)       will return 64
@Fmt(ORD,a)       will return 97

@Fmt(UPPER,<Text>)

This macros returns the upper case string of <Text>

Parameter Description Required/Optionnal
Text Text to be converted in uppercase. Required
Return value
Text converted in uppercase.
Examples
@Fmt(UPPER,loc software)       will return LOC SOFTWARE

@Fmt(TRIM,<Text>)

This macros removes leading and trailing space of string <Text>.

Parameter Description Required/Optionnal
Text Text to format. Required
Return value
Text converted in uppercase.
Examples
@Fmt(TRIM,      loc software     )       will return "loc software"

@Fmt(LEN,<Text>)

This macros returns the length of string <Text>.

Parameter Description Required/Optionnal
Text Text to be evaluated. Required
Return value
Length of string <Text>
Examples
@Fmt(LEN,loc software)   will return 12
@Fmt(LEN, loc software ) will return 18

@Fmt(POS,<SubString>,<Text>,<N>)

This macros returns the position of the <N>-th occurrence of a <SubString> in a <Text>.
The case is insensitive.
Return 0 if it is not found.

Parameter Description Required/Optionnal
SubString Substring to search for. Required
Text Text where the search is done. Required
N  N-th occurrence of the substring. Optionnal Default: 1
Return value
Position of the <N>-th occurrence
Examples
@Fmt(POS,ware,Loc Software,1)  will return 9
@Fmt(POS,Loc,Loc LOCK Loc,2)   will return 5
@Fmt(POS,Loc,Loc Loc,3)           will return 0
