mFloor - Multi Shipping

mFloor Selling

Information

This document describes how to work with the Multi shipping module and prepare shipping (pickup) of one or multiple orders at the same time.

Shelf Location requirements

In order to properly pick the items, the Shelf Locations of the store must be set in the SMS Auxiliary table.

  • Create all Shelf IDs and assign a section for each of them.

Item requirements

In order to be able to properly use this module, some configuration must be set on each item in the database.

  • The Shelf Location's ID has to be set;
  • The Shelf Location's section has to be set.

Reference of this steps can be found in the Shelf Table document under Item Maintenance in the SMS Members help file.

Reason Codes requirements

In order to use the proper reason codes when changing items' quantity, some specific information has to be set in the Reason codes.

  • The exception must be set to 3591

Reference of this step can be found in the Reason Code Table document under Auxiliary Maintenance in the SMS Members help file.

Shipping process

Step 1: Selecting one or multiple documents

The first step is to select the documents ready to be picked or shipped (1). The list allows to select more than one document.

Once done, press on the white/blue (2) arrow on the top right corner to move to the next step.

Step 2: Select the item to pick

Once in the list of items of all these orders, select the item to be picked. You can enter the name or the UPC code on field Item description or code to search for a specific item or it is also possible to Filter them to make the finding easier.

Item List All With picked items

Step 3: Pick the items

Once in the item's info page, enter the quantity of units picked (1) based on the quantity ordered (U.order). Then, select the shelf location (2) and section (3) where the item was picked. If the item doesn't all fit inside one bag, split the item into multiple to select a different bagging for each part (4). At last, enter the bag number used to pack this item (5).

If the quantity entered in the Unit field is different from the quantity in U.order, a list of reason codes will pop up, select the reason that best explains the change from the list.

Remove an item from the document

If you want to remove the item from the document you can delete it. Select the item and click on the minus button on the top right corner.

Substitute an item

If an item needs to be substituted for any reason (i.e. not enough inventory, item no more available, etc.) follow the next steps:

  1. Select the item to be substituted from the list of items.
  2. Enter a zero (0) in field Unit.
  3. Select the reason code that best explains the change.
  4. Scan or enter the UPC of the substitute item, the added item will be displayed on the screen.
  5. Enter the picked quantity in the Unit field or the weight in Weight field.
  6. Select the reason code that best explains the change.
  7. Select the Shelf location and Section from where the item was picked.
  8. Select or enter the bag number.
  9. The list of items will be shown, the substitute item appears on the list between asterisks (i.e. * Bananas*).

Split and item into multiple bags

If the item doesn't all fit into one bag, it is possible to split the quantity into different bags. Before deciding in which bag the item will be, split the item into different lines using the Split button.

When prompted, enter the quantity put into the CURRENT BAG.

Example: An item has a quantity of five, only two fits in the current bag. Then the rest will be calculated by SMS and posted into a new line where a new bag number can be selected.

Once split, enter the bag number for the current quantity. Then go back to the item list, select the newly created line where it has the splitted quantity, and select the bag number for that second part.

Viewing the selected modifiers

When modifiers were selected on the eCommerce platform, it is possible to view them inside mFloor to select the proper item configuration.

Item Page Modifier Prompt

Add an item

If an item needs to be added to the document (i.e. customer calls to add an item to his order) follow the next steps:

  1. Open the order from the list of documents;
  2. Scan or enter the item UPC, the added item will be shown on the screen.
  3. Enter the picked quantity in the Unit field or the weight in Weight field.
  4. Select the reason code that best explains the change.
  5. Select the Shelf location and Section from where the item was picked.
  6. Enter the bag number.
  7. The list of items will be shown, the substitute item appears on the list between asterisks (i.e. * Bananas*).

Once the item is picked, it will show up as light gray instead of black in the list, which makes it easier to find what has not been picked yet.

Step 4: Close the shipping document

Once all the items are picked, the document is ready for either to review or to be closed.

From the order list, press the close button on the wanted document. All documents that are fully picked will shown in light gray.
Once the button is pressed, the closed document will disappear from the list.
