eSuite - Deploy Changes

eSuite > Tools > Deploy

Information

The Deploy changes job sends changes from the Head office (HO) to the stores and from the backoffice server to the front-end workstations. The Deploy changes job can be used while the store is running.

The posted changes deployed by the Deploy changes job are:

  • Extended item changes (level 2)
  • Item changes (including any changes in name, normal pricing, etc.)
  • Item and level 2 deletes
  • Operator changes (only operators having a level of 6 or lower are deployed)

The Deploy changes job deploys the following tables:

  • Main item
  • Price
  • POS
  • Cost
  • Vendor
  • Mix & match
  • Kit
  • Electronic coupon link
  • Shelf
  • Inventory trigger
  • Scale information
  • Scale ingredients
  • Scale nutrifacts
  • Item alternate codes
  • Operator
  • Customer
  • Contract price
  • Customer alternate codes
  • Scale COOL text
  • Scale COOL short list
  • Scale COOL country list
  • Scale COOL production text

To perform a Deploy changes job:

  1. Click the arrow in the Tools entry in the Launcher to display its menu
  2. Select Deploy: the Start window opens in the Deploy work area
  3. Select Deploy changes

NOTE: The Display changes option in the Start window opens the list of changes that have been made since the last Deploy change job was executed.
