eSuite Manage - EFT Reports

eSuite Manage > Balancing > EFT

Information

The EFT reports produce information related to the EFT transactions that can be performed with eSuite.

They are available from eSuite Manage:

  1. In the launcher, click on Balancing.
  2. Click on EFT: the Start window opens, displaying the options related to the EFT transactions.
  3. Select the right type of report.

EFT reports:

  • Reprint bank report
  • Bank transactions
  • Bank Card report
  • SAF disputes
  • Card number
  • EFT balancing
  • Money order (under development)

Reprint bank report

The Reprint bank report wizard offers criteria selection that includes period range, target and report type. It allows breaking down the information by report type.

The Reprint bank report displays the bank option used, the job type (mode), date, store, terminal, batch, etc.

In the bottom menu:

  • Analyze provides access to Audit - Electronic journal list.

Bank transactions

The Bank transactions wizard offers criteria selection that includes period range, target, authorization number, condition and amount.

The Bank transactions report displays the transaction number, operator, transaction amount, card number, etc.

In the bottom menu:

  • Analyze provides access to Audit - Electronic journal list.

Back to top

SAF disputes

The SAF disputes wizard offers criteria selection that includes period range and target. It allows breaking down the information by report type.

The Pending error report gives the list of all SAF transactions that have been declined by the bank.
When a SAF transaction is declined by the bank, it is not possible to resend this transaction to the bank using eSuite; only the bank can approve the declined SAF transactions. This has to be done over the phone with the bank support.

The Fixed error report gives the list of all SAF transactions that have been approved over the phone with the bank provider.

The Rejected error report gives the list of all SAF transactions that have been declined over the phone with the bank provider.

In the bottom menu of the report:

  • Analyze provides access to Audit - Electronic journal list.

Back to top

Card number

The Card number wizard offers criteria selection that includes period range, target and card search. It allows selecting the card type among credit, debit, private and gift card and filtering data related to transactions paid by check. Additional criteria on gift card, account and marketing may be added with the Add criteria button.

The Card number report displays the transaction number, the operator's sequence, number and short name and the transaction amount.

In the bottom menu:

  • Analyze provides access to the details of the transaction selected in the report.
    The transaction window allows printing the receipt and displaying the keystrokes .

EFT balancing

The EFT balancing allows you to compare side by side Bank and SMS totals from End Of Shift reports (EOS) with the totals in the Revenue totalizers (900-999) and to group them by date / store / terminal. It allows selecting the card type among credit, debit, private and gift card and filtering data related to transactions paid by check. Please see New Bank Totalizer document in the members.chm
