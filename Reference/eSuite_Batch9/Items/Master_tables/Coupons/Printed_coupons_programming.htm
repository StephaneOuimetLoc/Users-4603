eSuite - Printed Coupon Programming

eSuite > Items > Item auxiliaries

Information

Printed coupons are vouchers used in future transactions; when the criteria of the coupon are respected, the coupon is printed at the end of the transaction instead of being subtracted from the transaction total.

Printed coupons are added as new entries (registrations) in the transaction, with the mention This is a printed coupon.

By default, the printed coupons are posted in the totalizer 3520 (function 552). To use another function, enter the function number in the Add using function field in the POS table.

NOTES:
  • The coupons are printed on the coupon printer; be sure the coupon printer is programmed and compatible with the coupon format.
  • The coupons are not printed while in the training mode.

Printed coupon example

The Coupon wizard is used to program the criteria that must be respected for the printed coupons to be applied.

Below are the steps for programming a Cents off printed coupon.

Buy 5 pounds (or kilograms) of ground beef for a total of $50.00 in the meat sub-department in the transaction, get $5.00 off on the ground beef item

 1. Add the electronic coupon criteria in the main section of the coupon wizard.
  Value field: select Cents off. 
  Discount field: enter 5. 
  Coupon type field: select Printed. 
  Description field: add a meaningful description. 
  Sub-dpt. field: select the meat sub-department. 
 
 
2. In the Criteria tab, click Add criteria to add criteria related to the item sales.
  Type field: select Item sales. 
  Equal to field: enter the ground beef item UPC. 
  Minimum measure field: select Weight. 
  Minimum value field: enter 5. 
  Select Make sure the discount affects the item(s) in these criteria.
This method (see Data view in Criteria tab) forces the system to apply the electronic coupon discount on the ground beef item. 
  Click OK to confirm the changes.  
 
 
3. In the Details tab: 
  Funding field: select Store or Vendor.
  Coupon template field: click and select the coupon format.
  Event type field: select Sale. 
  Enter a Start date. 
  Enter an End date. 
 
 
4. In the Descriptions tab:
Expanded description: enter the expanded description of the promotion.  
 
 
5. In the Criteria tab, click Add criteria to add criteria related to the meat sub-department.
  Type field: select Sub-department.  
  Equal to field: select the meat sub-department.
  Minimum measure field: select Dollar.
  Minimum value field: enter 50.  
  Add a Manual description (optional).
  Click OK to confirm the changes.
 
   
6. Click Save in the toolbar to finalize the coupon creation.
  The system will automatically assign the record a coupon code.
 
  The Criteria tab will display the following information (Text view and Data view):
      

In this example:

  • The customer gets $5.00 off when they buy 5 pounds (or kilograms) of ground beef for a total of $50.00 in the meat sub-department in the transaction.
  • Even if the customer buys 10 pounds (or kilograms) of ground beef for a total of $100.00 in the meat sub-department, they will only get $5.00 off. To repeat the electronic coupon, select Based on validated criteria in the Multiplication and accumulation field (Add criteria window of the coupon wizard) for each criterion that must be met.
