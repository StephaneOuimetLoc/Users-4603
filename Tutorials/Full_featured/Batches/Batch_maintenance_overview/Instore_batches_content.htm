Edit Instore Batches

eSuite > Items > Batches

Information

The eSuite software offers the tools to view and modify the content of an instore batch before activating it.

To view and edit the content of an instore batch:

  1. From the Batches entry under Items, access the list of batches.
  2. Select an instore batch and double-click on it: the list of items included in the batch opens on the desktop.
    By default, the first item in the list is selected.


  3. Open the Batch all changes utility to display the information in the current instore batch for the selected item.
    Information is grouped by table.

  4. Click on to open the respective table and modify the information.
  5. Repeat these steps for each item that needs to be modified.
  6. Save the changes with the Save button in the grid toolbar.
  7. Use or in the Batch header table to select another instore batch for modifications.
  8. Activate the instore batches.
