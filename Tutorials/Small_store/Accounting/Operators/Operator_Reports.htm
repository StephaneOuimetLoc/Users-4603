Operator Reports


Information


The Operator reports produce sales figures to balance the main financial information by operator.


ALERT : Now available in eSuite, "Operator Level Control" has been activated. This means that all created users with a "User Level" that is set to a lower "User level" than the other "Users" will no longer be able to change the current assigned "User level" that are higher than theirs. Ex: Logged in user "John Smith" with "User level 5" cannot change anyones "User level" higher than "User Level 5".

The user "John Smith" may now only change the "User levels" for users that are assigned with "User Level 5" and lower. The user "John Doe" may only upgrade "User levels" to "their" current assigened "User level". Ex: John Smith can change Jane Smith's "User level" from an asssinged "User Level 1" up to a "User Level 5" and not any higher.


They are available from eSuite Manage:


  1. In the launcher, click on Operators: the Start window opens, displaying the operator-related options.
  2. Select the right type of report.

Operator List


The Operators List wizard is a two way process


  1. It offers criteria selection that includes period, date range, totalizer and target. It allows breaking down the information by date and by store and showing only totalizers with no movement. Criteria related to movement may be added with the Add criteria button.



2.  Operators list report will populate a list of the operators that include their id number, level, first name, last name, etc.



Note:  To access the operator list then you can choose the 'All' from the operators menu.



Single total


The Single total displays operator total for a particular totalizer. The Single total wizard offers criteria selection that includes period range, totalizer and target. It allows breaking down the information by date and by store and showing only operators with no movement. Additional criteria may be added with the Add criteria button.



The Single totals report displays one total by operator for the selected totalizers, period and target.


In the bottom menu:


  • Edit provides access to the Operator table.
  • Analyze allows breaking down information by date and by store and performing subqueries.
    It also provides access to the list of transactions.

Multi totals


The Multi totals displays all operators totals for any period. The Multi totals wizard offers criteria selection that includes period range, target, compare target and totalizer range. It allows breaking down the information by date and by store and showing only operators with no movement. Criteria related to movement may be added with the More criteria button.



The Multi totals report displays all the accounting values per operator.


In the bottom menu:


  • Edit provides access to the Operator table.
  • Analyze allows breaking down information by date and by store and performing subqueries.
    It also provides access to the following reports: Operators - single total and Operators - compare totals.



 


Compare totals


The Compare totals wizard offers criteria selection that includes period, date, compare date, target, compare target and totalizer. It allows breaking down the information by date and by store and showing only operators with no movement. Additional criteria may be added with the Add criteria button.



The Compare totals report compares totals from the same or a different store.


In the bottom menu:


  • Edit provides access to the Operator table.
  • Analyze allows breaking down information by date and by store and performing subqueries.
    It also provides access to the list of transactions.




Orphan totalizer


The Orphan Totalizer report displays all store report information that does not have a link to the Totalizer table for the selected period. Orphan totalizers may occur when totalizers are deleted, but not the records with which they were attached to.


The Orphan totalizer wizard offers criteria selection that includes period range, start and stop range, and Target range.





Transaction details


The Transaction details wizard offers criteria selection that includes period range, start and stop range, Totalizer range and Target range. Additional criteria may be added with the More criteria button.





Exceptions


The Operator exceptions wizard offers criteria selection that includes period range, target and operator range. It allows breaking down the information by date and by store. Additional criteria may be added with the Add criteria button.



The Operator exceptions report contains information on net sales, bottle returns, corrections, etc.


In the bottom menu:


  • Analyze allows breaking down information by date and by store and performing subqueries.
    It also provides access to the list of transactions in the Electronic journal.

Discounts


The Operator discounts wizard offers criteria selection that includes period range, target and operator range. It allows breaking down the information by date and by store. Additional criteria may be added with the Add criteria button.



The Operator discounts report contains information related to discounts on net sales, global discount, electronic item discount, etc.


In the bottom menu:


  • Analyze allows breaking down information by date and by store and performing subqueries.
    It also provides access to the list of transactions.


  


Statistics


The Operator statistics wizard offers criteria selection that includes period range, target and operator range. It allows breaking down the information by date and by store. Additional criteria may be added with the Add criteria button.



The Operator statistics report displays statistics on sales per operator, for the selected period.


In the bottom menu:


  • Analyze allows breaking down information by date and by store and performing subqueries.
    It also provides access to the list of transactions.




Operator totals report


The Operator totals report displays operator balancing amount [Receive, Deposit, Short & Over] by media and a summary of sales.


The Operators totals report wizard offers criteria selection that includes period range, Operator number and Target range.



The operator field has a drop-down menu that you need to select from. You will need to select only one operator at a time.



The Operator totals report contains information related to drawer information, global discount, electronic item discount, etc.
