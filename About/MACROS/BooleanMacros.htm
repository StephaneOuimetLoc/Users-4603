Boolean routines

The macros in this section allow you to manipulate boolean.

@Fmt(FLG,<Value>,<TrueText>)

This macro returns a specific text if <Value> is equal to 1.

Parameter Description Required/Optionnal
Value Boolean value. Required
TrueText Returns this text if the value is 1. Optionnal
Default: checked
Return value
Boolean formatted value.
Examples
@Fmt(FLG,1)        will return "checked"
@Fmt(FLG,1,True) will return "True"
@Fmt(FLG,0)        will return ""
@Fmt(FLG,0,True) will return ""
