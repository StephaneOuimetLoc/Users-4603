eSuite - Price Level Table                         

eSuite > Items > Item auxiliaries > Price levels

Table of Contents

Information


The eSuite software provides the Price level table to support different price levels.

The Price table can contain different price records for the same item based on the price level.
Customer records in the Customer table are also linked to price levels in the Shopper level table.

Note: Because it contains vital information to the system functionality, the Price level table requires a high security level access (owner and programmer).

Displaying the Price Level table

From the launcher in eSuite Care, Manage, Buy or Sell click the down arrow in the Items entry to display the Item auxiliaries menu:

Note: In the Batch module, Item auxiliaries is in the main menu, not nested under Items

Click Item auxiliaries: the Item auxiliaries Wizard Start window opens in the Auxiliaries work area

Click Price levels selection to display the Levels tab and the Price Level List grid

Click on  in the grid toolbar to display the Price Level Edit table

To create a new Price Level

1. To create a Price Level, display the Price Level edit table

click on  in the Price Level List grid toolbar

click on  in the Price Level Edit window from a populated table

2. Enter value for level (required field)

3. Enter a descriptor to identify the level (not required but strongly recommended)

4. Enter PAL in the warehouse to host target  (required field)

Information: Please see the Reference Table for more details regarding target setup in more complex eSuite installations.

5. Click the  in the toolbar or hit enter on your keyboard to save

6. From the bottom grid menu, click refresh   to see your entry displayed
