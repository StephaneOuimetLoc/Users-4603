eSuite - Like Code Table

eSuite > Items > Item auxialaries > Like Codes

Information

The Like code table is used to maintain the available like codes that are then used to link items through the Like code field of the Price table.

Displaying the Like code table

In the Launcher, click the arrow in the Items entry to display its menu and then:

  1. Click on Auxiliary tables: the Start window opens in the Auxiliary work area.
  2. Click on Like Codes: the Like codes tab opens, displaying the Like Code List.
  3. To modify an existing like code, select it in the list and double-click on it, press the F3 key on the keyboard or click the Edit command in the bottom menu: the Like code edit table opens, overlapping the work area.
  4. To create a like code, click on Add in the bottom menu of the grid: the Like code table opens, overlapping the work area.

Layout

* Click on a field to have information on it.

Content

Here is a description of the Like code table fields and their functionalities:

FIELD DESCRIPTION
Code The Code field contains the like code used to group items.
It is a required field used to uniquely identify records in the Like Code List.
Descriptor The Descriptor field contains the description of the like code.
It is not a required field, however a descriptor should be entered to identify the like code in the Like Code List.
