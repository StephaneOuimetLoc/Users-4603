Information

Customer account in eSuite many be created manually in the database. The Customer table contains the customer information the customer information. In every eSuite module you can access the customer menu on the side panel.

Each account is identified by a code for which there are several rules that have to be observed:

  • The customer field is a 14-charater field. It is mandatory and unique, unless eSuite is configured for the multi-stores (multi-targets). In this case, it is possible to have more than one customer with the same number but with different targets
  • Although the customer ID is a non-numeric field, remember that it must be entered at the point of sale. Some front-end workstations may not have alpha keypads.
  • The customer ID is not editable: once it is created, it cannot be modified.
  • The customer ID and information are automatically deployed to the front-end workstation once the new account information is saved.

From the launcher in eSuite Care, Manage, Buy, Sell or Batch, click the down arrow in the Customer entry to display the Customer menu.

Click on Master: the Start window opens for Customers.

Creating a customer

  1. To create a new customer Click on New. A window will pop asking for a new customer code

  2. Enter customer ID

  3. Enter first and last name

  4. Select a shopper level

  5. Select a risk level

  6. To finalize the customer creation and save the new customer record you must click on the checkmark to save

NOTE: Customer ID, shopper level and risk level are mandatory fields when creating a customer. Once the mandatory fields have been entered, the 'save' checkmark will become available
