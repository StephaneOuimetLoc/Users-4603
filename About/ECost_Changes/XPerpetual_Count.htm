Perpetual Count

Initialize Inventory for a group of Products

1.  Select Items from the launcher.s

2.  Select Items from the item start widget.

3.  Click on More Criteria icon.

4.  Check Item code start and Item code end and press ok.

5.  Enter your Item UPC numbers in Item code start and Item code end in the wizard.

6. Click Execute.

7. Check the checkbox next to the Main Code column header to select all items in the grid.

8.  Select Action at the bottom of the grid.

9.  Select Initialize Inventory.

10.  Click Yes on the confirmation message.

A confirmation message wil be displayed confirming the Initialization has been done.

 


Create a Multi-Count Inventory Document

1.  Select Inventory in the launcher.

2.  Click New.

3.  Choose the date desired for the count/adjustment.

4.  Click Create.

5.  Back on the Multi-Count wizard click Items - All.

6.  Make the necessary adjustment to the number of units per items.

7.  Close or Suspend the document via the  icon in the Context tools window.
