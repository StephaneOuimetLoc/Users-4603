eSuite Care - Points Reports

eSuite Care > Points

Information

The Points reports allow the evaluation and the analysis of the loyalty program experience. These reports are related to the local and centralized point programs, depending on the point configuration.


 

The Points reports are available from eSuite Care:

  1. In the launcher, click on Points.
  2. Select the right type of report from the Start window.
 

Points reports:

  • Point balance
  • Single total
  • Multi totals
  • Orphans
  • Compensation
  • Point summary
  • Point matching
  • Points statement
  • Detailed transactions
  • Point programs

Point balance

The Point balance report offers criteria selection that includes account ID, legal name, date of visit, membership status and range of totalizer balance.

 With this report, you can view zero balances by selecting the checkbox at the bottom (Show zero balances)  allowing to show the zero balances only.

The More criteria button adds criteria related to points, accounts and marketing.

The Point balance report displays the movement on all active point programs by customer, based on the last visit that had a movement in the point program.

In the bottom menu:

  • Analyze opens the list of detailed transactions.
    It provides access to queries for further drill-down to previous results (points and customers).
  • Action allows adjusting points and assigning offers.

Single total

The Single total wizard offers criteria selection that includes period, date range, account ID, legal name, target and totalizer. It allows breaking down the report information by date, store and customer. .

The Single total report displays the totals of the selected point program totalizer.

In the bottom menu:

  • Analyze opens the list of detailed transactions.
    It provides access to queries for further drill-down to previous results (points and customers).
  • Action allows adjusting points and assigning offers.

Multi totals

The Multi totals wizard offers criteria selection that includes period, date range, account ID, legal name, target and totalizer range (issuance, redemptions, adjustments). It allows breaking down the report information by date, store and customer. The More criteria button adds criteria related to points, accounts and marketing.

The Multi totals report opens the total of points by customer per point program, according to the totalizer selected (issuance, redemptions or adjustments).

In the bottom menu:

  • Analyze opens the list of detailed transactions.
    It provides access to queries for further drill-down to previous results (points and customers).
  • Action allows adjusting points and assigning offers.

Compensation

The Point compensation wizard offers criteria selection that includes period, date range, totalizer balance range and target. It allows breaking down information by date, store and customer. The More criteria button adds criteria related to points, accounts and marketing.

The Point compensation report displays the total of points given and redeemed for each active point program movement.

In the bottom menu:

  • Analyze opens the list of detailed transactions.
    It provides access to queries for further drill-down to previous results (points and customers).
  • Action allows adjusting points and assigning offers.

 

NOTE: Compensation reports may be used by accounting to distribute the cost of point programs.


Point Summary

as well as the previous balance, the calculate and current difference.

In the Point Summary Wizard you select a Period to generate the report with a start and stop date.


You can filter by Customer ID, Company/Legal name, Points program, Customer type

and choose to breakdown by date, by store , by customer and to show only difference between calculated and current.

Point Matching

   

This report is grouped by date periods, calculated and compare points given, expired, adjusted, and redeemed amounts for Financial report, Monthly report and Transaction table.

In the Point Matching Wizard you can breakdown by date and by store and have the ability to show the discrepency between Financial and Transaction only.

Once you generate the report it shows the discrepancy between

Financial and Transaction totals for Given points, Expired points, Adjusted Points and Redeemed points.



In the bottom menu:

  • Analyze opens the list of detailed transactions.

Point statement

The Point statement wizard offers criteria selection that includes period, date range, customer ID, customer type, point programs and target. The More criteria button adds criteria related to customer IDs and groups.

The Point statement report displays customer points based on open balance, with an EOM total by points program. The report will be grouped by points program when using the SMSPay server for customer type: centralized since local accounts only look at the records in point program 1.


NOTE: Comma-separated searches such as Customer ID 2,5,7 have been enabled. This will only populate the points statement for point programs 2,5 and 7, even if others exist within that range.

Detailed transactions

The Detailed transactions wizard offers criteria selection that includes period, date range, totalizer range, membership, authorization number and target. The More criteria button adds criteria related to transactions, accounts and marketing.

The Detailed transactions report displays details of transactions carried out through certain customer accounts by means of active point programs, according to points.

In the bottom menu:

  • Analyze opens the electronic journal details and totals.
  • Action allows adjusting points and assigning offers.

Point programs

The system can manage up to 35 point programs, identified by a digit from 1 to 9 or a letter from A to Z. For example, the letter A corresponds to point program 10; B corresponds to point program 11 and so on, up to Z that corresponds to point program 35.

Point program 1 is considered the main one, all the others being secondary point programs.

The Point programs list displays all the 35 point programs in eSuite.

The Points program report displays the program and totalizer number, the totalizer description, the total number of points for each program and the total number of members in each point program.

In the bottom menu:

  • Analyze allows accessing the Point balance report by customer.
