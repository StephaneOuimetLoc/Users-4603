eCommerce - eStore Application

eCommerce

Information

The eStore application is the desktop version of the eCommerce solution that allow users to purchase from an online portal to LOC SMS, without having to maintain two different systems. For the mobile version application, refer to the mStore application.

There are multiple advantages with the eStore application since it is linked to LOC SMS and uses the same database, structure and features. All orders created in eStore application are retrieved, processed and closed at the store level or the warehouse.

The eStore application is built with the Sencha framework and is classified as a web application more than a typical web site.

According to the customer's needs, the eStore application can be customized by using the Sencha tools provided with SMS. A license is required to use Sencha Architect.

The 'Home' page

The Home page is the main web page all visitors see when accessing the eStore application. Creating an account is necessary to navigate in most menus. Here are the features of this page based on the demo of LOC Software:

  • Login
  • Register (removed when user is logged)
  • Account
  • Orders
  • Basket (basket empty before login)
  • Logout (displayed when user is logged)
  • Product description search field
  • Home (Featured Products are displayed when user is logged)
  • Shopping
  • Coupons
  • Recipes
  • Lists
  • Flyer
  • Clubs

The 'Login' window

The Login window allows the user to log in with their username or email and password. Visitors must create an account to navigate in the eStore application. Here are the features of this page based on the demo of LOC Software:

  • Continue as Guest (When pressing Continue as Guess for the first time, a new page called Store locator will ask you to select a store to continue shopping).
  • User name or Email
  • Password
  • Forgot your password?
  • Save password locally
  • Sign in (to validate the information entered)
  • Don't have an account? Sign up

The 'Register' window

The Register window allows the visitor to create an account to navigate in the eStore application. Here are the features of this page based on the demo of LOC Software:

  • First Name
  • Last Name
  • Phone number
  • EMail
  • Password and confirmation
  • My store (Select button)
  • Captcha
  • (Refresh captcha)
  • Receive our weekly flyer view email
  • Submit

The 'Account' page

The Account page gives the user acces to their own personal information for modification. Here are the features of this page based on the demo of LOC Software:

  • Contact (Name, phone number, email, language etc.)
  • Preferences (Handling, Communication preferences, Food preferences)
  • Pickup Store (Stores addresses)
  • Main address
  • Shipping address
  • Cards (All cards linked to the account)
  • Points and clubs (All registered points program)

 

The 'All Orders' page

The All Orders page gives access to the user's older orders and it is also possible to reorder items from this page. Here are the features of this page based on the demo of LOC Software:

  • Status (filter orders by Not completed, Submitted or Closed)
  • Age (filter orders from the last 30 days, 90 days or 180 day)
  • Re-open an order (requires an empty basket)
  • View items (add items to the current transaction)
  • Receipt (view the order receipt)

 

The 'Basket' page

The basket page gives the user access to their shopping cart and displays the current order to complete it. Here are the features of this page based on the demo of LOC Software:

  • Items (modify quantity)
  • X (delete item)
  • Checkout
    • Confirm address
    • Handling
    • Payment
    • Place order
  • Save
  • Delete transaction
  • Change (Current Store)

NOTE: When the order is received in the selected store, an order confirmation email is sent to the user.

The 'Logout' option

The Logout option allows the user to disconnect from the eStore application.

The 'Search' field

The Search field allows the user to search for an item by UPC code or product description. One or more words contained in the product description can be entered.


The 'Featured products' section

The Featured products section of the Home page is used to promote specific products and they are displayed when the user is logged in. Refer to the eCommerce - Items_management topic for details.

The 'Shopping' page

The Shopping page allows the user to add items to the current order or to a saved order when it has been previously selected. Different search methods are available to filter the items. Here are the features of this page based on the demo of LOC Software:

  • General search
  • Purchase history (items from previous orders only)
  • Manual entry (UPC code)
  • Page size (display 16 items, 50 items or 200 items per page)
  • The "Refine your Search" section filters
  • the search by:
    • Departments
    • Categories
    • Promotions (Weekly flyer, Long-term and Manager)
    • Preferences (Organic, Peanut/Nut free, etc.)
  • Featured items (selected in store)

The 'Coupons' page

The coupons page gives access to all regular and targeted coupons available to the connected user. Coupons must be clipped before being used in an order. Here are the features of this page based on the demo of LOC Software:

  • All available offers
  • My offers ready to redeem
  • Old and used offers
  • Page size (display 16 items, 50 items or 200 items per page)
  •  (Refresh)
  • The "Refine your Search" section filters the search by:
    • Promotions (Weekly flyer, Long-term, etc.)
    • Departments
    • Categories

 

NOTE:Electronic coupons are not displayed in the application, they are applied in the order when the criteria are reached.

The 'Recipes' page

The Recipes page displays featured recipes and when the user selects a recipe, the detailed view opens. Search by Category displays all the recipes and allow the user to filter by type of recipes or by serving. Here are the features of this page based on the demo of LOC Software:

  • Search by Category
    • Recipes search
      • Search recipes (description)
      • Page size (display 6, 25 ou 100 items)
      • Clear All Filters (to clear the previous selection)
      • Type of dishes
      • Servings
  • Featured recipes
  • Select recipe (view details)
    • Ingredients
      • Add to list
      • Add to cart
    • Nutrition
    • Directions

NOTE: To create and display recipes as Featured recipes, contact the reseller.

 

 


The 'Lists' page

The Lists page allows the user to add, manage and remove their shopping lists. Here are the features of this page based on the demo of LOC Software:

  • Create a new list
  • Item search (in all the lists)
  • Sort order (Last modified, Last created, Biggest size and Name)
  • Search (of the search or sort defined)
  • Rename (the list)
  • View items (into the list)
  • Duplicate
  • X (delete the list)

The 'Flyer' page

The Flyer page displays the store flyers for the current period, the user can view the pages using the arrows and select an item to view its details. Several flyers can be displayed. Here are the features of this page based on the demo of LOC Software:

  • Select Flyer
  • Arrows (to view the pages)
  • Click on an item to view its details

To displayed the store flyers, contact the reseller.


The 'Clubs' page

The Clubs page allows you to view the balance of each point program for which the user participates as well as the items available for free with their number of points. Here are the features of this page based on the demo of LOC Software:

  • Points program Name & Balance
  • Search (points program)
  • Page size (display 16 items, 50 items or 200 items per page)
  •  (Refresh)
  • Points program's promotional items & total points to redeem them
  • Points missing to obtain an item for free
  • Add to cart

There is a function to send an email to notify the store staff that an online order is received, at the store level, and requires preparation.

To enable this function, an email address must be entered in the operators table (it can be an invidual or a group email address).
