Direct store delivery (DSD) Overview                       

eSuite Buy > Buy > DSD

Information

The DSD entry opens a list of options for the management of direct store deliveries (DSD). The DSD flow represents a complete process in itself and includes the receiving and billing operations.

This document provides an overview of the features specific to each option:

  • Document (Full)
  • Deliveries
  • Deliveries - All
  • Deliveries - New
  • Vendors
  • Vendors - All
  • Items
  • Items - All

The following options are currently under development: Match costs, Returns, Import, A/P posting.

Document (Full)

The Document (Full) option in the Start window is enabled when a billing document is opened from the list of documents (Documents and Documents - All).

When enabled, it opens the Document window with information on the delivery document:

The general information in the receiving document may be edited with the Transaction header table.

The information on the items in the receiving document may be edited with the Transaction line delivering table.

Deliveries

The Deliveries option in the Start window opens the Document search wizard that provides access to the list of open and closed delivery documents.

The Document search wizard offers criteria selection that includes vendor ID and name and item code and description. Documents may be sorted by state: open deliveries and closed deliveries. Additional criteria on POS description, sub-department, family and category may be added with the Add criteria button.

When executed, the wizard displays the Open document grid with the list of the documents that correspond to the chosen criteria:

In the bottom menu, Action allows to:

  • Open the selected document, which enables the Document (Full) option.
  • Show details of the document selected in the Document window.

Deliveries - All

The Deliveries - All option in the Start window gives direct access to the Open document grid, without having to go through the Document search wizard.

Deliveries - New

The Documents - New option in the Start window provides the tools for the creation of delivery documents.

When selected, it opens the New document window, it allows the selection of a vendor and the creation of the document:

See Creating DSD documents for details.

Vendors

The Vendors option in the Start window opens the Vendor search wizard that provides access to the list of vendors.

The Vendor search wizard offers criteria selection that includes vendor ID and description and item code and description. Additional criteria on POS description, sub-department, family, category and shopper level may be added with the Add criteria button.

When executed, the wizard opens the list of vendors:

In the bottom menu, Action allows creating a new purchase order for the vendor selected in the list. See Creating a purchase order for details.

Vendors - All

The Vendors - All option in the Start window gives direct access to the Vendor list, without having to go through the Vendor search wizard.

Items

The Items option in the Start window opens the Items - list of items wizard. This provides access to the items of the vendor in the current document.

The Item wizard offers criteria selection that includes item code and description. The Item in the current document checkbox may be selected to display only the items offered by the vendor in the current document. Additional item criteria may be added with the Add criteria button.

When executed, the wizard opens the list of the items corresponding to the chosen criteria:

 

In the bottom menu:

  • Edit provides access to all item tables.

Items - All

The Items - All option in the Start window gives direct access to the Item list, without having to go through the Item search wizard.
