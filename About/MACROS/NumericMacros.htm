Numeric routines

The macros in this section allow you to manipulate numbers and currency.

@Fmt(<NumericDisplayFormat>,<Value>)

This macro formats number and currency.

Parameter Description Required/Optionnal
NumericDisplayFormat Display format of a numeric value.
 Required
Value The numeric value to format. Required
Return value
The numeric value formatted
Examples
@Fmt(Z$10:2,0.0)       will return ""
@Fmt($10:2,10.253)    will return " $10.25"
@Fmt(#10:2,10.253)    will return " 10.25"
@Fmt(%10:2,10.253)   will return " 10.25%"
@Fmt(Z%10:2,10.253) will return " 10.25%"

@Fmt(!<PadCharacter>:<LengthInCharacters>:<NumberOfDecimal>,<Value>)

This macro formats number/currency and replace white spaces by <PadCharacter>.

Parameter Description Required/Optionnal
PadCharacter The character is used to replace white spaces in the formatted value. Required
LengthInCharacters Maximum length for the formatted value. Required
NumberOfDecimal Number of decimal of the formatted value. Required
Value The numeric value to format. Required
Return value
The numeric value formatted
Examples
@Fmt(!0:10:0,10.253)    wil return "0000000010"
@Fmt(!0:10:1,10.253)    will return "00000010.3"
@Fmt(!Z:5:0,100)         will return "ZZ100"

@Fmt(NUL,<Value>)

This macro returns an empty string if <Value> is equal to n/a.

Parameter Description Required/Optionnal
Value Value to be formatted. Required
Return value
Value formatted
Examples
@Fmt(NUL,n/a)        will return ""
@Fmt(NUL,1234.55)  will return "1234.55" 

@Fmt(TRIMZERO,<Value>)

This macros removes leading zeros of numeric <Value>
This macros also removes leading and trailing space.

Parameter Description Required/Optionnal
Value Numeric value Required
Return value
Value formatted.
Examples
@Fmt(TRIMZERO,000200)            will return "200"
@Fmt(TRIMZERO,000200.200)      will return "200.200"
@Fmt(TRIMZERO,    000200    )    will return "200"

@Fmt(MOD10,<IntValue>)

This macros returns the modulus 10 (Luhn algorithm) check digit of <IntValue>

Parameter Description Required/Optionnal
IntValue Integer value. Required
Return value
Number between 0 and 9.
Examples
@Fmt(MOD10,202)   will return 8
@Fmt(MOD10,203)   will return 5
@Fmt(MOD10,204)   will return 2

@Fmt(MOD10P,<IntValue>)

This macros returns the modulus 10 (Plessey algorithm) check digit of <IntValue>

Parameter Description Required/Optionnal
IntValue Integer value. Required
Return value
Number between 0 and 9.
Examples
@Fmt(MOD10P,202)   will return 2
@Fmt(MOD10P,203)   will return 0
@Fmt(MOD10P,204)   will return 8

@Fmt(HUDON,<IntValue>)

This macros returns the Hudon check digits of <IntValue>

Parameter Description Required/Optionnal
IntValue Integer value. Required
Return value
Check digits
Examples
@Fmt(HUDON,10202) will return 102020
@Fmt(HUDON,9203)   will return 092031

@Fmt(HENING,<IntValue>)

This macros returns the Henning Brauer check digits of <IntValue>

Parameter Description Required/Optionnal
IntValue Integer value. Required
Return value
Number between 0 and 9.
Examples
@Fmt(HENING,202) will return 2
@Fmt(HENING,203) will return 0
@Fmt(HENING,208) will return 9

@Fmt(LTR<Language>,<IntValue>)

This macro converts a number <IntValue> in a literal format.

Parameter Description Required/Optionnal
Language EN   English
FR   French
ES   Spanish		 Optionnal
Default: EN
IntValue Must be greater than zero.
Value 0 returns an empty string.
If the value contains decimals, they will be truncated before processing.		 Required
Return value
String.
Examples
@Fmt(LTR,1001)    will return "One thousand, one"
@Fmt(LTRES,1001) will return "MIL UNO"
@Fmt(LTRFR,1001) will return "Mille un"

@Fmt(TRUNC<FloatingPointNumber>)

This macro returns the integer part of a floating point number.

Parameter Description Required/Optionnal
FloatingPointNumber Number that can contain a fractional part. Required
Return value
Integer value.
Examples
@Fmt(TRUNC,12.55)    will return 12
@Fmt(TRUNC,-10.99)  will return -10
@Fmt(TRUNC,10.001)  will return 10

@Fmt(RANDOM<MaxNumber>)

This macro generates random number larger or equal to 0 and strictly less than <MaxNumber>.

Parameter Description Required/Optionnal
MaxNumber Integer value. Required
Return value
Random integer value.
Examples
@Fmt(RANDOM,5) will return 4
@Fmt(RANDOM,5) will return 1
@Fmt(RANDOM,5) will return 0
