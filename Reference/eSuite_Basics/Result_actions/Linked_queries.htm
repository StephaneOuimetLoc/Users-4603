eSuite - Linked queries

eSuite > Launcher entry > Start page > Grid or list > Analyze

Information

Linked queries in eSuite allow you to perform advanced queries based on a single record selected from the current query results.

The Linked query option is available from the Analyze control menu in most grids and reports. All the windows related to the linked query will open on the same desktop.

Layout

The Linked query page layout is identical across modules. It includes the following sections:

  1. Tab bar
  2. Navigation bar    
  3. Title bar
  4. Content panel    
  5. Resize handles

Content

This section focuses on the components of the Linked query page and on the actions they trigger.

Tab bar

The tab bar is located at the top left of the Linked query page. It includes the Start tab that displays a panel with options based on the current query.

The active tab is marked by a blue horizontal line.

NOTE: On some pages, the Start tab is replaced by the Start icon .

Navigation bar

The navigation bar is located at the top right of the Linked query page.

It displays the following controls:

CONTROL ICON DESCRIPTION
Freeze window size Freezes the window size to that set by the user.
It reverses the action of the Freeze window size button.
Restore to default window size Resets the window size to the default one.
Fit window to full screen Sets the window size to that of the screen.
Restore down from full screen Resets the window size to the default one.
It reverses the action of the Fit window to full screen button.
Fit window to desktop Sets the window size to that of the desktop.
Restore down from desktop Resets the window size to the default one.
It reverses the action of the Fit window to desktop button.
Move window Moves the window around the desktop and screen.
Close window Closes the window.

Title bar

The title bar is located at the top of the Linked query page.

It contains the name of the panel.

Content panel

The content panel is located in the central area of the Linked query page. It contains the options that are based on the entry selected in the launcher.

Resize handles

The Linked query page provides eight resize handles, one in each corner and one on each side.

They allow you to resize the page horizontally, vertically and diagonally.

The resize handles are visible when you hover the mouse over the Linked query window.

Resulting report

When a linked query is performed, the resulting report will:

  • Refresh its data according to the selection you make in the initial report.
  • Allow you to show all report criteria (CTRL + D) and customize your own linked query as you go.
  • Allow you to navigate backwards and add or remove search criteria in the wizard.
  • Allow you to navigate backwards and select a different option in the LINKED window.
