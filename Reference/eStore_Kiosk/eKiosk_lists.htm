The My Lists page of the eKiosk allows the customer to mange their shopping lists.

My Lists Features

This section of the eKiosk contains certain features to help the customer to (view/manage/print/send via email) their Lists. Here is a list of what this page supports:

  • Create a new list:Adding the new list using virtual popup keyboard.
  • Duplicate a list:Duplicates the list with any preferer name.
  • Rename a list:Renames the list with desired name.
  • Delete a list:Compleetly deletes the list from My Lists page.

  • Search in list:Helps the customer to find the specific Item from the list.
  • Add new Item to the preffer list:Allows the customer to add an extra item like a regular line in the list.
  • Send the list to an email address:Allows the customer to send the list for any email address.
  • Printing selected list using Kiosk receipt printer:No need write the list at home, the customer could prepare and just print his list using Kiosk and walk throught the store with printed shopping List.
