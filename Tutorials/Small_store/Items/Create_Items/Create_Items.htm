eSuite - How to Create Items

eSuite > Items > Master > Item list (New)

Information

The eSuite software allows creating items through the Item creation table, which includes the main information in the four master tables: Main item, POS, Price and Cost.    

Once the new items have been created, changes in the database have to be deployed. The new information will be sent to the front-end workstations and the cash registers on the network.

Displaying the Item creation table

In the Launcher, click the arrow in the Items entry to display its menu and then:

  1. Click on Master: the Start window opens on the Items desktop.
  2. Click on New.

     The Item Creation table opens in the work area.
    By default, the only enabled field is Item number. The other fields become available when the new number is entered.


Layout

The Item Creation table consists of four sections: General information, POS information, Price information and Cost information.

By default, when the Item creation table opens, all the fields are disabled except the UPC field. They become enabled once the new item UPC is entered.

Creating an item

  1. Open the Item creation table.
    By default, the only enabled field is Item number.
  2. Type a new item code and press Enter on the keyboard.
  3. If the UPC already exists in the database, the Item Creation table displays the information of the corresponding item.
    If the UPC does not exist in the database, the following confirmation dialog is displayed:
  4. Click Yes to confirm.
    All fields are enabled for editing.
  5. Enter the item information.
  6. Save the changes and close the table.
  7. Refresh the item grid to display the new item.
  8. Deploy changes.

Item information may be further edited from the master tables. You can access the full edit window for each table by clicking to top left corner button of each corresponding zone of the item creation window.

Random weight codes

Random weight codes are identified by the digit 2 at the beginning of the UPC. They contain the product number on 5 digits and the price of the item encoded in the last 5 digits.

In eSuite, the Item creation table provides a tool that generates the random weight codes when creating the new item: the forward slash on the numerical keyboard. For example, by typing 12345 in the Item number field and hitting / on the numerical keyboard, the code will automatically change for 0021234500000.


Freeze current entry button

Currently under development.

The Freeze current entry button is used to freeze a part of the main code (UPC) in the Item number field.

It can be used for creating items from the same manufacturer. Since the first digits of the UPC are the same, they can be frozen and only the last digits will need to be entered for the new items.


To unfreeze the digits, click again on the Freeze current entry button.
