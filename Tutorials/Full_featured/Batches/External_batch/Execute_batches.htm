Execute Batches

How to Execute External Batches

1.  From the BATCH module, select Imports

2.  On the Links start widget, select External Batches 

3.  The List Batch Wizard will come up.  You can refine your seach by selecting from the different search criteria in the wizard

4.  Click Execute.

By Default it will list all OPEN batches.  If you are looking for a old batch, select the OLD checkbox at the bottom of the Wizard

5.  The list of open batches will be presented in the grid

6.  Select the batch(s) you want to execute from the list and click the Execute button in the bottom toolbar

7.  Refresh the list of batches using the refresh button in the toolbar to confirm the selected batches dont appear in the list of Open batches anymore.  You can also check urgent messages to see if any errors occured during execution.
