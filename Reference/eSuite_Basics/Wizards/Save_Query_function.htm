Save Query functionality

The powerful function allows the end-user to save many preferences that they would use on a regular basis. Saving time and enabling other users the ability to benefit from these saved queries in or to provide them with the exact information they desire to view with simply selecting the created Save Query of their choice. No more re-renting the same details over and over every day.

Weather it be for a specific Targets, Vendor, Sub-department, Category that may also include additional details from the More Criteria options, selections from the Grab Fields functionality, preferred result grid Layout and table information such as POS, Price, Cost even the Breakdown of the information (ex: by store, by date, by item). We have also included the fields of the Top Results, Top Pivot records and the Server Timeout(s) which are found when you click on the Gearbox icon to the right of the Execute button. In other words anything this that in available in the Wizard can be a part of the created Save Query Functionality.

The Save Query functionality can be found within all Wizard windows of eSuite. Look for this icon on the right hand side in the blue line just below the "X" icon action to close the Wizard. Once you have entered all you desired field values you may then create your Save Query by clicking on the above mentioned Icon. How to use the Save Query functionality

When inside a Wizard window once you have selected the desired fields and values, click on the Save Query icon you will then be prompted "Saved New Configuration" as seen below

Clicking on this will open a "Save Search" pop-up window. You must enter a name in order to save the query then click on the "Ok" button. You may also choose to have your created Save Query, made "Public" by selecting the checkbox. Allowing all created end-users to access this Saved Query or keep it "Private", which mean only you will have access to this Saved Query should you not select the checkbox.

This example of Saved Query is call, "Query2022" and once saved you may then click on "Execute" button to view your desired query.

This will populated only the details that are pertaining to what you have requested to be viewed in the grid results.

You may create any number of Saved Queries and they will be listed when you click of the icon as seen here below.

You may delete any created Saved Query simply by selecting the query and then clicking back in the Save Query icon and the delete option will be shown toward the bottom as seen in the image below.

Note: The saved queries will be listed in alphabetical order.
Should you like to add other fields to an exsiting Saved Query you may simply re-save the query by selecting, "Save Query2022 Configuration" located just below the "Delete" option in the above mentioned screenshot.
