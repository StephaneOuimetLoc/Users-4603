Review Old Transaction

ePOS > Functions > Transaction > Review

Table of content

Information

The details of completed transactions can be reviewed and displayed on the POS screen. The review function lets you review all completed transactions performed on the front-end workstation.

The following transactions can be reviewed:-

  1. Closed transactions (Regular transactions and refunds)
  2. Backed up transactions

To review a transaction, please follow the next steps:-

  1. Log into ePos
  2. Go to Functions
  3. Go to Transaction
  4. Go to review and select the transaction that you would like to review
  5. It will display the details of the transaction

Note: The reviewed transactions cannot be modified since those transactions are completed and closed. However, they can be cancelled using the cancel previous order function but you will need to have manager access to complete the process.
