eSuite - Item Orphans

eSuite  > Item > Orphans

Information

Orphan items are a result of deleting items while there is still inventory on them, but there is no existing record for these items in the main item table.

Displaying the item orphans report

To access the list of item orphans:

  1. In the launcher, click the arrow in the Items entry to display its menu.
  2. Click on Master: the Start window opens in the Items work area.
  3. Click on Orphans to open the wizard:
  4. Set the necessary criteria.
    To group the report information, select the corresponding radio button in the Group by section:

    To break down the report information by date or by store, check the corresponding box in the Breakdown section:
  5. Click Execute to display the report.

In the bottom menu, Analyze allows displaying the report information by date or by store.
