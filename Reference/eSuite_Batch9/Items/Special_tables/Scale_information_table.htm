eSuite - Scale Information Table

eSuite > Items > Master > Item list

Information

The Scale information table document explains configuration, for information that can be printed on packaged item labels, generated at production scales.

Displaying the Scale information table

In the Launcher, click the arrow in the Items entry to display its menu and then:

  1. Click on Master: the Start window opens in the Items work area.
  2. Click on Item list: the Wizard tab opens, displaying the Items - General list window.
  3. Enter the search criteria and click Execute: results will display in a record grid, in the Items tab of the same window.
  4. Select a record in the list.
  5. Click Edit in the bottom menu to display the options.
  6. Select Scale items, then Scale information: the Scale information table opens, overlapping the work area.

Alternate Description tab

 
*Click on a field to get information on it.

Content

Here is a description of the Scale information table fields and their functionalities:

FIELD DESCRIPTION
Main section
Target The Target field contains the target identifier.
The default value is PAL.
Ingredients info The Ingredients info field links the scale information to the scale ingredient information.
Nutritional info The Nutritional info field links the scale information to the nutrition fact information.
Product life The Product life field contains the number of days before the item should be consumed.
The Best before date is calculated from number of days specified in this field.
Tare A (grams) The Tare A (grams) field contains the tare value in grams of the whole packing.
Tare B (%) The Tare B (%) field contains the tare value in percentage of the individual packing.
Class (report) The Class (report) field contains the number used to classify the item for reporting.
Text link The Text link field links the scale information to scale ingredients table.
Nutrition link The Nutrition link field links the scale information to the nutrition fact information.
Note: To delete the nutrition fact from the item; Place a 0 in this field, then save and deploy.
Label tab
Label format 1 The Label format 1 field contains the number of the primary label format to use for the item at the scales.
Label format 2 The Label format 2 field contains the number of the secondary label format to use for the item at the scales.
Logo ID The Logo ID field contains the identifier of the logo to be printed on the label for the item.
Graphic The Graphic field contains the identifier of the graphic to be printed on the label for the item.
The graphic may be an image of the item.
Message The Message field contains the number of the text message to be displayed on the label of the items or as a marquee at the scales.
Action The Action field contains an action number used to tell large wrapper scales how to handle the item or where to place the label.
It is used specifically for pre-pack environments.
Allergen The Allergen field links the allergen instructions that may be printed on scale labels.
Cooking The Cooking field links the cooking instructions that may be printed on scale labels.
Text 1 The Text 1 field contains user-defined text that may be printed on scale labels.
Text 2 The Text 2 field contains more user-defined text that may be printed on scale labels.
Rotation 1 The Rotation 1 field contains label rotation (clockwise/counter-clockwise).
Rotation 2 The Rotation 2 field contains more label rotation (clockwise/counter-clockwise).
Shelf life The Shelf life field contains the shelf life of the product (day/hour) that may be printed on scale labels.
Force (Shelf life) The Force (Shelf life) field specifies whether the information on shelf life should be entered manually by the operator at the scale when printing.
Barcode The Barcode field ocntains the item barcode number.
Use by The Use by field contains the number of days by which a product should be used.
It may be printed on scale labels.
Force (Use by) The Force (Use by) field specifies whether the Used days should be entered manually by the operator at the scale when printing.
Storage The Storage field contains the storage instructions of the item, which may be printed on scale labels.
COOL Text item label The COOL Text item label field links the item to a COOL text.
This is used to print the COOL text on the item label.
COOL Pre-text The COOL Pre-text field links the item to a COOL pre-text.
The pre-text is just another record in the COOL Text table that is flagged as pre-text. It is typically used to print production information in front of the COOL text.
COOL Short list The COOL Short list field links the item to a COOL short list.
COOL Class The COOL Country/production text field links the item to a class of COOL texts (COOL country and COOL production texts).
COOL Force The COOL Force field pecifies whether the COOL text should be selected manually by the operator (checked) or not (unchecked).
COOL Tracking information The Tracking information field contains the tracking information to print on the label.
The tracking information is store specific and used for traceability.
Alternate description tab
Description 1 The Description 1 field contains the item description that may be printed on scale labels.
Font size (1) The Font size field associated to the Description 1 field contains the font size for the item description 1.
Description 2 The Description 2 field contains an additional item description that may be printed on scale labels.
Font size (2) The Font size field associated to the Description 2 field contains the font size for the additional item description (description 2).
Description 3 The Descriptor 3 field contains an alternate item description that may be printed on scale labels.
It is mainly used for a second language item description.
Font size (3) The Font size field associated to the Description 3 field contains the font size for the alternate item description (description 3).
Description 4 The Descriptor 4 field contains an additional alternate item description that may be printed on scale labels.
It is mainly used for a second language alternate item description.
Font size (4) The Font size field associated to the Description 4 field contains the font size for the additional alternate item description (description 4).
Table grid section
Target column The Target column contains the target identifiers of all scale information records in the table.
