Overview

The eSuite wizards are powerful search assistants. They provide multiple criteria that help you refine searches and customize grids.

This document describes the sections that are common to all eSuite wizards. It will then focus on the following:

  • The Items - General list wizard
  • The Customers - General list wizard
  • The Batches wizard

Layout

Wizards provide a user-friendly layout that facilitates the search process.

The following sections are common to all wizards:

  1. Tab bar
    Enables navigation through content panels.
  2. Navigation bar
    Provides wizard navigation controls.
  3. Title bar and filter controls
    Contains the wizard's name.
  4. Filter controls
    Enable clearing fields and saving queries.
  5. Content area
    Provides inputs that help ypu refine the search.
  6. Command area
    Provides access to the wizard menu and to the Execute button and settings.

Command area

The command area is located at the bottom of the wizard.

The following elements are available:

Element Type Description
Menu Button Offers several options, depending on the wizard:
  • Copy and paste item filters
    Available in the item wizard.
    Opens the Clipboard paste search window.
    Allows you to paste relevant item data that has been copied from e-mail, Word, NotePad, CSV or Excel documents.
    When the query is executed, the copied and pasted data will be displayed in the result grid.
  • Paste customers from Excel
    Available in the customer wizard.
    Opens the Clipboard paste search window.
    Allows you to paste relevant customer data that has been copied from e-mail, Word, NotePad, CSV or Excel documents.
    When the query is executed, the copied and pasted data will be displayed in the result grid.
  • Add scheduled tasks
    Available in wizards across eSuite modules.
    Opens the Edit task window.
    Allows creating new tasks or editing existing ones.
  • Show scheduled tasks
    Available in wizards across eSuite modules.
    Opens the Tasks grid.
  • Clear all search fields
    Erases the values of all the fields in the wizard.
Execute Button Launches the search according to the wizard criteria.
Opens the result grid.
Settings Button Opens the parameters window, with the following input fields:
  • Top result records
  • Top pivot records
  • Server timeout(s)

Information

The wizard is a powerful search assistant. It boosts efficiency by offering multiple search fields and allowing layout customization.

All search-related elements are grouped in the same window.

Displaying the wizard

The wizard is available across modules, from certain entries in the launcher. It can be accessed by clicking on the respective entry in the Launcher and making the right selection in the Start window.

The layout includes the same common elements, with differences according to the selected entry (items, customers, offers, points, campaigns, etc). Below are a few examples:
