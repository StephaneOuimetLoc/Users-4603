Manage Urgent Messages

eSuite System > Messages

Information

Urgent messages provide information about errors, with details on the error types and the steps required to correct the situation.

All urgent messages must be verified and actions should be taken to correct any eventual failures. The urgent message list builds up over time and requires regular maintenance.

Viewing urgent messages

To view an urgent message:

  1. In the launcher, click the Messages entry: the Start window opens in the Messages work area.
  2. Click on Urgent: the Files grid opens, displaying the list of urgent messages.
    For details on the fields and functionalities of the grid, see the Messages topic.
  3. Select a file and double-click on it to open the message.

Layout

*Click on a field to get more information.

Content

The View file contains the urgent message header, with general information on the file and the message content, with details on the file.

Urgent message fields and functionalities:

FIELDS DESCRIPTION
Remove archive bit Under develoment.
Download The Download field allows saving the urgent message on the local machine.
Filename The Filename field displays the name of the file.
Date The Date field displays the date and the time of the urgent message.
Attributes The Attributes field contains the properties of the urgent message:
A= Archive
R=Read-only
To The To field contains the node number of the PC to which the urgent message is sent.
From The From field contains the node number of the PC that sends the urgent message.
Subject The Subject field contains the topic of the urgent message.
Message content The message content shows details on the urgent message topic.

Printing urgent messages

To print an urgent message:

  1. Open the View file window for the urgent message.
  2. Click Print in the toolbar.

  1. Click Print to set the parameters and complete the printing.

Deleting urgent messages

Urgent messages may be deleted from the Files grid:

  1. In the launcher, click the Messages entry: the Start window opens in the Messages work area.
  2. Click on Urgent: the Files grid opens, displaying the list of urgent messages.
  3. Select the message to delete.
  4. Click Delete data in the toolbar.

  1. Click Refresh in the bottom menu of the grid to confirm that the urgent message has been deleted.
