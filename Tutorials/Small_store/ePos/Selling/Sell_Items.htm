Sell Items

Information

The selling process of the Point of sale is used for direct sales as well as the invoice payments. Each of those operations involve opening a new invoice, then entering the items from the current invoice. A new transaction is automatically created when you start entering items or any information; you do not have to click on a specific button or select a specific option.

NOTE: Usually items are scanned. However, sometimes an item must be entered manually. There are 3 different ways to manually enter an item on a front-end workstation: enter the UPC code, enter the vendor code or enter an amount in a sub-department. In this example, we will use the UPC code.

In this document we will be reviewing how to enter a Direct sale using a manual entry of a UPC code.

From the POS main menu:

  1. If applicable, enter customer code and click on Customer
  2. Enter the UPC code and click on PLU
  3. Your Item will display on the detail selling screen 
  4. Click on Subtotal
  5. Select the tender i.e. cash and enter amount

NOTE: There are some cash tender button on the main screen to which you can quickly access i.e. cash, cash $5, cash $10, cash $20.00. You can configure what you see on the main screen through the Button Editor providing you have the correct operator level and security profile.
