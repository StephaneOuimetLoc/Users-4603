mFloor - How to Print Labels

mFloor Item Management

Information

mFloor provides users the ability to print labels, either immediately to a chosen printer by executing Print label or extracting information as an Instant label batch to be executed later from a SMS workstation or backoffice server.

This document explains how to print labels from the mFloor application and covers the following procedures:

  • Printing labels immediately to a chosen printer using Print label
  • Preparing Instant label batches
  • Preparing/Extracting Information using Store Scan
  • Working with Store Scan batch in mFloor
  • Printing Store Scan batch labels in SMS

Printing labels immediately to a chosen printer using Print Label

mFloor has the ability to print labels from the current selected item to a printer of the user's choice, this allows convenient opportunity printing of labels on the floor if one is found missing, torn or with an error. In this case the item can be corrected and a new label printed without leaving the store floor.

To print labels immediately in mFloor to a chosen printer follow the steps below:

  1. If not already done so, log into mFloor and and on the Main menu click Products.
  2. Select an item by scanning it, entering the code manually in the Enter item code box or using either Quick search or Advance search
  3. At the Item Information screen for the selected item click then click Print label
  4. At the Print label screen click the Label field to select a template to use for printing the label.

    NOTE: If the Label field selection list comes up empty when it is clicked, contact the reseller, as a configuration must be made to the SMS system.

  5. Click the Printer field and choose desired printer.
  6. If you want you can check the Print on scan checkbox.
    Once it is enabled, if you scan an item in the UPC field at the top of this page,
    it will automatically print out one label to the chosen printer and use the label template select.
  7. Click to print the label to the chosen printer.

Preparing/Extracting Instant label batches

mFloor offers the ability to extract information for label batches directly in the application and provides a convenient method of preparing a series of labels for when items are missing labels or when labels must be changed in the event of a price change, and then execute those labels at the backoffice server or workstation as is normally done

NOTE: Internet access is required to view video

To extract information as an Instant label batch for a series of items to be executed at the backoffice server or workstation, follow this procedure:

  1. If not already done so, log into mFloor and on the Main menu click Products.
  2. Select an item by scanning it by clicking or entering the code manually in the Enter item code box, or using either Quick search or Advance search
  3. At the Item Information screen for the selected item click then click Instant Label
  4. To add additional items to the Instant label batch, scan items using the scan button or enter the item code manually in Enter item code box.
  5. When all items have been added, on the Instant label screen select the Label template and enter the quantity of desired labels, then tap or click to create the label batch on the backoffice server.

Printing the instant label batches from SMS

Once the Instant label batch has been extracted, printing it at the SMS backoffice server or workstation is the same as printing any label batch and information on the procedure can be found in the article Printing labels - SMS application

Preparing/Extracting Information using Store Scan

mFloor provides the Store scan function that allows users to scan items, place them on the shelves, and create an Item Label extract, which they can then print labels at the SMS backoffice server or workstation

To perform a Store scan, follow these steps:

  1. If not already done so, log into mFloor and on the Main menu click Products.
  2. Tap or click Store scan and then select the Store scan preferences such as Use batch, Initialize location, etc...
  3. Ensure that the Label checkbox is selected and tap or click
  4. At the Scan item screen, scan the first and all other items by tapping or clicking or entering the code manually in the Enter item code.
  5. When finished tap or click to return to the Store scan screen and again to return to the Main menu.

Working with Store scan batch in mFloor using Select batch.

Once the Store scan has been created, users have the ability to work with Store scan batches directly in mFloor where they can view, add/remove or activate items in a Store scan batch.

To work with Store scan batches in mFloor, follow this procedure:

  1. If not already done so, log into mFloor and on the Main menu tap or click Products.
  2. Tap or click Select batch.
  3. At the Open batch screen tap or click on the desired batch.
  4. At the Batch header screen click the Application menu icon to access the commands to work with items in the Store scan batch.
  5. To perform the actions described below on a Batch header tap or click the associated menu action.
    • To add an item to the current Batch, tap or click Add item.
    • To activate an item in the current Batch, tap or click Activate item.
    • To view items which the Store scan adds to the Batch, tap or click View Items to Add.
    • To view items which the Store scan activates in the Batch, tap or click View Items to Activate.
    • To return to main product menu, tap or click Product Menu.

Printing Labels generated by Store Scan in SMS

After having performed a Store scan in mFloor, the list of labels to print is accessible under the Extract label menu in SMS/SMS Pro.

To print the labels generated by the Store scan in mFloor in SMS, follow these steps:

  1. In SMS click the Sell tab and click Labels under the Reports menu or in SMS Pro click the Labels tab.
  2. Click Item label in the Extract label menu, and click Labels from RFC scan.
  3. On the Labels from RFC Scan options screen select a Label Format and a Shelf location range.
  4. Click the Launch button to extract the information for the labels.
  5. Click on Print label in the Execute label menu.
  6. On the List of all labels to print screen click the Print button
  7. On the Label Parameters dialog box select the Printer and select Date added in the upper Order by box, to print labels in the order in which they were scanned, and click Launch.

NOTE: The RF Activity is a report found in the SMS PRO application, under the Tools link of the Journal tab. It displays how many items are scanned and entered manually from mFloor when setting items on the shelf locations, using the settings described in this document.
