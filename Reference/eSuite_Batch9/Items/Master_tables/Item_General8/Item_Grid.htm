eSuite - Items Grid

eSuite > Items > Master > Item list

Information

The item grid lists the results of the search performed through the wizard.

It offers item-related information such as UPC, POS description, price and cost. Data is organized in columns and rows (records).

Displaying the Item grid

In the launcher, click the arrow in the Items entry to display its menu and then:

  1. Click on General: the Start window opens in the Items work area.
  2. Click on Items: the Wizard tab opens in the same window, displaying the Items - General List (see the note below).
  3. Enter the search criteria and click on Execute: results will display in a record grid, in the Items tab of the same window.
NOTE: The eSuite software detects automatically whether the system is single-target or multi-targets. The search fields in the item wizard will display accordingly, with the choice of target in the multi-target installation:

Item grid layout

The default view of the Items grid displays a toolbar (1), the result list (2) and the bottom menu (3):

The toolbar contains buttons that initiate each a specific functionality. Click on the links for more information:

  • Search in results (CTRL + F)
  • Save changes from grid (CTRL + S)
  • Undo changes from grid (Ctrl + Z)
  • Add
  • Excel
  • Pivot
  • Print (Ctrl + P)
  • General settings

The Items grid content consists of multiple views (item code, POS description, price, cost etc.) displayed in columns and rows (records) with flexible behavior:

  • Columns are movable and resizable.
  • They have multi-level headers that allow adding/hiding other columns based on drop-down selection.
  • They can be sorted in an ascending or a descending order.
  • Records may be grouped by dimension (departments and subdepartments).

The menu on the bottom of the grid offers further customizing options. Its functionalities include:

  • Limiting the number of results to display in one grid page.
  • Refreshing the grid view.
  • Establishing the page size (selecting the number of records to display per page).
  • Navigating the grid pages (first page, previous page, page number, next page, last page).

It also displays three buttons: Edit, Analyze and Actions.

Editing Information

Some record information (POS description, regular price and quantity etc.) may be edited directly in the Items grid:

  1. Double-click the information to be edited:
  2. Edit the information and press the Enter key on the keyboard:
  3. Save the new information by clicking the Save changes from grid in the toolbar or the Ctrl + S keys on the keyboard.

Other record information may be edited by accessing the item-related tables with the F3 key on the keyboard or the Edit control in the bottom menu.

Editing information with the Edit control

The Edit control in the bottom menu allows displaying a menu that gives access to all item-related tables. By clicking on Edit, users have access to several menus and options:

The Preferred menu allows editing:

  1. A specific field by way of the Based on the grid position (F3) option. This will open the item-related table specific to that field:
  2. The general item information by means of the All in one basic item option:
  3. The main item information, the POS information, the price and the cost when the option Multi core tables is selected:

  4. The Master tables menu allows accessing and editing the following tables:

    1. Main item
    2. POS Description
    3. Price
    4. Cost
    5. Shelf
    6. location

    The Special items option allows accessing and editing the following tables:

    3. Future sales price
    4. Future in-store price

    The Cost batch menu allows accessing and editing the following tables:

    2. Future allowance
    3. Future rebate

    5. Future break quantity
    6. Future bill back
    7. Future general cost

    Analyzing item-related information

    The Analyze control on the bottom of the item grid offers two options for obtaining item information based on user-defined criteria:

    1. The Item subquery option groups and breaks down the information contained in the entire item grid.
    2. The Item-linked query allows joining additional tables with information related to the item that has been selected in the grid.

    Performing actions on grid items

    The Actions button displays a menu allowing users to assign or unassign promotion IDs to the selected items.
