Manual coupon programming

Manual coupons are store and vendor coupons that need to be keyed in manually by the cashier, unlike electronic coupons which are issued by the system at the end of the transaction. Like any item in the system, manual coupons have their own UPCs; they are considered negative items.

  • By default, the manual store coupons are added as new registrations in the transactions and they affect the net sales.
  • By default, the manual vendor coupons are added as new registrations in the transactions and they do not affect the net sales.

Manual coupons may be programmed by entering their POS and price information and defining the criteria by means of the Electronic coupon link table. The conditions are validated when the cashier enters or scans the manual coupon.

However, eSuite offers the Coupon wizard that facilitates the creation of coupons by grouping all criteria (type, method, quantity, amount, etc.) that must be respected for the coupon to be applied.

For the manual coupons to be issued, the Apply coupon link field of the Shopper level table must be checked in the corresponding shopper levels.

Each item sold may be associated with more than one manual coupon, but only one manual coupon discount may be applied on an item. The multiplication methods Based on validated criteria and Based on quantity sold are not used, meaning thar the manual coupon is not multiplied even if the criteria are reached more than once. To be able to issue the manual coupon a second time, the cashier has to enter the manual coupon a second time.

However, some methods may be set in the manual coupon definition to change the default behavior. For details, see Coupons.

Manual coupon example

The Coupon wizard is used to program the conditions that must be respected for the manual coupons to be applied.

Many different manual coupons can be programmed; however, there are some limitations compared to electronic coupons.

Below are the steps for programming a Cents off manual coupon. For more examples, see Manual coupons - examples.

Get $0.50 off when you purchase a small coffee

1. Add the electronic coupon criteria in the main section of the coupon wizard.
Value field: select Cents off.
Discount field: enter $0.50.
  Coupon type field: select Manual. 
  Description field: add a meaningful description.  
  Sub-dpt. field: select the right sub-department.  
 
 
2. In the Criteria tab, click Add criteria.  
  Type field: select Item saless.  
  Equal to field: select the small coffee item. 
  Minimum measure field: select Quantity.  
  Minimum value field: enter 1. 
  Select Make sure the discount affects the item(s) in these criteria.
This method (see Data view in Criteria tab) forces the system to apply the manual coupon discount on the coffee item. 
  Add a Manual description (optional).
  Click OK to confirm the changes.  
 
 
3. In the Details tab:  
  Funding field: select Store or Vendor.
  Event type field: select Sale.  
  Enter a Start date.  
Enter an End date.  
 
 
4. In the Descriptions tab:
Expanded description: enter the expanded description of the promotion.  
 
 
 5. Click Save in the toolbar to finalize the coupon creation.
  The system will automatically assign the record a coupon code.  
 
  The Criteria tab will display the following information (Text view and Data view):
      

In this example:

  • The customer buys a small coffee that costs $1.25.
  • As the item is part of the Cents off promotion, the cashier applies the manual coupon of $0.50.
  • The customer will only pay $0.75 plus taxes.
