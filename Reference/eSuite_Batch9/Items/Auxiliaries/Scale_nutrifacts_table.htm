eSuite - Scale Nutrifact Table

eSuite > Items > Item auxiliaries > Scale tables

Information

The Scale nutrifacts table contains the nutritive elements for items that are made in the store. The Scale nutrifacts table works with the Scale information table.

It includes a description, a declaration of the serving, a label template, as well as the mandatory (energy, proteins, fat and carbohydrates) and voluntary nutrients.

Displaying the Scale nutrifacts table

In the Launcher, click the arrow in the Items entry to display its menu and then:

  1. Click on Auxiliary tables: the Start window opens in the Auxiliary work area.
  2. Click on Scale tables and then on the Scale nutrifacts: the Scale nutrifacts tab opens in the same window, displaying the Scale nutrifacts List.
  3. To modify an existing scale nutrifact, select it in the list and double-click on it, press the F3 key on the keyboard or click the Edit command in the bottom menu: the Scale nutrifacts table opens, overlapping the work area.
  4. To create a new scale nutrifact, click on Add in the bottom menu of the grid: the Scale nutrifacts table opens, overlapping the work area.

The Sub-department list

The Sub-department (S-Dept) links the scale nutrifacts to a sub-department.

In most POS systems, item sales are linked to sale categories for reporting purposes. Because different POS systems use different terminology for sale links, it is important to understand the eSuite software terminology and how it relates to the POS system. The eSuite software uses a three-tier sale links level organization:

  1. Item sales are accumulated in sub-departments;
  2. Sub-department sales are accumulated in departments (a department's sale total contains sales from one or more sub-departments);
  3. Department sales are accumulated in group sales (a group's sale total contains sales from one or more departments).

The eSuite software sale link levels (sub-departments, departments and groups) are defined in the Sub-department and Department tables.

Whether or not the POS system uses the same terms, eSuite software requires that items be linked to a sub-department or to a first sale link level.

To link the Scale Nutrifacts information to a sub-department, type the sub-department number in the S-Dept field or click on the chevron in the S-Dept field to select a sub-department from the drop-down list (records must already exist in the Sub-department table for the list to be displayed).

The Calories from saturated fat field

The Calories from saturated fat field contains the energy value, expressed in calories (cal), which comes from saturated fat associated to the defined serving. The energy value that comes from saturated fat is a voluntary nutrition information.

The number of calories coming from saturated fat is part of the number of calories coming from saturated and trans fats.

Example:

  • Calories from saturated + trans = 50
  • Calories from saturated fat = 30

This means that the serving provides 50 calories coming from saturated and trans fats, from which 30 calories come from saturated fat only.
