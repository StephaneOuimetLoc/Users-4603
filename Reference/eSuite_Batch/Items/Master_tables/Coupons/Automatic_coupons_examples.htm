eSuite - Automatic Coupons Examples

eSuite > Items > Coupons list

Information

Automatic coupons are store and vendor coupons given by the system when the specified criteria are met. They may be programmed with the help of the coupon wizard.

Here are some examples of how to create automatic coupons.

Examples

  1. Buy for a total of $10.00, get a 50% discount on a bag of Uncle Bens instant rice
  2. Buy 2 cans of corn cream, get the third one free
  3. Buy a bottle of Pepsi and a bag of Ruffles chips for $3.00
  4. Buy $10.00 worth of groceries, get $5.00 off if the customer has accumulated at least 5000 points in the main point program 

Buy for a total of $10.00

Get a 50% discount on a bag of Uncle Bens instant rice

1. In the main section of the coupon wizard, add the following coupon information:
  Value field: select Percent.
  Percent field: enter 50.
  Coupon type field: select Automatic. 
  Description field: add a meaningful description. 
  Sub-department field: select a sub-department. 
 
 
2. In the Criteria tab, click Add criteria to add the first coupon criteria.
  Type field: select Item sales. 
  Equal to field: enter the Uncle Ben's item UPC. 
  Minimum measure field: select Quantity. 
  Minimum value field: enter 1. 
  Multiplication and accumulation field: Select Exclude from accumulation.
This method excludes the bag of rice from the required $10.00.
  Select Make sure the discount affects the item(s) in these criteria.
This method forces the system to apply the automatic coupon discount on the bag of rice.
  Select The item(s) in this condition can only participate in one offer.
This method limits the participation of this item towards other coupons.
  Add a Manual description (optional).
  Click OK to confirm the changes and to close the Add criteria window.
 
 
3. In the Criteria tab, click Add criteria to add the second coupon criteria.
  Type field: select Total sales. 
  Totalizer field: leave the default value to 3. 
  Minimum measure field: select Dollar. 
  Minimum value field: enter 10. 
  Select The item(s) in this condition can only participate in one offer.
  Add a Manual description (optional).
  Click OK to confirm the changes and to close the Add criteria window.
 
 
4. In the Details tab:
  Funding field: select Store or Vendor.
  Event type field: select Sale.
  Enter a Start date.
  Enter an End date.
 
 
5. Add any necessary description in the Descriptions tab.
 
6. Click Save in the toolbar to finalize the coupon creation.
  The system will automatically assign the record a coupon code. 
 
  The Criteria tab will display the following information (Text view and Data view).
      

In this example:

  • The customer gets 50% off the bag of Uncles Bens instant rice when they buy 1 bag of rice and an additional total of $10.00 in the transaction.

Buy 2 cans of corn cream

2. In the Criteria tab, click Add criteria to add the coupon criteria.

  Minimum measure field: select Quantity.

  Select Make sure the discount affects the item(s) in these criteria.
This method forces the system to apply the automatic coupon discount on the cans of cream corn.
  Select The item(s) in this condition can only participate in one offer.
This method limits the participation of this item towards other coupons.
  Add a Manual description (optional).
  Click OK to confirm the changes and to close the Add criteria window.
 
 
3. In the Details tab:
  Funding field: select Store or Vendor.
  Event type field: select Sale.
  Enter a Start date.
  Enter an End date.
 
 
4. Add any necessary description in the Descriptions tab.
 
5. Click Save in the toolbar to finalize the coupon creation.
  The system will automatically assign the record a coupon code. 
 
  The Criteria tab will display the following information (Text view and Data view).
      

In this example: 

  • The customer buys 2 cans of corn cream in a transaction and gets the third one for free.

Buy a bottle of Pepsi and a bag of Ruffles chips for $3.00

In this example, all flavors of Ruffles chips must be included in the same promotion; the same promotion identifier must be entered in the Promotion ID field of the POS table for each flavor of Ruffles.

1. In the main section of the coupon wizard, add the following coupon information:
  Value field: select Bundle price.
  Price field: enter 1.
  Coupon type field: select Automatic. 
  Description field: add a meaningful description. 
  Sub-department field: select a sub-department. 
 
 
2. In the Criteria tab, click Add criteria to add the first coupon criteria.
  Type field: select Promo ID sales. 
  Equal to field: enter the promotion identification for the Ruffles. 
  Minimum measure field: select Quantity. 
  Minimum value field: enter 1. 
  Multiplication and accumulation field: select Accumulate for bundle or percent (A).
This specifies that the Ruffles chips are part of the bundle (A).  
  Add a Manual description (optional).
  Click OK to confirm the changes and to close the Add criteria window.
 
 
2. In the Criteria tab, click Add criteria to add the second coupon criteria.
  Type field: select Item sales. 
  Equal to field: enter the Pepsi UPC. 
  Minimum measure field: select Quantity. 
  Minimum value field: enter 1. 
  Multiplication and accumulation field: select Accumulate for bundle or percent (A).
This specifies that the Pepsi items are part of the bundle (A).  
  Select Make sure the discount affects the item(s) in these criteria.
This method forces the system to apply the manual coupon discount on the Pepsi items (M). 
  Add a Manual description (optional).
  Click OK to confirm the changes and to close the Add criteria window.
 
 
3. In the Details tab:
  Funding field: select Store or Vendor.
  Event type field: select Sale.
  Enter a Start date.
  Enter an End date.
 
 
4. Add any necessary description in the Descriptions tab.
 
5. Click Save in the toolbar to finalize the coupon creation.
  The system will automatically assign the record a coupon code. 
 
  The Criteria tab will display the following information (Text view and Data view).
     

In this example:

  •  The customer pays $3.00 when buying a bottle of Pepsi and any bag of Ruffles chips that are part of the Promotion ID.
  • The automatic coupon discount, which is the difference between the total value of the 2 items and $3.00, is applied on the bottle of Pepsi.

Buy $10.00 worth of groceries

Get $5.00 off if the customer has accumulated at least 5000 points in the main point program

1. In the main section of the coupon wizard, add the following coupon information:
  Value field: select Cents off.
  Price field: enter 5.
  Coupon type field: select Automatic. 
  Description field: add a meaningful description. 
  Sub-department field: select a sub-department. 
 
 
2. In the Criteria tab, click Add criteria to add the first coupon criteria.
  Type field: select Minimum point balance. 
  Program field: enter 1.
The main point program is identified by the character 1.
  Minimum balance field: select 5000. 
  Add a Manual description (optional).
  Click OK to confirm the changes and to close the Add criteria window.
 
 
2. In the Criteria tab, click Add criteria to add the second coupon criteria.
  Type field: select Total sales. 
  Totalizer field: enter 3. 
  Minimum measure field: select Dollar. 
  Minimum value field: enter 10. 
  Add a Manual description (optional).
  Click OK to confirm the changes and to close the Add criteria window.
 
 
3. In the Details tab:
  Funding field: select Store or Vendor.
  Event type field: select Sale.
  Enter a Start date.
  Enter an End date.
 
 
4. Add any necessary description in the Descriptions tab.
 
5. Click Save in the toolbar to finalize the coupon creation.
  The system will automatically assign the record a coupon code. 
 
  The Criteria tab will display the following information (Text view and Data view).
     

In this example:

  • The customer gets $5.00 off when they buy at least $10.00 worth of groceries in the transaction.
  • The customer must have accumulated at least 5000 points.
  • The customer ID must be entered in the transaction for them to get the points balance.
