Create Receiving

To create a new receiving document:

1. In the launcher, click the arrow in the Buy entry to display its menu.
2. Click on Receive: the Start window will appear.

    

3. Click on New: the New document window opens on the desktop.

 

4. Select a vendor from the Vendor field.
5. Click on Create to confirm the creation of a new receiving document for the selected vendor.

 When a receiving document is created:

  • It is listed in the Open document grid.
  • Its status is OPEN.
  • The (Full document) option in the start menu is enabled.
    This gives access to the Document window with information on the document and tools to edit it.
  • It allows adding, removing or editing the items.
