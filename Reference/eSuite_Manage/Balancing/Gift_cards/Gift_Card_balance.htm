eSuite Manage - Gift Card Balance

eSuite Manage > Balancing > Gift card

Information

  • Open session in LOC SMS eSuite Manage in order to have access to the gift card operations.

Transfer gift card value

  • Open the Balancing menu and click on Gift
  • from the Gift card menu, select Balance and click on Execute. You can use the Add Criteria button to refine the search
  • Select a gift card from the list. In order to do a value transfer, the gift card must be assigned to a customer, if no customer is assigned to the gift card a warning message will pop up. Refer to section Associate a gift card to a customer to know how to do this process
  • click on button Action at the bottom of the window
  • select Balance transfer, the Prepaid balance transfer window will open
  • on the left side of the Prepaid balance transfer window, on field Number, select the gift card from the list, the gift card info will be displayed
  • on the right side select the Destination gift card
  • enter the amount to be transferred and click on Submit

Associate a gift card to a customer

A gift card number can be associated to an existing customer. Be aware that eSuite will not validate if the associated gift card exists on its database.

  1. Open session in LOC SMS eSuite, Management Suite in order to assign a gift card to a customer.
  2. Open the Customers menu and click on Clients.
  3. Click on General List;
  4. search for the client to whom the gift card will be assigned using one of the available options and click on Execute;
  5. The window Customers - General List will pop up.
  6. Select the customer using the checkbox at the left of the Customer ID.
  7. Click on Edit at the bottom of the window and select Alternate codes.
  8. Add a gift card:
    • Click on
    • on field Alternate number write down the gift card number
    • on Type, select Gift Card from the drop-down menu or write down GIFT
    • select one status from the drop-down menu
    • optionally, add a Description to the gift card
    • select the Recharge Type from the drop-down list
    • select the recharge Frequency
    • set the field Value
    • set the field Minimum
    • Save the changes using
