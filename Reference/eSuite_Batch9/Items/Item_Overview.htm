eSuite - Items Overview

eSuite  > Items

Information

The eSuite software keeps the item information in different tables, called item tables. They are essential for creating items, generating prices and tracking cost information. Item tables can be accessed from the Items menu in the launcher.

The Items entry consists of three sub-entries:

  • Master, providing price, cost, POS and main item information and allowing the creation of items.
  • Batches, providing batch information and allowing the creation of batches.
  • Item auxiliaries, providing additional item information, such as price levels, item classification, promotions.

Master

Provides access to the list of changes in the item master and future tables.

Provides access to the list of all items in violation of validity checks.

Provides access to the full history of changes made to records.

Provides access o the Inventory history report.

Provides access to the list of single total reports.

Provides access to the list of multi totals reports.

Batches

The Batches sub-entry opens the Batches start window with a list of options offering batch information and tools for batch maintenance.

The following options are available from the Batches start window:

OPTION DESCRIPTION
Batches Opens the List batch wizard.
Provides access to the list of batches.
All Opens the list of batches, without going through the wizard.
New Displays the list of batch types. 
Provides access to the batch header table for the creation of new batches.
Items Opens the Items - general list wizard.
Provides access to the list of items with the main item information.
Prices Opens the Price batch wizard.
Provides access to the list of the items in the selected price batch.
Suggestion Opens the Price generate batch wizard.
Provides access to the list of items in the selected price batch.
Costs Opens the Cost batch wizard.
Provides access to the list of the items in the selected cost batch.
POS information Opens the POS info batch wizard.
Provides access to the list of items in the selected POS batch.
Main item Opens the Main item batch wizard.
Provides access to the the list of items in the selected main item batch.
Delete batch Opens the Delete batch wizard.
Provides access to the list of items in the selected delete batch.
Deploy batch Opens the wizard prompting the selection of the batch to deploy. 
General changes Opens the General changes wizard.
Provides access to the list of changes made to batches.

For details, see Batches - overview.

Item auxiliaries

The Item auxiliaries sub-entry opens a list of options offering information related to item sales and tools for auxiliary data maintenance.

The following options are available from the Item auxiliaries start window:

OPTION DESCRIPTION
Price levels Opens the list of price levels.
Provides access to the Price level table.
Like codes Opens the list of like codes.
Provides access to the Like code table.
Mix & matches Opens the list of mix and matches.
Provides access to the Mix & match table.
Bottle and environment fees Opens the list of bottle fees.
Provides access to the Bottle link table.
UCC coupon value table Opens the list of coupon values.
Provides access to the Coupon value table.
Restrictors Opens the list of restrictions.
Provides access to the Restriction table.
Recipe editing Opens the list of recipes.
Provides access to the Recipe information table.
Shelf location Opens the list of shelf locations.
Provides access to the Shelf location table.
Promotion IDs Opens the list of promotions.
Provides access to the list of items in the selected promotion.
Scale tables Opens the list of scale tables.
Provides access to the following tables: Scale ingredients, Scale nutrifacts, Scale storage instruction, Scale COOL text, Scale COOL short, Scale COOL production, Scale COOL country.
Tare table Opens the list of tares.
Provides access to the Tare table.
Classification Opens the list of classifications.
Provides access to the Classification table.
NACS Opens the list of NACS codes.
Provides access to the NACS table.

Context tools

The Context tools window is always active on the right side of the desktop, in all eSuite modules. It adjusts its layout according to the entry selected in the launcher and displays information relevant to the element that is active on the desktop.
