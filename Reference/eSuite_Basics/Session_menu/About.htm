Table of contents

Information

The About functionality in eSuite allows you to get detailed information on the version and the licence.

It is available from the Session menu in the eSuite taskbar.

Layout

The About button opens the About window.

Content

The following elements are available in the About window:

ELEMENT DESCRIPTION
About Title
Shows the window's name
LOC Store Management Suite Logo
Shows the developer's logo.
Version Text field
Shows information on the current version.
Licensed for Text field
Shows information on the end user.
View end user license agreement Link
Opens the End user license agreement (EULA) page.
Copyright Text field
Shows information on the copyrights.
Show options Button
Opens the Installed options/samples window.
Close Button
Closes the window.

The Installed options/samples window

The following elements are available:

ELEMENT DESCRIPTION
Installed options/samples Shows the window's name.
Type Indicates the content type: option or sample.
Allows you to sort the column contents in ascending and in descending order.
Allows you to show/hide columns.
Hidden by default.
Name Indicates the name of the option or that of the sample.
Allows you to sort the column contents in ascending and in descending order.
Allows you to show/hide the Type column.
Last update Indicates the time of the last update.
Allows you to sort the column contents in ascending and in descending order.
Allows you to hide/show columns.
Type: Option Shows the name of the Option panel.
Option panel Displays the installed options.
Type: Sample Shows the name of the Sample panel.
Sample panel Displays the name of the installed samples.
Collapse Hides the Option panel.
Hides the Sample panel.
Expand Shows the Option panel.
Shows the Sample row panel.
Resize handles
Resize the page horizontally, vertically and diagonally.
They are visible in each corner and on each side of the window when you hover the mouse over the borders.
OK Closes the Installed options/samples window.
