eSuite Care - Overview

eSuite Care

Information

eSuite is a module-based web interface for viewing and managing information in SMS and will eventually replace the  SMSPRO application.

The documents in this section cover the maintenance and reporting features of the Care module.

The following is a brief list of some of the Care module features:

  • Reporting and maintenance of Items within the report grid itself.
  • Reporting and maintenance on Dimension tables (formerly called Auxiliary).
  • Flexible, customizable target reporting.
  • Creating offers and assigning them to customers.
  • Reporting on Loyalty, including points balance, customer history and performing points adjustments if required.
  • A new powerful sub-reporting feature known as sub-queries, which provides the ability to run reports based on the results from previous reports.
