eSuite - How to Generate Advertising Slide From a Template

eSuite > Items > Item auxiliaries

Information

To be revised and updated once the feature is available in eSuite.
 SMS image for reference only; please replace it with its equivalent in eSuite. (Raluca, 2019-09-03)

The Advertising table allows creating dynamic advertising slides from a template, usually from items that are on sale. To create the advertising slides, the Option_pub option must be installed.

The eSuite software provides one template named pub_tpl_item_promo.htm, which can be extracted from the Grocery lite sample. You can create your own templates using the HTML file designer (Edit HTML templates option from the Execute pull-down menu). The template file names must start by pub_tpl_ and must be located in the Office/Cgi directory of eSuite.

If an image of the item is displayed in the advertising slide, the image file must be stored in the Bitmaps directory, in the pub_[upc code].bmp format.

Steps:

  1. From the Launcher, select Items, then Master: the Start window opens on the desktop.
  2. Click on Item list.
  3. In the wizard, enter the search criteria and click Execute: the item list displays the results on the desktop.
  4. Select an item.
  5. Open the Advertising table.
  6. Click Add in the toolbar to add a new advertising slide.
  7. In the Ordering name field, enter the description (or name) of the slide.
  8. In the Generate from template section of the Advertising table:
    1. Double-click in the Entry box and select the desired slide template.
    2. In the Source price sub-section, select the price to use in the advertising slide.
    3. Click on Generate. This creates the file name of the advertising slide in the URL field, in the template name_UPC code.htm format.

    4. Click on Edit result. This edits the new slide in the HTML file designer.
