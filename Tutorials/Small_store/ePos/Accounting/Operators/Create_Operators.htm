Create Operators

This is how to create Operators:

1.  From the MANAGE module

2.  Click Operators

3.  From the Start widget click Operators - All

4.  Click the add button   on the top-right toolbar

5.  Enter the Operator Information.

 Fields in RED are required

6.  Press SAVE in the Operator Edit toolbar


Hint: Refresh the grid  to see the new operator in the list.
