Information

The Cancel function allows you to cancel transactions, by default the function cannot be performed without the manager login.

There are two types of cancels that can be called : Function 630 Cancel order and Function 670 Cancel previous order.

Cancel order

The Cancel order function is the function  used to delete the current transaction. The function cannot be called if the transaction has not been started. The Cancel order function keeps all entries of the transaction and adds a new entry (Cancel order) that specifies the deletion of the transaction.

To delete the current transaction using the Cancel order function, follow these steps:

  1. Open an epos session and login.
  2. Start a transaction by selling items.
  3. Login as manager.
  4. Call the Cancel order/transaction button.
  5. A prompt in the detail window will appear :
    Cancel order 1=Yes/0=No ?
  6. Select 1 to go forward with the cancel order or 0 to cancel the operation, then press the Enter key or click on the E button to validate your entry..
  7. If you validated the entry 1, the current transaction gets cancelled instantly.
  8. The detail window of the pos will display "End of transaction " "Cancel order $00.00".
  9. The transaction has been cancelled and you can start a new one.

Cancel previous order

A completed transaction cannot be modified, but it can be reviewed and, if needed, cancelled. The Cancel previous order function creates a new transaction that "reverses" the one you cancelled (a refund is created to cancel a sale, a sale is created to cancel a refund).

To cancel a previous transaction using the cancel previous order function, follow these steps:

  1. Open an epos session and login.
  2. Login as manager.
  3. Select the Cancel previous order button.
  4. A prompt in the detail window will appear "Cancel previous order 1=Yes/0=No?"
  5. In the Entry window, enter 1 to proceed with the operation or 0 to cancel.
  6. Press the Enter key or click on the E button to validate your entry.
  7. A refund transaction is processed in the background to cancel the sale and it is
    automatically completed. You will see the message "Change $00.00" to indicate that the process has been completed.


NOTE: Since a completed transaction cannot be modified or removed, SMS creates a new transaction that "reverses" the one you cancelled (a refund to cancel a sale, a sale to cancel a refund). The new transaction contains the same items as the one you cancelled, and is automatically completed, so it cannot be modified.  A receipt is printed.
