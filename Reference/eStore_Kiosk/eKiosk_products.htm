eKiosk - Products

eKiosk

Information

The Products page of the eKiosk allows the customer to view product's information without having to do any purchase. It acts like a price checker.

Products Features

This section of the eKiosk contains certain features to help the customer to find his item quickly. Here is a list of what this page supports:

  • Search: The customer can either type in the barcode of the item or type any text and SMS will find the items that has this code or text in it.
  • Preferences: Allows the customer to select specific preferences like Gluten Free or Vegan items only.
  • Search by Category: Allows the customer to search their items via categories like Bakery, Deli, Fish, etc.
  • Search by Brand: Allows the customer to view the product for a specific brand only.
  • See Featured Items only: This filter allows the customer to view only the preferred items (same as on the Start page).
  • See only items with points: This allows the customer to view only the items that gives points.
  • See only promotions: This allows the customer to see the items in promotion in different categories such as This Week's Specials, Long Term Specials and Store/Manager's Specials.

The Start Over button at the top is to refresh the search and go back to the original page.
