eSuite - Labels by Batch

eSuite > Tools > Labels > By batch

Information

The Label by batch utility allows extracting labels on the items that are part of the following types of batches: instore price, future price, sale and TPR.

Displaying the Labels by batch utility

In the launcher, click the arrow in the Tools entry to display its menu and then:

  1. Click on Labels: the Start window opens in the Labels work area.
  2. Click on By batch to display the options.
  3. Select any of the four options:
    • Instore price batch
    • Future price batch
    • Sale batch
    • TPR batch

The Labels by batch wizard opens on the desktop.

*Click on a field to get more information.

Layout of the Labels by batch wizard

Here is the description of the fields and functionalities in the Labels by batch wizard:

FIELD DESCRIPTION
Batch start The Batch start field gives access to the list of batches.
It works with the Batch stop field to set the starting batch number and define the batch range.
Batch stop The Batch start field gives access to the list of batches.
It works with the Batch start field to set the ending batch number and define the batch range.
Label format Label format is a mandatory field that contains the following predefined label formats:
  • CUSTOMER_CARD.lbz
  • OPERATOR_CARD.lbz
  • REG_14UP_05162.lbz
  • REG_16UP_SHELF.lbz
  • REG_16_320_120.lbz
  • REG_16_325_120.lbz
  • REG_XMAS_2x2.lbz
  • SAL_10UP_5963.lbz
  • SAL_1_10_75.lbz
  • SAL_24UP_COMPARE.lbz
  • SAL_28UP_COMPARE.lbz
When the Label format field is populated, the Execute button becomes enabled.
Quantity for each label The Quantity for each label field specifies the number of labels to print for each extracted label.
The default value is 1.
Price level start The Price level start field specifies the starting price level (first data record).
The default value is 0: All.
It works with the Price level stop field to set the data range for the label template.
Price level stop The Price level stop field specifis the ending price level (last data record).
The default data is 99999: All.
It works with the Price level start field to set the data range for the label template.
Item selection The Item selection field specifies if the items that are flagged Not for sale are included in the selection.
If the item changes are received from the host, it is possible that you are not selling some of these items. In this case, these items are flagged Not for sale and you may choose not to print labels on them.
The default value is All items except the ones having "NOT FOR SALE flag on.
Price target The Price target field contains the target identifier.
Breakdown target by store The Breakdown target by store field will display the targets by store.
Use quantity set in item location The Use quantity set in item location field will look at the Quantity field in the Shelf location table to determine the number of labels to print.
Only items having label format in location The Only items having label formation in location field allows printing labels for the items that have a label template set in the Shelf location table.
Reset The Reset field resets any prior values defined in the window.
Execute The Execute field allows generating the label file for the selected items.
Disabled by default.
Enabled when the Label format field is populated.

Print label list

When the Labels by batch utility is launched, it opens the Print label list window.

*Click on a field to get more information.

Here is the description of the columns and the information they may contain:

COLUMN DESCRIPTION
Checkboxes The checkboxes allow selecting and deselecting labels.
User ID The identifier of the user who has created the list of labels by batch.
Label format The label format for printing.
Quantity The number of labels to be printed.
Target The target identifier.
Created date The date the list of labels was created.

In the bottom menu of the grid:

  • By store allows breaking down the list results by store.
  • Action allows to:
    • View the items to be printed with the selected label.
      This opens the Show print items grid.
    • Delete the selected label.
    • Print the label.
