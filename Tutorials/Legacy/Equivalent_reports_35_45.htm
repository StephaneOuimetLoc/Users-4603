1. Cost of Goods by department

In SMS Pro, the only report by department is currently the Cost of Goods by Item, which allows the user to enter a department range.

 

In eSuite, use the Items - Cost of Goods window to add criteria and group it by department and generate the equivalent report.

  1. Select the Items entry in the launcher, then click on Master.
  2. In the Start window, select Cost of goods.
  3. Click on Add criteria to open the Select criteria window.
  4. In the Classification ranges section, select Department number start and Department number stop.
  5. Click OK.
  6. Enter the department range.
  7. Click Execute.
     

The report is grouped by department.

When the expand button is active, the report shows the item detail for each department.
When the collapse button is active, the report shows only the sub-totals.

2. Cost of Goods by Items

SMS Pro - Reports - Others - Items - Cost of Goods Detailed by Items

In eSuite, use the default Items - Cost of goods window to generate the equivalent report.

  1. Select the Items entry in the launcher, then click on Master.
  2. In the Start window, select Cost of goods.
  3. Uncheck the Breakdown by sub-department box.
  4. Click Execute.
