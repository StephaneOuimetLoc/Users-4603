Pool parameters

These macros have been developed to work with the parameter pool.

@WizReset

This macro resets all the parameter in the pool.

Return value
None
Example
@WizReset

@WizSet(<Name>=<Value>)

This macro creates and sets the value of a pool parameter if it does not exist.

Parameter Description Required/Optionnal
Name Pool parameter name Required
Value Value Required
Return value
None
Example
@WizSet(V01=Hello)

@WizRpl(<Name>=<Value>)

This macro replaces the value of a pool parameter.

Parameter Description Required/Optionnal
Name Pool parameter name Required
Value Value Required
Return value
None
Example
@WizRpl(V01=Bye)

@wizAppend(<Name>=<Value>)

This macro appends the value <Value> to the current value of a pool parameter.

Parameter Description Required/Optionnal
Name Pool parameter name Required
Value The append value Required
Return value
None
Example
@wizSet(V01=Loc)
@wizAppend(V01= Software)
@wizGet(V01)   will return "Loc Software"

@WizClr(<Name>)

This macro clears the value of a pool parameter.

Parameter Description Required/Optionnal
Name Pool parameter name Required
Return value
None
Example
@WizClr(V01)

@WizGet(<Name>)

This macro returns the value of a pool parameter.

Parameter Description Required/Optionnal
Name Pool parameter name Required
Return value
Value of the pool parameter
Example
@WizGet(V01)

@WizIsBlank(<Name>)

This macro checks whether a pool parameter is missing or this value is empty.

Please note:  Space character is not an empty value.

Parameter Description Required/Optionnal
Name Pool parameter name Required
Return value
1 if the pool parameter is missing or this value is empty otherwise 0
Example
@WizIsBlank(V01)

@WizExist(<Name>)

This macro checks if a pool parameter exist in the pool.

Parameter Description Required/Optionnal
Name Pool parameter name Required
Return value
1 if the parameter exist otherwise 0
Example
@WizExist(V01)

@WizExistAll(<NameMask>)

This macro checks if a pool parameter name beginning with <NameMask> exist in the pool.

Parameter Description Required/Optionnal
NameMask The first characters of a parameter name. Required
Return value
1 if at least one parameter exist otherwise 0
Example
@WizExistAll(V)

@TOOLS(PARAMTOLINE)

This macro returns a string containing all the pool parameters with their value.

Return value
Format : /<Name>=<Value> /<Name>=<Value> /<Name>=<Value> â€¦
Where <Name> is a pool parameter name and <Value> is the value assigned to the parameter.
Example
@WizRpl(PARAM1=Value1);
@WizRpl(PARAM2=Value2);
@WizRpl(PARAM3=Value3);
@TOOLS(PARAMTOLINE)     will return  "/PARAM1=Value1 /PARAM2=Value2 /PARAM3=Value3"
