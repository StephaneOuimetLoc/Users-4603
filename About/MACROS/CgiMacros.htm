CGI Macros

Some macros have been developed specifically to process CGI files.

@dbHot(TABLE_RECORD)

This macro returns the current row number of the executed dbTable.
Works exclusively with <dbTable> or <PRE> XML tags.

Return value
Current row number
Example
@dbTable(Q1,Select top 2 F1039 from TLZ_TAB);
<dbTable>
   <record>
      <RowNumber>@dbHot(TABLE_RECORD)</RowNumber>
      @dbRow(Q1,Nodes)
   </record>
</dbTable>

will be converted to:

<record>
   <RowNumber>1</RowNumber>
   <F1039>TOTAL SALES</F1039>
</record>
<record>
   <RowNumber>2</RowNumber>
   <F1039>Net Sales</F1039>
</record>
