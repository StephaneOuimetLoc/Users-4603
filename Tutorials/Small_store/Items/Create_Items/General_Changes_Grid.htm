General Change Grid

 eSuite > Items > General Changes                         

Information

This document describes the General Changes feature is located under the Items section of eSuite Manage/Batch/Care/Buy or Gas.


NOTE: The feature was formerly called Batch All Changes in SMS Pro (prior to eSuite release).


Concept

The General Changes is a report showing all the changes contained in un-executed batches sitting in the instore batches and data in the Item tables(POS,PRICE,COST,etc).  It will display the current value for any items and the new value coming from the batch and the batch number.


1.  To access the GC report click on the Items section in the laucher

 

 2.  Select General Changes

       

    You will be prompted with the General change log Wizard

    Enter your search criteria.  Check or uncheck as many or as little item tables you wish to in the report.

   

3.  Click Execute to run the report and you will be presented with the report:

   


The table below explains the GC report grid and its functionalities.


NO. NAME DESCRIPTION
Date This is the date on which the new value took effect or will take effect if the change is in a future batch.
Item Code UPC number of the item in question.
User name Operator number who made the change
Source Where the change comes from.  Will be a Batch Number or 'REV' which means the change is being audited from the revision tables.  All item changes are tracked in the revision tables.
Field ID Field number and description that was modified.
Old Value Outgoing Data
New Value Incoming Data
 Descriptor  Provides additional information about the item in question
