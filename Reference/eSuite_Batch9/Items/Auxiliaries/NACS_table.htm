eSuite - NACS Table

eSuite > Items > Item auxialaries > NACS

Information

NACS (National Association of Convenience Stores) codes are used by gas stations to identify what is being charged to a fleet credit card.

The NACS table contains NACS codes that are linked to the items.

Displaying the NACS table

In the Launcher, click the chevron in the Items entry to display its menu and then:

  1. Click on Auxiliary tables: the Start window opens in the Auxiliary work area.
  2. Click on NACS: the NACS tab opens in the same window, displaying the NACS List.
  3. To modify an existing NACS record, select it in the list and double-click on it, press the F3 key on the keyboard or click the Edit command in the bottom menu: the NACS table opens, overlapping the work area.
  4. To create a NACS record, click on Add in the bottom menu of the grid: the NACS table opens, overlapping the work area.

Layout

* Click on a field to get inforation on it.

Content

Here is a description of the NACS table fields and their functionalities:

FIELD DESCRIPTION
Code The Code field contains the NACS code.
Descriptor The Descriptor field contains the description of the NACS code.
