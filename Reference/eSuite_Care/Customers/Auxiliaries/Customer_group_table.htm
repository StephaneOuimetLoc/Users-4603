eSuite - Customer Group Table

eSuite > Customers > Customer Groups

Information

The Customer group table is useful in creating relationships between customers that have similarities other than the same shopper level or risk level (for example, senior customers, customers grouped by company, customers grouped by payment type, etc.).

Customer groups

The Customer groups list displays the customer group numbers and description.

In the bottom menu:

  • Edit provides access to the Customer group table.

Below is a brief description of the information displayed by column in the default view of the Customer groups list.

COLUMN DESCRIPTION
Number The number corresponding to the customer group.
Description The description of the customer group.

Edit Customer group table

In the Launcher, click the arrow in the Customers entry to display its menu and then:

  1. Click on Client auxiliaries: the Start window opens in the Auxiliary work area.
  2. Click on Customer groups: the Groups tab opens in the same window, displaying the Customer groups - List.
  3. Select a record.
  4. Click Edit in the bottom menu: the Customer group table opens on the desktop.

Layout

* Click on a field to get information on it.

Content

Here is the description of the Customer group table fields and their functionalities.

FIELDS DESCRIPTION
Number The Number field contains the customer group identifier.
It is a required field used to uniquely identify records in the Customer Group List.
Descriptor The Descriptor field contains the description of the customer group.
