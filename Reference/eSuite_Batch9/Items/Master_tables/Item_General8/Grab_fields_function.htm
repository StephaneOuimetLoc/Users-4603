eSuite - Grab Fields Functionality

eSuite

Information

The Grab Fields functionality is available only in Customers and Items, within certain wizards.

The enhancement to the Customer and Items wizards are so that the end-user are no longer limited to what is listed in More Criteria. It looks similar to what was inside the SMS Pro application, as most users enjoyed that quick ease of use in being able to add unique criteria to their searches.

The Grab field button works in combination with the Add selected button (the Add selected button is enabled when the the Grab field button is used).

How to use the Grab Fields functionality

The most efficient way to use the feature is to open the needed Edit window for which additional fields would be required into the wizard query. This example is based on Items - General list for which fields from the POS master and Price master tables are added.

  1. Run the default query by clicking Execute.

  2. From the result grid, select Edit > Master tables > POS master.

  3. In the POS Information Edit table, pin the table.

  4. Repeat the same steps to display and pin the Price master table.
  5. Go back to the Wizard of the Items - General list, and select Grab fields.

  6. In the POS Information Edit and Price Edit windows, click inside the fields for which you would like to do a search.

    The selected fields have a light blue background.

    NOTE: To unselect a field, click again on this field; its light blue background color is removed.

  7. Once the selection of the fields is completed, go back to the Wizard and click Add selected.

    The selected fields are displayed in the Wizard.

  8. You can save this new layout by clicking if you intend to use it on a regular basis.

  9. Set the conditions of the selected fields, and then click Execute to get the expected results.

    NOTE: The fields selected using the Grab fields functionality can be used with the More Criteria option as well as any information entered in the top fields of the Wizard to get the expected results.
