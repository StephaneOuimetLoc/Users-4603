eSuite - Storage Instruction Table

eSuite > Items > Item auxialaries > Scale Tables

Information

The Storage instruction table is used for storing recommendations. Storage instructions are mandatory for foods that require a best before date and when the storage conditions differ from the normal room temperature. Examples include phrases such as "Keep refrigerated" and "Store in a cool dry place".

Displaying the Storage instruction table

In the launcher, click the arrow in the Items entry to display its menu and then:

  1. Click on Item auxiliaries: the Start window opens in the Auxiliaries work area.
  2. Click on Scale tables.
  3. In the list of scale tables, select Scale storage instruction: the Storage instruction tab opens, displaying the Storage instruction list.
  4. To modify an existing record, select it in the list and click Edit in the bottom menu: the Storage instruction table opens over the work area.
  5. To create a new storage instruction record, click on Add in the grid toolbar: the Storage instruction table opens over the work area.

Content

Here is a description of the Storage instruction table fields and their functionalities:

FIELD DESCRIPTION
Target The Target field contains the store group identifier.
Number The Number field contains the number given to the storage instruction template.
The number assigned does not need to match the item code.
It is possible to have one storage instruction template for multiple items.
Font size The Font size field contains the size of the font in which the details will be printed on the label.
Instruction The Instruction field contains details on the proper method for storing the item.

Setting up storage instructions

To create a storage instruction:

  1. In the Storage Instruction table, click Add in the toolbar.
  2. Assign a number for the storage instruction template in the Number field.
  3. Enter the instructions in the Instruction field.
  4. Enter a font size.
  5. Save the changes.

Back to layout

Linking storage instructions to items

To link a storage instruction to an item:

  1. In the item list, select the item you want to link storage intructions to.
  2. Click on Edit in the bottom menu.
  3. Click on Special items, then on Scale information: the Scale information table opens in the work area.
  4. In the Storage field, add the storage instruction number.
  5. Save the changes.
  6. Instructions will be sent to the scale upon deployment.
