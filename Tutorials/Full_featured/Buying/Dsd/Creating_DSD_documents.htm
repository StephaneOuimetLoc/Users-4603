eSuite Buy - How to Manage DSD Documents

eSuite Buy > Buy > DSD

Information

The eSuite Buy module offers the tools for creating and editing purchasing and billing documents for Direct Store Deliveries (DSD).

This document shows how to:

  • Create and Edit a DSD document
  • Adding items     
  • Remove items
  • Manage quantities             
  • Canceling a DSD document
  • Suspend or close the DSD document
  • Managing the Bill

Direct store delivery documents are created with the New option in the start menu.

Creating a DSD

  1. In the launcher, click the arrow in the Buy entry to display its menu.
  2. Click on DSD: the Start window opens on the desktop.

     
  3. Click on New: the New document window opens on the desktop.

  4. Select a vendor from the Vendor field.
  5. Click on Create to confirm the creation of a new DSD document for the selected vendor.
  6. Confirm the payment type     

    The final bill can be paid according to four (4) different methods of payment:

    • Cash payment       
    • Check payement       
    • Drop payment       
    • Charge to account       

          

  7. Enter the total amount to be paid, then save and close the window

When a DSD document is created:

  • It is listed in the Open document grid.
  • Its status is OPEN.
  • The Document (full) option in the start menu is enabled.
    This gives access to the Document window with information on the document and tools to edit it.

Adding items to the DSD document

The second step in the creation of a DSD document is adding items.

  1. Select Items in the Start window to open the item search wizard.
    The Vendors from current documents checkbox is selected by default and disabled: only items from the current vendor will be displayed in the resulting grid.

         
  2. Add all necessary criteria and click Execute: the list of items opens on the desktop.
    The Amount, Case, Unit, Weight and Cost fields in the Document column are editable, allowing to set new values for each item.

  3. Double-click in any of these fields and add ordered quantities.
    Press Enter after each new value.

  4. Click on Save in the toolbar: the new quantities are displayed in the Document column.
    The system calculates the amounts and costs according to the new values.


Editing the DSD document

Before closing the delivery document, you may want to revise and edit the information in it. This is done through the Document window, which offers the tools for editing general and specific information, managing items, suspending the document or closing it.

To display the Document window:

  1. From the start menu, click on Documents - All: the list of delivery documents opens in the Open document grid.
  2. Select the document you want to edit information for.
  3. Right-click to display the list of options.

      
  4. Select "Open"
  5. From the Start window, select Full document the Document window opens on the desktop, displaying the information in the delivery document.
       

The Document window of an open DSD document is also available from the Document (full) option in the start menu.

Editing general information

General information related to vendor and shipping may be edited in the Transaction header table.

  1. From the Document window, click on Edit header in the upper part of the document.
    The Transaction header table opens on the desktop.

      
  2. Make all necessary changes.
  3. Click on Apply changes in the toolbar.

Editing specific information

Information related to items may be edited in the Transaction line order table.

  1. From the Document window, select an item in the Lines section.
  2. Click on Edit selected line in the upper part of the section.
    The Transaction line order table opens on the desktop.

        
  3. Make all the necessary changes.
  4. Click on Apply changes in the toolbar.

Editing lines

Quantities (cases, units, weight) and amounts of the delivery document may be modified directly in the Lines section.

  1. Double-click in the case, unit or weight column of the item.
  2. Enter or select the new value.
  3. If necessary, modify the amount.

      

Voiding items

Items may be removed from the delivery document.

  1. In the Line section of the Document window, select the item to remove.
  2. Right-click to display the menu.
  3. Click on Void selected line: a confirmation message is prompted.
  4. Click on Yes to delete the item from the document.

Canceling a DSD document

Open delivery documents

  1. In the Context Tools, click on the hamburger icon  and select Cancel the document

  2. Click on Yes to confirm
  3. Retrieve the Open documents from the wizard, the canceled document should removed from the list.

Suspending a DSD document

Open delivery documents may be suspended and edited at a later time.

  1. Open the Document window for the delivery document.
  2. From the Context Tools, click on the hamburger icon
  3. Click on Suspend the document in the toolbar.
    The Document window closes.

To open a suspended document:

  1. From the Open documents grid, right-click on the suspended delivery document.
  2. Select Open from the menu.
    The delivery document is available from the Document (full) option in the start menu.

Managing the invoice

Adding taxes and fees

The Add to bill feature allows adding different taxes and fees to the delivery document.

  1. From the Context Tools  toolbar, click on the hamburger icon  and select Complete functions

       
  2. Make all necessary changes.
  3. Click on Save all changes in the toolbar.

Substracting from the DSD

The Subtract from bill feature allows deducting different amounts from the delivery document.

  1. From the Context Tools  toolbar, click on the hamburger icon  and select Complete functions

       
  2. Make all necessary changes.
  3. Click on  Save all changes in the toolbar.

Paying the invoice

The final bill can be paid according to different methods of payment:

From the Context Tools  toolbar, click on the hamburger icon  and select Complete functions

Add the amount for the appropirate payment type under the Entry column and save.       

          

Closing the DSD document

To close the delivery document:

  1. From the list of documents, open the delivery document.
  2. From the Context Tools click on the hamburger icon  and select Close the document
  3. Click on Yes to confirm
