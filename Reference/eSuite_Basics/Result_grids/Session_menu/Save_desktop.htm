eSuite - Save desktop

eSuite > Taskbar > Session

Tablessss of contents

Information

The Save desktop functionality in eSuite allows you to save the layout of the active desktop widgets.

It is available from the Session button in the eSuite taskbar.

Layout

The Save desktop button opens the Layout setting window.

Content

The following elements are available in the Layout setting window:

ELEMENT DESCRIPTION
Layout setting Shows the window's name.
Layout Allows you to enter text or select text from the list.
Save Saves the changes in the Layout field.
Delete Deletes the value selected in the Layout field.
