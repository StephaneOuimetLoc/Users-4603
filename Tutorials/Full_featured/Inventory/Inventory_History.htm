eSuite - Inventory history

eSuite > Items

Information

The Inventory history  shows the current and previous week inventory values (quantity and dollar amounts) of the selected item(s) as well as the details of the movements for the selected period.

The Inventory history report is available from multiple modules including eSuite Buy:

  1. In the launcher, click on Items.
  2. Select Inventory history

Wizard

From the wizard, you can narrow down your search by specifying the Period, date(s), item(s), target and additional criteria from the "More criteria" option and finally breakdown the results by store and Execute

 

Using Item linked query

You can use the Item linked query in order to display the results for each item selected in the grid.

  1. In the launcher, click on Items.
  2. Select Items

 

3. In the Wizard, you can retrieve all items or a range of items by using the More criteria. In this example, we're using a range of items from 0 to 1000 and then Execute.

 

4. From the items grid, select Analyze and Item linked query

 

5. The start window will appear displaying items (LINKED) at the top which means the grid is now linked. Select Inventory history.

6. Fill in the information in the Wizard and Execute.

 

 

It will display the results for the item selected in the items grid. The inventory history results will be refreshed each time a different item is selected.

Inventory history result

The results will contain the Movement, Previous Inventory and Current Perpetual Inventory.

  • Movement will show the qty and amount broken down by the totalizer and description of each movement. In the example below, we see the sales, inventory count, purchases and adjustments that were done during the selected period.
  • Previous Inventory is based on the start date selected, the report will get the end of the previous week to display the previous inventory.  The same record will be repeated for each line.
  • Current Perpetual Inventory, will display the current inventory and will repeat for each line as well.
