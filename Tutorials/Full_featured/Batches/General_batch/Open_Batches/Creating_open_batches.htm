Create Open Batches

eSuite > Items > Batches

Information

The open batches are used to manage item changes on multiple items in various aspects. For example, an open batch can include a future cost, a future price and future sale information on multiple items.

New batches are created using the Batch header table that sets the batch type and its unique number, and contains the general batch information.

Creating an open batch

In the launcher, click the arrow in the Items entry to display its menu and then:

  1. Click on Batches: the Start window opens in the Batch work area.
  2. Click on New and select Open batch.
  3. Enter the required information in the Batch header table.
  4. Click on Save in the toolbar to open the selection window:
  5. Check the required boxes and click OK.

To open the list of open batches:

  1. Select the Batches entry from the Start window.
  2. Click in the Batch type field.
  3. Select Open batch from the drop-down menu.

    The In-store batches grid displays the open batches.

Adding items to the open batch

  1. In the Start window, select either of the following entries: Prices, Costs, POS information or Main item.
  2. Click on the Batch field to open the Batch window.
  3. Click the arrow in the Macro field and select Open batch.
  4. Click Filter: the Batch window displays the list of open batches.
  5. Select the newly created open batch and click OK on the bottom of the window.
  6. In the grid, select the batch type corresponding to the change.
  7. Enter all the other necessary search criteria.
  8. Click Execute: the list of items is diplayed in the batch grid.
  9. Make all the necessary changes to the items that will be part of the open batch.
  10. Click on Save in the toolbar to add the modified items to the open batch.

Deploying the batch

  1. In the Start window, select Deploy batch: the Wizard window opens for batch selection.
  2. Click in the Select a batch field.
  3. Select the open batch in the list.
  4. Click on Confirm: the Wizard window opens for target selection.
  5. Select a target and click on Confirm to deploy the batch.
