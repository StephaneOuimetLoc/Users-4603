eSuite - Manual Coupons Examples

eSuite > Items > Master > Coupon list

Information

Manual coupons are store and vendor coupons that need to be keyed in manually by the cashier, unlike electronic coupons which are issued by the system at the end of the transaction. They may be programmed with the help of the coupon wizard.

Here are some examples of how to create manual coupons.

List of examples

  1. Buy a small coffee, get $0.50 off
  2. Buy two bottles of Gatorade, get one bottle for free
  3. Buy a bag of Uncle Ben's instant rice, get 10% discount
  4. Buy $5.00 worth of grocery, get 100 points
  5. Buy a bottle of Pepsi and a bag of Ruffles chips for $3.00

Buy a small coffee

Get $0.50 off

1. In the main section of the coupon wizard, add the following coupon information:
  Value field: select Cents off.
  Discount field: enter 0.5.
  Coupon type field: select Manual. 
  Description field: add a meaningful description. 
  Sub-department field: select a sub-department. 
 
 
2. In the Criteria tab, click Add criteria to add the coupon criteria.
  Type field: select Item sales. 
  Equal to field: enter the coffee UPC. 
  Minimum measure field: select Quantity. 
  Minimum value field: enter 1. 
  Select Make sure the discount affects the item(s) in these criteria.
This method forces the system to apply the manual coupon discount on the coffee item.
  Add a Manual description (optional).
  Click OK to confirm the changes and to close the Add criteria window.
 
 
3. In the Details tab:
  Funding field: select Store or Vendor.
  Event type field: select Sale.
  Enter a Start date.
  Enter an End date.
 
 
4. Add any necessary description in the Descriptions tab.
 
5. Click Save in the toolbar to finalize the coupon creation.
  The system will automatically assign the record a coupon code. 
 
  The Criteria tab will display the following information (Text view and Data view).
      

In this example:

  • The customer buys a small coffee for $1.25.
  • The cashier applies the manual coupon of $0.50, sothe customer only pays $0.75 plus taxes.

Buy two bottles of Gatorade

Get one bottle for free

In this example, all Gatorade flavors must be included in the same promotion; the same promotion identification number must be entered in the Promotion ID field of the POS table for each flavor.

1. In the main section of the coupon wizard, add the following coupon information:
  Value field: select Buy X get Y for free.
  Buy X field: enter 2.
  Get Y field: enter 1.
  Coupon type field: select Manual. 
  Description field: add a meaningful description. 
  Sub-department field: select a sub-department. 
 
 
2. In the Criteria tab, click Add criteria to add the coupon criteria.
  Type field: select Promotion ID sales. 
  Equal to field: enter the promotion identification for the bottles of Gatorade. 
  Minimum measure field: select Quantity. 
  Minimum value field: enter 2. 
  Select Make sure the discount affects the item(s) in these criteria.
This method forces the system to apply the manual coupon discount on the bottles of Gatorade.
  Add a Manual description (optional).
  Click OK to confirm the changes and to close the Add criteria window.
 
 
3. In the Details tab:
  Funding field: select Store or Vendor.
  Event type field: select Sale.
  Enter a Start date.
  Enter an End date.
 
 
4. Add any necessary description in the Descriptions tab.
 
5. Click Save in the toolbar to finalize the coupon creation.
  The system will automatically assign the record a coupon code. 
 
  The Criteria tab will display the following information (Text view and Data view).
      

In this example, the cashier must scan at least two Gatorade items before being able to apply the manual coupon.

Buy $5.00 worth of grocery

Get 100 points

1. In the main section of the coupon wizard, add the following coupon information:
  Value field: select No discount.
  Leave the No discount field empty.
  Coupon type field: select Manual. 
  Description field: add a meaningful description. 
  Sub-department field: select a sub-department. 
 
 
2. In the Criteria tab, click Add criteria to add the coupon criteria.
  Type field: select Total sales. 
  Totalizer field: leave the default value to 3. 
  Minimum measure field: select Dollar. 
  Minimum value field: enter 5. 
  Add a Manual description (optional).
  Click OK to confirm the changes and to close the Add criteria window.
 
 
3. In the Details tab:
  Funding field: select Store or Vendor.
  Event type field: select Sale.
  Enter a Start date.
  Enter an End date.
  Bonus field: enter 100. 
  Point program field: select 1. 
 
 
4. Add any necessary description in the Descriptions tab.
 
5. Click Save in the toolbar to finalize the coupon creation.
  The system will automatically assign the record a coupon code. 
 
  The Criteria tab will display the following information (Text view and Data view).

Buy a bottle of Pepsi and a bag of Ruffles chips for $3.00

In this example, all flavors of Ruffles chips must be included in the same promotion; the same promotion identifier must be entered in the Promotion ID field of the POS table for each flavor of Ruffles.

1. In the main section of the coupon wizard, add the following coupon information:
  Value field: select Bundle price.
  Price field: enter 1.
  Coupon type field: select Manual. 
  Description field: add a meaningful description. 
  Sub-department field: select a sub-department. 
 
 
2. In the Criteria tab, click Add criteria to add the first coupon criteria.
  Type field: select Promo ID sales. 
  Equal to field: enter the promotion identification for the Ruffles. 
  Minimum measure field: select Quantity. 
  Minimum value field: enter 1. 
  Multiplication and accumulation field: select Accumulate for bundle or percent (A).
This specifies that the Ruffles chips are part of the bundle (A).  
  Add a Manual description (optional).
  Click OK to confirm the changes and to close the Add criteria window.
 
 
2. In the Criteria tab, click Add criteria to add the second coupon criteria.
  Type field: select Item sales. 
  Equal to field: enter the Pepsi UPC. 
  Minimum measure field: select Quantity. 
  Minimum value field: enter 1. 
  Multiplication and accumulation field: select Accumulate for bundle or percent (A).
This specifies that the Pepsi items are part of the bundle (A).  
  Select Make sure the discount affects the item(s) in these criteria.
This method forces the system to apply the manual coupon discount on the Pepsi items (M). 
  Add a Manual description (optional).
  Click OK to confirm the changes and to close the Add criteria window.
 
 
3. In the Details tab:
  Funding field: select Store or Vendor.
  Event type field: select Sale.
  Enter a Start date.
  Enter an End date.
 
 
4. Add any necessary description in the Descriptions tab.
 
5. Click Save in the toolbar to finalize the coupon creation.
  The system will automatically assign the record a coupon code. 
 
  The Criteria tab will display the following information (Text view and Data view).
     

In this example:

  • The customer pays $3.00 when buying a bottle of Pepsi and any bag of Ruffles chips that are part of the promotion.
  • The manual coupon discount, which is the difference between the total value of the 2 items and $3.00, is applied on the bottle of Pepsi.
