eSuite - Dimensions Overview

eSuite > Dimensions

Information

The Dimensions entry is common to both the Care and the Manage modules. It gives access to lists (departments, sub-departments, categories, etc.), it provides financial information and it allows producing reports on various aspects of retail management (total sales, sales by vendor, department, category, etc).

The Dimensions entry is split in several sub-entries:

  • Financials
  • Departments
  • Sub-departments
  • Categories
  • Families
  • Report codes
  • Vendors
  • Hourly
