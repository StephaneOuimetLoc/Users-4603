Add Cost to Items

To add a cost to an item, first you will need to retrieve the item by selecting "Items" and "Costs" from the Start window

From the Wizard, enter the item number to which you want to add the cost and execute

From the grid select the item and click on Edit at the bottom of the screen then select Master tables and Cost master

On the Cost Edit window, Click on the Add  to fill in the cost information and save the changes.
