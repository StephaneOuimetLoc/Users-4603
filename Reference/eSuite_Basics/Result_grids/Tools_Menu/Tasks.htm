eSuite System > Scheduler

Information

      The scheduler module is split on 3 sections on the eSuite Host system.
  • Scheduler
  • Periods
  • Tasks (covered in this section)

Displaying the task list

In the Launcher, click the Periods entry: the task list displays all the tasks.

This view will allow you to consult your scheduled tasks, along with the target on which they execute and details regarding their last and their next execution time and date.

Editing a task

Double-click any task from the list to open the task in a new window. You can edit the execution target, the type of period and the start and ending dates and provide a description. Once finished, you can save the task using the Save button (identified with a checkmark).

You can also insert a new task by clicking on the Add button (+), remove a task using the Delete button (-) or undo changes by pressing on the Undo button (<).
