eSuite - Desktops

eSuite

Information

The eSuite interface includes a work area called desktop.

The desktop allows you to organize your tasks in multiple views. It is configured to display information windows, wizards, grids and tools.

The desktop is located in the central area of the eSuite application.

Layout

The Home desktop is displayed by default when you open eSuite. Its layout is slightly different from the desktops that are activated when you select launcher entries. It also differs in the information that it displays from one module to another.

Apart from the Home desktop, all the other eSuite desktops open when you click on the entries in the launcher.

The desktop will display the following sequence of pages:

  1. Start
  2. Wizard
  3. Grid

By default, each page opens in a new tab of the same main window.

The desktop also includes a contextual window. Information about it is presented in the Context tool article.

Behavior

This section focuses on the behavior of the start, wizard and grid pages in the eSuite desktop.

The start page

The Start page opens on the desktop when an entry is selected in the launcher. It displays a list of options that are related to that entry.

For example, when you click on Items > Master in the launcher, the desktop will display the Start page with the options related to items.

The options in the start page will open the wizard in a new window tab.

The wizard window

The Wizard opens on the desktop when an option is selected in the start page. It displays search fields and buttons based on that option.

For example, when you click on Items in the start page, the desktop will display the wizard in a new tab of the same window, with search criteria that are related to items.

The search will open the grid page in a new window tab.

The grid page

The grid page opens on the desktop when the search is executed from the wizard. It displays the records that correspond to the search criteria.

For example, when you enter item-related information in the wizard and you launch the search, the desktop will display the results grid in a new tab of the same window.

The name of the grid tab will change depending on the object of the search.
