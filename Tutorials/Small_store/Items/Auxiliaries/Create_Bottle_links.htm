Table of Contents

 Information

The Bottle link table contains bottle deposit and environmental values that are linked to items.

NOTE:  The eSuite software does not support bottle link programming for all types of points of sale. Users may not be able to send the bottle link information to their points of sale. In that case, they must program the Bottle link table so that it reflect the same values as the Bottle link table in their points of sale.

Displaying the Bottle link table

From the launcher in eSuite Care, Manage, Buy or Sell click the down arrow in the Items entry to display the Item auxiliaries menu:

Click Item auxiliaries: the Item auxiliaries Wizard Start window opens in the Auxiliaries work area

Note: In the Batch module, Item auxiliaries is in the main menu as Auxiliaries, not nested under Items

Click Bottle and Environment Fees selection to display Bottle List grid

Click on  in the grid toolbar to display the Bottle Link Edit table

The Bottle Link Edit table is displayed

To create a Bottle link

1. To create a new bottle link, display the Bottle Link Edit table

click on  in the Bottle List grid toolbar

or click on  in the Bottle Link Edit table from a populated table

2. Enter Number, Descriptor, Value and Totalizer 39 for the bottle deposit link (minimum requirement)

3. Enter the Value & Totalizer 40 for the environmental fees as required

4. Enter case deposit if applicable

5. Check off the applicable attributes for tax, food stamp, WIC, or Follow Sub

6. Click the  in the toolbar or hit enter on your keyboard to save

7. From the bottom grid menu, click refresh   to see your entry displayed

To modify a bottle link

Warning: Modifying a bottle link will affect all items linked to it, please take appropriate measures to ensure the change should apply to all items.

1. From the Bottle Link grid, select/highlight the entry you wish to edit

2. Double click on the entry, press F3 on your keyboard or click on Edit from the bottom grid menu.

3. The Bottle Link Edit table will open

4. Modify as required

5. Save
